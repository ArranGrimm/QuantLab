## 目标策略 (Benchmark Target)

> 本项目的截面打分模型旨在复刻/逼近以下策略的日频基线指标。
> 来源：小红书博主公开帖 (两篇帖子 + 评论区)，以下为去主观化的工程级拆解。

### 策略全局架构

| 维度 | 描述 |
|---|---|
| 流派 | 纯多头，空气指增 (不跟踪特定宽基，全市场选股) |
| 主轨 | 尾盘 T+1 截面打分轮动 (日频选股) |
| 辅轨 | 1 分钟级高频日内择时，对底仓做 T+0 降本 |
| 集成 | 12 个子策略 Ensemble，对冲单一逻辑失效 |
| 回测基建 | 自研框架 (2 年)，内置未来函数/流动性/停牌退市/更名/分红复权检测 |

### 数据与特征

| 维度 | 描述 |
|---|---|
| 回测起始 | 2020-09 (创业板注册制，涨跌幅扩至 20%) |
| 量价窗口 | 过去 **128 个交易日** 日 K 线 |
| 基本面 | 有引入，主要作为过滤池 (规避财务暴雷)，非短期 Alpha |
| 标的池 | 仅正股，剔除可转债和 ETF |

### 选股与执行

| 维度 | 描述 |
|---|---|
| 流通市值 | **80 亿 – 500 亿**，专注中盘弹性股 |
| 买卖时点 | 14:30–15:00 尾盘交易，今日买次日卖，日频无缝滚动 |
| 滑点控制 | TWAP / VWAP 拆单 |
| 交易成本 | 双边 **千分之二 (0.2%)** |

### 仓位与资金管理

| 维度 | 描述 |
|---|---|
| 最大持仓 | 同时 **20 只** |
| 平均仓位 | **50%** — 一半现金过夜 (防黑天鹅)，次日盘中供高频 T+0 调用 |

### 业绩指标 (日频回测基线)

| 指标 | 值 |
|---|---|
| 5 年年化 | **50.42%** |
| 最大回撤 | **9.13%** |
| 5 年交易样本 | 7475 笔 (日均换仓 ~6 只，非全量换血) |
| 单次期望收益 | **0.3%** |
| 日收益偏度 | **0.9** (正偏：常态小止损 + 偶发大暴涨) |
| 实盘增益 | 叠加 1 分钟高频 T+0 后，年化再 +10%–20%；已稳定运行 8 个月 |

### 策略瓶颈

博主明确表示单次 0.3% 已是 128 天日 K 模型极限。突破路径仅两条：
1. 全面转向高频量价信号
2. 引入另类数据 (情绪因子、文本数据)

### 与本项目的差距分析

| 指标 | 目标策略 | 本项目 EXP-002 | Gap |
|---|---|---|---|
| 毛年化 | 50.42% | 50.3% | ≈持平 |
| 最大回撤 | 9.13% | 36.0% | **-26.9pp** (核心差距) |
| 偏度 | 0.9 | 0.30 | -0.60 |
| 单次期望 | 0.30% | 0.187% | -0.113pp |
| 日均换仓 | ~6 只 / 20 只 | 68% 换手 (~14 只) | 换手远高于目标 |
| 交易成本 | 双边 0.2% | 双边 0.2% | ✓ 一致 |
| 集成策略 | 12 子策略 | 单模型 | 无集成 |
| 量价窗口 | 128 天 | 20/60 天 | 缺长周期因子 |

> **核心差距**: 回撤控制 (9% vs 36%)、偏度 (0.9 vs 0.3)、换手率。
> 可能方向：增加长周期因子 (128d)、多策略集成、仓位/风控模块、换手惩罚优化。

## EXP-002: +处置效应因子 (42 因子)

- **日期**: 2026-03-22
- **变更**: 新增 `disp_bias_20`, `disp_bias_60` (EWM估算持仓成本偏离)
- **因子库**: `rotation_factors.py` v2 — 7 类 42 个因子
- **其他参数**: 与 EXP-001 完全相同

| 指标 | 毛收益 | 真实成本净 | 100%换手净 |
|---|---|---|---|
| 年化 | 50.3% | 8.2% | -7.3% |
| Sharpe | 1.52 | 0.41 | -0.11 |
| 最大回撤 | 36.0% | 44.2% | 52.5% |
| 日均收益 | 0.187% | 0.051% | -0.013% |
| 偏度 | 0.30 | 0.28 | 0.30 |

| 参数 | 值 |
|---|---|
| 换手率 | 68.0% |
| 日均成本 | 0.136% |

| 年份 | 净收益 | 毛收益 | 超额(净) |
|---|---|---|---|
| 2023 | 0.2% | 37.3% | -9.5% |
| 2024 | -9.9% | 30.7% | -9.0% |
| 2025 | 62.7% | 121.7% | +19.8% |

### vs EXP-001

| 指标 | Delta |
|---|---|
| 毛年化 | +9.3pp |
| 净年化 | +7.0pp |
| 毛Sharpe | +0.20 |
| 净Sharpe | +0.22 |
| 最大回撤 | -6.9pp |
| 偏度 | +0.29 |

**结论**: 处置效应因子通过 GBM 交互效应显著提升毛收益 alpha，偏度翻正。但净收益仍跑输基准 (8.2% vs 13.7%)，瓶颈在换手成本而非因子 alpha。
