# 目标策略 (Benchmark Target)

> 本项目的截面打分模型旨在复刻/逼近以下策略的日频基线指标。
> 来源：小红书博主公开帖 (两篇帖子 + 评论区)，以下为去主观化的工程级拆解。

## 策略全局架构

| 维度 | 描述 |
|---|---|
| 流派 | 纯多头，空气指增 (不跟踪特定宽基，全市场选股) |
| 主轨 | 尾盘 T+1 截面打分轮动 (日频选股) |
| 辅轨 | 1 分钟级高频日内择时，对底仓做 T+0 降本 |
| 集成 | 12 个子策略 Ensemble，对冲单一逻辑失效 |
| 回测基建 | 自研框架 (2 年)，内置未来函数/流动性/停牌退市/更名/分红复权检测 |

## 数据与特征

| 维度 | 描述 |
|---|---|
| 回测起始 | 2020-09 (创业板注册制，涨跌幅扩至 20%) |
| 量价窗口 | 过去 **128 个交易日** 日 K 线 |
| 基本面 | 有引入，主要作为过滤池 (规避财务暴雷)，非短期 Alpha |
| 标的池 | 仅正股，剔除可转债和 ETF |

## 选股与执行

| 维度 | 描述 |
|---|---|
| 流通市值 | **80 亿 – 500 亿**，专注中盘弹性股 |
| 买卖时点 | 14:30–15:00 尾盘交易，今日买次日卖，日频无缝滚动 |
| 滑点控制 | TWAP / VWAP 拆单 |
| 交易成本 | 双边 **千分之二 (0.2%)** |

## 仓位与资金管理

| 维度 | 描述 |
|---|---|
| 最大持仓 | 同时 **20 只** |
| 平均仓位 | **50%** — 一半现金过夜 (防黑天鹅)，次日盘中供高频 T+0 调用 |

## 业绩指标 (日频回测基线)

| 指标 | 值 |
|---|---|
| 5 年年化 | **50.42%** (毛年化 90.25%) |
| 最大回撤 | **9.13%** |
| Sharpe (adj.) | **0.76** |
| 5 年交易样本 | 7475 笔 (1192 交易日, 日均 ~6.3 笔) |
| 胜率 | **54.01%** |
| 盈亏比 (PL ratio) | **3.04 : 2.02** (均盈 3.04%, 均亏 2.02%) |
| 单信号收益 | **0.254%** (Profit per sig) |
| 平均持仓天数 | **2.8 天** |
| 平均持仓比例 | **55.86%** |
| Alpha | 0.60 |
| Beta (R²) | 1.78 (0.28) |
| 日收益均值 | **0.50%** |
| 日收益标准差 | 3.87% |
| 日收益偏度 | **0.90** (正偏：常态小止损 + 偶发大暴涨) |
| 日收益峰度 | **28.15** (极厚尾) |
| Capture up/down | 233.4 / 86.6 |
| 实盘增益 | 叠加 1 分钟高频 T+0 后，年化再 +10%–20%；已稳定运行 8 个月 |

## 策略瓶颈

博主明确表示单信号 0.254% 已是 128 天日 K 模型极限。突破路径仅两条：
1. 全面转向高频量价信号
2. 引入另类数据 (情绪因子、文本数据)
