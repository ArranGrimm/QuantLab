  - 这不是模型学歪了，而是 **教科书结构定义本身就和强收益负相关**
  - 此结论与之前 `structure_score` 是负排序信号、`final_score = structure_score * payoff_score` 会破坏排序的发现完全一致，现在有了统计底表层面的根因解释
- `textbook` 双阶段当前定位:
  - `Stage 1 (is_textbook_b1)` 作为收益排序工具**已失效**，不再适合承担"筛选强收益样本"的角色
  - `structure_score` 不应参与收益排序乘法，也不应作为训练侧的正样本标签权重放大器

## 活跃市值口径更新（`2026-04-14`）

### 机械规则定义

- 多头: `活跃市值` 单根、2 根或 3 根累计涨幅超过 `4%`
- 空头: `活跃市值` 单根下跌 `2.3%`
- 上述规则完全机械，不带人工主观修正

### 当前新认识

- 之前文档中的高收益，建立在**手工 hindsight `LOOSE_PERIODS`** 之上: 先看完整段活跃市值走势，再回头圈出宽松期。
- 这种做法适合作为“理想 market regime 下模型上界”的研究工具，但**不等于可执行择时基线**。
- 当 `LOOSE_PERIODS` 改成上述机械触发规则后，系统表现明显恶化:
  - 交易笔数翻倍
  - 胜率显著下滑
  - 长窗回撤显著放大
- 后续对 `selected` 做最小替换（`bad_k_count -> key_k_recent_20`，特征哈希 `7a606966`）后，长窗交易笔数几乎不变，系统性恶化仍在。
- 这说明当前问题**不在冻结特征集的尾部细节**，而在于: 机械 `活跃市值` 规则放进来了太多“不适合买入但依然被允许出手”的时点。

### 本轮结论

- 旧高收益结果不再作为“当前可实盘收益率”引用。
- 旧高收益结果保留为:
  - 人工 hindsight 上界
  - 验证“在理想多头窗口里，B1 + ML 排序本身有能力”的研究样本
- 后续所有优化，应优先与**机械可执行 regime 基线**比较，而不是继续和 hindsight 上界比较。

### 当前判断（标签维度）

- 没有单一标签能兼得远端高收益和近端低回撤。
- `fwd_mfe_risk_adj_10d` 未能成为银弹：MAE 惩罚力度不足，模型学到的排序与纯 mfe 差别不大，近端 MDD 反而局部恶化。

### 关键发现

- **收紧 seed 比换标签有效得多**: `seed_strict` 在 2022~2025 四个窗口全面碾压 `seed_mid`，收益高出 27~72 个百分点。
- **2024 窗口最亮眼**: 收益从 +106% 跳到 +179%，MDD 从 14.44% 降到 10.21%，胜率 60.6%。
- **交易减少但质量提升**: 笔数减少约 10~20%，但每笔平均收益和胜率显著提高。
- **2026 近端仍亏损**（-5.33%），MDD 略有改善（9.40% vs 11.67%），样本仅 13 笔。
- **2021 全窗口是唯一劣势**: MDD 从 12.73% 恶化到 16.13%，收益略低。
- **核心洞察**: 模型排序能力一直都在，关键是给它排什么样的票。TRIGGER 条件虽简单，但有效地把"没有资金介入迹象"的票挡在外面，大幅提高了命中质量。

## 退出参数优化实验（`2026-04-13`）

### 实验设计

以 `seed_strict` + `fwd_mfe_10d` + `ed42d05d` 为基础，每轮只改一个变量。

### 第一轮: 止损 3% → 5%

| 起始日期 | SL 3% 收益 / MDD | SL 5% 收益 / MDD | SL 5% 胜率 |
|---|---|---|---|
| `2021-01-01` | +286.4% / 16.4% | **+301.4%** / 19.1% | 52.0% |
| `2022-01-01` | +284.4% / 16.3% | +264.5% / 18.8% | 53.2% |
| `2022-01-01` | +207.6% / 13.3% | **+238.9%** / **9.6%** | 55.3% |
| `2023-01-01` | +178.5% / 10.2% | +167.9% / 10.4% | 58.0% |
| `2024-01-01` | +71.4% / 9.6% | **+78.2%** / **9.3%** | 53.8% |
| `2026-01-01` | -5.3% / 9.4% | -4.8% / **8.9%** | 33.3% |

- **SL 5% 优于 3%**: 2022-2025 窗口 MDD 从 13.3% 大幅降到 9.6%，收益 +15%。
- 核心原因: A股 T+1 下 3% 止损太容易被次日低开缺口"洗"出去，5% 给了持仓更多缓冲。
- 全周期 MDD 从 16.4% → 19.1% 是唯一劣势（个别极端交易）。
- **结论: 止损从 3% 调整为 5%**。

### 第二轮: TP 15/30 → 20/40（基于 SL 5%）

| 起始日期 | TP 15/30 收益 / MDD | TP 20/40 收益 / MDD | TP 20/40 胜率 |
|---|---|---|---|
| `2021-01-01` | **+301.4%** / **19.1%** | +288.9% / 23.7% | 46.8% |
| `2022-01-01` | 264.5% / **18.8%** | +264.4% / 24.1% | 49.6% |
| `2022-01-01` | **+238.9%** / **9.6%** | +188.7% / 13.6% | 54.3% |
| `2023-01-01` | 167.9% / **10.4%** | **+188.5%** / 13.8% | 53.7% |
| `2024-01-01` | 78.2% / **9.3%** | **+79.6%** / 15.3% | 51.0% |
| `2026-01-01` | **-4.8%** / **8.9%** | -9.5% / 13.4% | 21.4% |

- **TP 20/40 全面恶化**: MDD 几乎每个区间翻倍，2026 亏损加倍（-4.8% → -9.5%）。
- 原因: 等更高止盈目标的票更多等不到就回撤，白白还利润。TP 15% 是合理的"落袋为安"点。
- **结论: TP 保持 15/30 不动。**
