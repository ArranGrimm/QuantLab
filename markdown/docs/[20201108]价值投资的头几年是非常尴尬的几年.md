#  价值投资的头几年是非常尴尬的几年

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

在我开始投资股票之前我进行了大量的阅读和学习。除了阅读大家耳熟能详的书籍，还大量阅读了那些取得了长期良好成绩的个人投资人回顾的早年走的弯路和踩过的坑相关的文章。考虑到自己的风格偏好和未来长远的预期目标，我最终决定通过价值投资的方式投资股票，期望较长时间后能获得良好的回报。

投资股票这几年来，资金是逐步投入的。一来适应波动需要一个过程，二来怕自己还有一些坑没有识别到。总的来说这几年运行的还算平稳，回撤也不大，收益也比理财要好不少。

但是不得不说，价值投资的头几年是非常尴尬的几年。

普通人(没有从父母那里继承财产或者早年碰到好机会赚了一大票的情况下)投资的头几年是没有多少本金的，少的可能就只有小几万，多的也不过小几十万。

通常来说普通人投资股票平均复合年化收益10%~20%就算比较好的成绩了，按年化15%的话就是5年一倍。然而这个5年一倍并不是每年都有15%，而是可能第一年几个点，第二年几个点，后面3年完成一倍，也可能是前面两年完成一倍而后面三年基本没什么增长。

放到今年来说如果是分散持有低估的优质企业，到现在也还处于还算低估的位置，并没有多少涨幅，而有些新入场的股民或者基民如果踩中了上半年的医疗科技消费等，可能半年就收获了50%的涨幅，吊打价投。这时价投就显得很尴尬了
，  如果想要说价值投资如何如果，在别人看来就像小酒馆里面的孔乙己一样  。

如果选择价值投资的人已经完成理解并认同了价投的投资风格，可以无视其他人短期内的超额收益，那他还得面临另外一个尴尬的处境，那就是回报什么时候才能变成实质性享受。

在刚开始的几年本金不大的时候，每年能获得的收益额是非常有限的，而且由于为了能将雪球滚大，还需要不断的增加投入来推进投资的进度，这个时候就变成了不但不能把收益拿出来享受生活，还需要克制消费将更多的结余投入其中，对家里人来说就是看着市值在波动曲折中慢慢增加，但是又暂时不变现改善生活而仅仅只是一个数字。

这个尴尬的局面至少要维持到本金达到百万级别，到了这个时候自己和家里人经过几年的时间已经习惯了这个状态，并且由于本金稍微大了一点，年收益就算差股息也能有小几万，而如果收益好的话市值可能会增长十来万到小几十万。

这样至少看上去还像那么回事，而这也是很多人在本金少的时候不愿意选择价值投资的重要原因，因为刚开始的几年真的是太尴尬了。

所以，刚开始的几年大概率是尴尬的，不过只要渡过了第一个5年，后面的情况会改善很多。5年一倍，第一个5年将1变为2，第二个5年则能将2变为4。

而将回报从10倍增长为20倍需要多久？大概还是一个5年。
