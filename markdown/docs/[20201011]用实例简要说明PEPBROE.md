#  用实例简要说明PE/PB/ROE

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

投资股票获得回报的方式有两个端点，一个端点是通过博弈获得回报，另外一个端点是通过企业盈利获得回报。技术派趋势派等流派离前面一个端点比较近，而价值派离后面一个端点比较近。

如果想用价值派的方式获得回报，则应该将重点放在企业的财务数据上，而PE/PB/ROE是企业财务数据的入门要素。今天用一个例子来简要说明一下这3个要素，其他要素例如应收应付、存货、在建工程、有息负债、商誉等等暂时忽略。

注意：本文以万科A为例做计算，不构成投资建议。由于存在优先股及港股，所以计算结果会略有误差。另外万科A上一个交易日(2020年9月30号)收盘价28.03元，最近一年不复权最高价33.6元相当于28元上涨20%，最低24.18元相当于28元下跌13.6%。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfL57OiaHjw8QFc7m0wAhJyoGfROga7NCTb1Mv5NGVyicK2npkEEP8lGrf49moNh7QxwXpziavu8rEZxg/640?wx_fmt=png)

##  PB

股价28.03元，每股净资产16.84元，市净率PB=28.03/16.84=1.66，这个意思就是说现在总市值3256.45亿是公司总的净资产的1.66倍，假设现在直接按破产变卖清算，理论上能得到3256.45/1.66=1961.7亿，相应的每一股能得到28.03/1.66=16.88元(四舍五入略有差异)。

##  PE

每股收益(TTM)3.4元，市盈率PE =
28.03/3.4=8.24，这个意思就是说以现在的股价28.03元买入一些股票或者买下整个公司，按目前的每股收益，需要8.24年收回成本。注意，这里的收回成本类似于投资100W开一个火锅店每年赚20W，5年可以收回成本，火锅店仍然在手里，还能把火锅店出售变现。8.24年收回成本后，股票(公司)还可以卖出变现。

那么是不是说现在买入股票，一定需要8.24年才能收回成本呢？并不是，因为这个实际取决于ROE(以及其他要素)。

##  ROE

看一下最近5年的净资产收益率在20%上下。因为没有ROE(TTM)，这里就来计算一下，ROE=每股收益/每股净资产=3.4/16.88=20.14%。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfL57OiaHjw8QFc7m0wAhJyoGwkaXuXuoBzNwUTUclGibbicgyLSqsclVc7FVGtSvf948jsNQRZsHn0UA/640?wx_fmt=png)

回到刚才回本的问题，当前每股收益是3.4，假设未来几年ROE都保持20%，那么每股收益和每股净资产如下(假设不分红)：

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfL57OiaHjw8QFc7m0wAhJyoGFStILjBKcViayibjp9Dnv3f8uV4aTP7VmXgxDXrJElhWCJzhHQNfzDYQ/640?wx_fmt=png)

可以看到，在净资产收益率保持20%的情况下，每股累计收益在大约5.5年后达到了当前的股价相当于收回了投资成本，而每股净资产在6年后达到了50.24元。如果市净率维持在1.66的话，那股价将是50.24*1.66≈83.4元，复合增长率约等于19.9%，约等于ROE。

因为万科每年有分红，股息支付率如下

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfL57OiaHjw8QFc7m0wAhJyoGL7HgYDNMhsDxnkibhyEicqZyTwicjwm7Ol4GNibhdQO14Ogl4TxjRDG2eg/640?wx_fmt=png)

假设未来几年股息支付率为30%，按分红复投再计算一下，为了便于计算按起始10000股计算。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfL57OiaHjw8QFc7m0wAhJyoGlkpcCd2wfTPicG6I4gmvZE5qTwLKxicudrcUsShFOWouTXCibVh7rqOicg/640?wx_fmt=png)

可以看到，在每年将利润的30%用于分红然后用分红再次购入股票的情况下，7年后总股数的累计收益达到283376元相当于收回了初始28.03元买入10000股的成本。如果市净率维持在1.66的话，总市值为12445
*  (42.18  *  1.66)=871384元，7年复合增长率为17.6%，略低于ROE。

可以看到，在PB大于1的时候分红会降低收益，因为1元利润留存将增加1元的净资产，而1元利润用于分红再买入股票的话得到的净资产将小于1元。这也是巴菲特的公司不分红的原因，因为利润留存带来的回报更高，当然这里有另外一个问题，不分红的情况下如果利润增长的速度跟不上净资产的增长速度将会导致ROE下降。

反过来看，用分红买入PB小于1的股票，则会提高收益，因为1元的利润分红后再买入股票能得到的净资产将大于1元。

##  问题来了

  * 市净率能维持在1.66倍吗？ 

  * ROE能在未来6年维持在20%吗？ 

  * 现实会按理论计算进行吗？ 

  

市净率能维持在1.66倍吗？

参考过去5年的市净率，目前处于百分位14.57的位置，这表示过去5年有85.43%的交易日市净率高于当前市净率，市净率在2018年1月23日达到高位3.41。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfL57OiaHjw8QFc7m0wAhJyoGSSE9MX8icMXMKh5pia0aRlsLmqyZoOuNzTF8libfjxXJyDgFiaauTqOibNg/640?wx_fmt=png)

参考过去10年的市净率，目前处于百分位34.07的位置，这表示过去10年有65.93%的交易日市净率高于当前市净率，市净率2014年3月3日达到低位0.95。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfL57OiaHjw8QFc7m0wAhJyoG5OmhAVicSUdiawfFdA31iaF4WqgOv0CbHB02Diawjw2kg997G17LbZrzEQ/640?wx_fmt=png)

ROE能在未来6年维持在20%吗？

这个真没人能知道，  哪怕是万科的管理层都不能打包票。
只能说能维持的概率比较大，未来可能会降低一点也有可能会增加一点。这个就留给万科的管理层去操心吧，毕竟他们要制定KPI。

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfL57OiaHjw8QFc7m0wAhJyoGGtibw5nuyMBYCUM4jBUb0v6MRHC2R86XPRQE3xpEBOS3HFaqCcWA9Dg/640?wx_fmt=jpeg)

本来第3个10年的目标是十年十倍复合增长26%，结果一不小心7年复合增长39%提前完成目标，又用来3年复合增长26%又增长了一倍。当然这些都是过去的成绩，王石离开了万科，而万科早年的高增长也难以复现，但是管理层依然要制定KPI。

现实会按理论计算进行吗？

现实是一定不会按理论计算进行的，股票市场的运行不可能像计算的那么稳定，通常会按这个样子进行

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfL57OiaHjw8QFc7m0wAhJyoGez6icCjcWKKJdE867U7WIp1YdylJfRGJ8uBnKgWAVoJbH5T3Z5I8RmQ/640?wx_fmt=png)

理论计算的值在两条虚线之间，而股价则很可能按波动曲线走。
