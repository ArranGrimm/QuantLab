#  第一个100万完成

本周(2021年2月第一周)收到了一些奖金，可投资资金(含市值)突破了100万，算是完成了一个里程碑，简单记录一下想法，做为春节前的最后一更。

之前写过两篇关于我自己投资目标的文章，分别是《 [ 假如有300万人民币的现金，可以全职投资吗？
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483721&idx=1&sn=e8c74374d572bc80e89d932003a5621f&chksm=cf17b0c3f86039d5f8e78b389eb0c7cee680649df4b2baa3b87fc34476e66c59fa2db30d2493&scene=21#wechat_redirect)
》和《 [ 通过储蓄率来计算一下财务自由到底需要多少钱
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483726&idx=1&sn=610baa301da3805f8f29d4ee9059b14f&chksm=cf17b0c4f86039d2bc75acf4a65254603cc4ed89d00e1e9220be5a70b6692f170179193f918b&scene=21#wechat_redirect)
》。

我的财务自由目标制定的并不高，不是网上一开口就是1000万勉强够至少3000万，或者什么“一线城市2.9亿，二线城市1.7亿”。

按家庭的消费水平和房贷以及预估未来几年的消费情况，我自己测算的结果是大概300万就可以达到不上班的自由。消费自由是无底洞，我只需要达到不上班的自由就行，这样我就可以把原本用来上班的时间拿去做一些有趣但是可能不赚钱的事情。

我之前的设想是这样的：

>
> 目前我的组合加权股息率是4.55%，也就是说几年后如果我的资产雪球滚动到市值300W的时候，每年的股息带来的现金流将大约有13.6W，加上大约有2%~3%的打新收益，整体能带来的现金流将能达到大约20W到23W。

4.55%的股息现金流加5.45%的市值增长，相当于年化10%，再加上平均2.5%的打新收益，相当于年化12.5%，折算为6年一倍，如果增长稍微高一点年化15%，那么折算就是5年一倍。

经过最近几年的实践，去年(2020年)股票组合的xirr收益是15.05%，由于市值不大加上运气欠佳，没有中新股。

在《 [ 一个上班族的2020投资回顾和2021投资计划
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483944&idx=1&sn=f702f29d9cdd111fd3a9e52bf1d1200a&chksm=cf17b3a2f8603ab43dca4a10fe365d95fedd8ff05e675709f146482324fb46bfc5aea8b99236&scene=21#wechat_redirect)
》有提到今年增加双低可转债摊大饼。目前资金的分配是40%股票，15%可转债，45%的理财(长期短期搭配)，后续的增量资金将根据市场的情况分配到股票和可转债中，这样理财资金的占比将会逐步降低。

按目前的市值以及后续的增量资金投入，中新股的概率已经较大了，希望今年能中新股，另外今年准备开港股参与港股打新，再加上可转债打新的收益，整体收益达到10%~15%还是很有希望的。

作为一个还在上班的业余投资者，在不花费多少时间的情况下10%~15%的收益率是一个比较容易达到并且合理的目标。

可能很多人觉得年化10%~15%的收益太低了，毕竟去年很多人买个基金就能有50%的收益。现实的情况是长期看年化超过20%收益率的就是明星基金经理了，如果一个基金某一年达到了50%，那么接下来的两三年能继续做到50%收益的可能性非常小，就算有基金能做到我也没有能力选中。

这问题我在《 [ 价值投资的头几年是非常尴尬的几年
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483849&idx=1&sn=cac519308dd5d17928965a89403510b1&chksm=cf17b043f8603955ed79f3342199a326b6d5024ab99e5101307242965bcd56b15ad865513da6&scene=21#wechat_redirect)
》也有提到过，在本金较少的阶段这个收益率确实会显得很尴尬。资产增值的特点是收益率不变的情况下，后期的收益额增长会越来越快，按5年一倍计算，10W增长到20W需要5年，100W增长到200W也是五年。

如果说第一阶段的增长主要靠工资收入的结余不断投入，接下来的第二阶段可以算是收入结余投入和资产增长双轮驱动的阶段，预计这一阶段(目标300W)需要的时间可能和第一阶段差不多甚至可能更短，再然后就是第三阶段以资产增长为主，工资收入可有可无。

祝大家早日进入第三阶段。
