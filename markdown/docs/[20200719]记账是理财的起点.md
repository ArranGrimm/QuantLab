#  记账是理财的起点

理财需要本金，不论是定期存款、理财产品、基金股票、套利，如果没有本金，这一切都无从谈起。相比没有结余做为本金，如果还要疲于应对信用卡的分期，则信用卡的利息会额外的吞噬本就不充裕的现金流，使得财务状况雪上加霜。想要开始理财，没有数据支撑，也是比较难以挑选合适的品种，这个数据就是自己的财务状况数据。

我对记账最初的印象，来源于父母在老家开的小店铺，小本生意，每天晚上计算当天的营收和开支，然后记录在账本上，便能够知晓当天的经营情况，通过查看历史记录可以回顾也可以规划下一步的计划。

后来我工作了几年以后也开始记账，几年后记账的优势开始逐步凸显，让我增加了积蓄并且理财收入逐步增长。

##  记账的第一步

先理清楚收入和支出，不用关注太多明细，目的是养成习惯并且让收入支出做到心里有数。参考方法如下：

  1. 发了工资记一笔收入。 

  2. 转了钱到微信，由于微信用的频率较高而且数额较小，直接记录一笔开销。需要看明细的时候可以在微信中查看。 

  3. 取了现金到钱包，由于现在现金使用频率较低，记录一笔从工资卡到钱包的转账。周末或者月底清点一下，差额记录一步开销。 

  4. 如果有信用卡，由于使用频率也不高，每次消费完随手记录一步开销。 

以上4个步骤，可以先只记录大类开销不用记录的非常明细，主要目的是养成习惯，做到心里有数。

##  记账的第二步

经过一段时间比如一年的记账，回顾一下，就会有一点不一样的感觉，会比较清楚的知道自己一年的收入，一年的开销，也会开始反思自己是不是可以找一份工资更高的工作，是不是有些东西买来就没有用过或者只用过一次，是不是有些钱花的比较没有价值可以省下来，等等。

如果养成了习惯，从第二年开始，会明显的发现自己的冲动消费减少了，消费更加理性了，心里也更踏实了。

##  记账的第三步

如果平时消费习惯比较理性，则开始记账时就会有一些结余了，而如果是月光族，经过两年的记账也会告别月光开始略有积蓄。

有了积蓄，并且也能很清楚的知道下个月，下个季度，下一年会有多少积蓄，就可以开始规划理财了，这将是一个长期的过程。
