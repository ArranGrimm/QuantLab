#  面对浮亏你焦虑吗

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

在浮亏面前如果感到焦虑，那一定是投资系统不对，或者说至少是不完备的。

以年初为起点，这个月我的股票账户开始浮亏了，经过这周后面4天的回升，目前浮亏比例不到5%。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfLG2NVtJI75uWMRCYicvHanic3RcaRVicOrxS5gMAPQEOjV59EBduZQibWdWS7esKtKmbl2AckR6RtleQ/640?wx_fmt=png)

这只是股票部分，转债部分的收益不是今天的重点。

这个浮亏还是建立在我的前3大持仓相对年初显著下跌的基础上的：

万科A持仓占比11.4%，年初价格28.7，当前价格24.51，下跌幅度14.6%

中国平安持仓占比11%，年初价格86.98(不复权)，当前价格66.68，下跌幅度23.3%

保利地产持仓占比9.4%，年初价格15.82，当前价格12.47，下跌幅度21.1%

  

之前在豆瓣上看到两个帖子，基本上是一个意思，分别是

> 想问问大家 没啥本金还持续亏钱的投资苦手该怎么调整心态？
>
> 你们是如何来平复自己连续亏损的心情的？

  

我是这样回帖的：

> 在入市之前，就已经把浮亏的心理准备做好了。
>
> 我在不能承受浮亏的时候一直没有进场。直到有一天我理解了股票的本质，明白了什么是价值投资，并且能够接受和适应浮亏了， 然后我才进场。
>
>
> 经过这几年的实践证明，我能适应浮亏，浮亏不会影响我的情绪，也不会影响我的投资计划，甚至我的投资计划里面有一部分就是浮亏情况下的应对计划。最后的结果就是从我入市到现在，每年都是盈利的。虽然每年都平平无奇，但是积累起来还是很可观的。进场赌博，和进场投资，是不一样的。

  

我今天想重点来说说，为什么浮亏不会影响我的情绪，也不会影响我的投资计划。

抛开那些复杂的计算和道理，实际上面对浮亏的时候，只需要问自己5个问题：

  1. 国家会遭遇毁灭性的破坏吗？ 

  2. 国家的经济还能继续平稳增长吗？ 

  3. 持仓的组合，还有希望吗？ 

  4. 持仓的某个股票，还有希望吗？ 

  5. 现金流还能维持吗？ 

这五个问题，由大到小，决定了我们的投资，能不能从浮亏走向浮盈。

  

先看第一个问题：国家会遭遇毁灭性的破坏吗？

这个可能性是极小的，并且如果真的遭遇了，那应该考虑的不是亏钱的问题，而是保命的问题，所以这个没什么好说的。

  

第二个问题：国家的经济还能继续平稳增长吗？

这个可能性是极大的。这里不需要给国家唱赞歌，相信大家都有感受。

  

第三个问题：持仓的组合，还有希望吗？

我的持仓组合，当然是没有问题的，毕竟那么分散，最高的占比也才略超过10%而已，即使这个企业从地球上消失，这个股票直接归零，也只会对我造成10%的亏损。并且我持仓的股票除了个别估值稍微高一点，其他的都是估值偏低。

当然我经常会听到有人说，低估肯定是有原因的，高估也是有原因的。言下之意就是说，低估的这些企业肯定有什么问题市场才不买账，高估的企业一定是有什么优势，市场才会追捧。

那么我想说，如果是这样的话，牛市和熊市为什么总是交替？为什么那么多人投资股票亏钱？

因为他们不会在低估里面看机会，不会在高估里面看风险。这里的会，不是指学会的会，而是指会不会去做的那个会。

正确的思路应该从低估中看到机会，在高估里看到风险。但是这个说起来简单，真正能做到的人却很少。

所以最终能在股票市场赚钱的人是少数，而多数人不论中间账面浮盈了多少，最终基本上都无法盈利甚至亏损。

  

第四个问题：持仓的某个股票，还有希望吗？

对于这个问题，我的答案是，不知道，但是大概率是有希望的。这里的希望指的是企业的经营没有问题，利润依然会增长。至于股价，最终会反应企业的业绩，但是这个时间可能很长。

当然，我并不是说万科平安保利是好还是不好，因为任何一个企业都有可能短时间内就不行了，只不过对不同的企业这个概率不同而已。

所以我看到满仓单吊一个股票的人，我只能祝他好运。

  

第五个问题：现金流还能维持吗？

经常看到有人发帖，诸如：“我有一笔X万的钱，一年(两年/三年)用不到，买什么股票好？”。

这其实是非常不妥的，因为真正适合投入股票的钱，应该是闲钱。

对于闲钱，有两个主要的特点：

一个是不被计划为生活费，也就是说这笔钱应该是规划在你的生活之外的，专门用来作为投资的钱，对比在房产领域，就是用来买第二套第三套的钱，而不是用来买第一套的钱。

另一个是这笔钱的周期应该很长，不是什么两三年用不到，而是十年八年甚至更长的时间用不到，又或者说是计划永久投入在股票上的钱。

如果用于投资股票的钱是这样的真正的闲钱，那肯定不会在意一时的浮亏。

这个也是为什么很多人的模拟盘收益很好的原因，因为其实稍微有点投资常识的人都能分辨出什么时候应该投入，什么股票应该投入，而对很多人来说模拟盘的钱才是真正的闲钱，不能用来生活，永远取不出来。而到了实盘，真正是用闲钱投入的人，其实是非常少的。

然后就是现金流，如果没有稳定的现金流，在漫漫熊市，如果需要用到一大笔钱，则无法通过拆借筹款用现金流还钱的方式来解决，就只能低位割肉套现来解决。

而如果有稳定的现金流，那熊市将成为不可多得的播种机会，等到市场复苏时将会获得丰厚的回报。

  

如果面对浮亏你感到焦虑，请仔细思考上面的五个问题。
