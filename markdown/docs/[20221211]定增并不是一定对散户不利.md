#  定增并不是一定对散户不利

在小说阅读器中沉浸阅读

最近 市场上比较引人关注的是 万科计划增发，我持仓的另外一个股票 伟星股份 也计划增发12亿

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLRIAickRbXKkM4YicFFRMHicdfG2qVtqjEpCfA5ClLg5sZP6N6say766I6VAYq24YoI7wyyV6xFeialw/640?wx_fmt=jpeg)

那么增发对散户有利还是不利？

通常散户认为定向增发股票是机构占了散户的便宜，因为增发价格通常低于市场价(华夏银行这种高于市场价增发的例子也有)，散户觉得自己手里的股票xx元，机构却只需要支付更低的价格，自己吃亏了。

然而散户真的一定是吃亏了吗？

我们来计算一下

假设某个企业目前的每股净资产是10元，总股本10亿，则企业的净资产是100亿，此时股价是X元

假设企业的ROE是10%，分红比例是50%

某个投资者原来持有10000股，市值10000*X元

一年后每股收益是 1元(10*10%)，分红5毛，另外5毛增厚净资产

如果估值不变的情况下，每股净资产从10元提升到10.5元，则股价提升到1.05X元

则投资者的市值为 10000*1.05X+5000元(分红所得现金)

又一年后每股收益是1.05元(1.05*10%)，分红0.525元，另外0.525元增厚净资产

如果估值不变的情况下，每股净资产从10.5元提升到11.025元，则股价提升到1.125X元

则投资者的市值为 10000*1.125X+10250元(分红所得现金)

以上是没有增发的情况

假设企业在第一年结束后增发2亿股，每股价格16元(高于净资产)，并且增发后ROE维持不变，估值也维持不变

一年后每股收益是 1元(10*10%)，在利润分配前增发，则总股本增加为12亿，每股净资产为(10元*10亿+16元*2亿)/(10亿+2亿)=11元

每股收益被摊薄为 (1元*10亿)/12亿=0.833元，分红0.416元，另外0.416元增厚净资产

如果估值不变的情况下，每股净资产从10元提升到11.416元，则股价提升到1.1416X元

则投资者的市值为 10000*1.1416X+4160元(分红所得现金)

又一年后每股收益是1.1416元(11.416*10%)，分红0.5708元，另外0.5708元增厚净资产

如果估值不变的情况下，每股净资产从11.416元提升到11.986元，则股价提升到1.1986X元

则投资者的市值为 10000*1.1986X+9868元(分红所得现金)

对比不增发和增发可以看到，增发后当年会摊薄每股收益，导致分红减少，但是由于增发融资的资金会直接增加到净资产上，使净资产的增厚程度显著高于不增发。

**而如果ROE保持不变的情况下**

增发后接下来的年份每股收益的增加幅度会显著高于不增发，对应的分红也会更高

如果继续计算，会发现增发股份后的几年后，累计分红会大于不增发的情况，而如果估值不变的话股价也会高于不增发的情况。

现在再来看看原来的股价X元

原来的股价X元比增发价格高还是低，并不影响结果，因为增发增厚的净资产与股票的市价没有关系。

要说有关系的大概只有散户的心情

如果原来的股价X低于增发价格，散户会觉得自己没吃亏

而如果原来的股价X高于增发价格，散户会觉得自己吃亏了，凭啥机构拿到的股票比我便宜？

实际上只要增发价格高于净资产，原股东就占了便宜，区别在于占的多还是占的少。

这里的前提条件就是

**ROE保持不变**

如果一个企业盈利能力不行，增发后roe更低了，那定增的作用就很差

不过这样的企业，本来就不应该投资，不是吗？
