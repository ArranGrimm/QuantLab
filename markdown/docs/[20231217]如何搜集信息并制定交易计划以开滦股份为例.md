#  如何搜集信息并制定交易计划-以开滦股份为例

在小说阅读器中沉浸阅读

上周有一个在公募基金工作的朋友，问了我一些问题

因为最近三年市场普遍都是跌的，大多数人都是亏钱的，而我这三年都是盈利的，回撤也很小。

所以他问了一些选股和制定交易计划的细节问题，聊的比较晚，还有一些方面没有涉及到。

加上之前也有几个网友问我关于数据在哪里看的问题

所以今天干脆详细的写一下我搜集信息并制定交易计划的过程和逻辑，供大家参考

至于选开滦股份为例子，是因为它的要素比较全面。

数据查看的几个来源分别是：雪球、同花顺(游客登录即可，我没有注册)，上海证券交易所官网

####  基本信息

企业属性：国有企业，河北省国资委控股

图片来源：雪球

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIV8JqwC04lIck3O5RgSnSlOicvFjsG1bolqVoJPx2GJ1P6bZQw0ubCRsWfAdgqmFrz1aQMUAXrq3A/640?wx_fmt=jpeg&from=appmsg)

这个属性决定了它破产和退市的风险极小，可以忽略不计，在历史低位加仓不用过于担心。

  

股东和控股情况，图片来源：同花顺

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIV8JqwC04lIck3O5RgSnSlC1R3XO2ibgk4qKnSfRvrxVL8eicAVZQ55IMOZV4ibnEAHnl2tl12yCUPw/640?wx_fmt=png&from=appmsg)

大股东是自己，这个决定了控制权和经营决策的稳定性，以及在维护自身利益方面(比如给自己分红)的动机和能力。

  

产品和业务：挖煤，卖煤，制造并销售以煤炭为原料的其他化工产品

相对单纯的挖煤卖煤，被煤价影响的程度小一些，因为其他化工产品用的是自己挖的煤。

主要客户是几个国有钢铁厂，图片来源：同花顺

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIV8JqwC04lIck3O5RgSnSlAWQDdIEoGtkYfOlWiaPAJuU9a3SRz7libK8oIhNKRu7RP0MiauPforWMg/640?wx_fmt=png&from=appmsg)

这个决定它的基本盘，收入相对稳定

####  决策逻辑

现在把时间拉回到2023年4月18号，当时只能看到2022年的年报，看不到2023年的一季报

看到最近5年的利润情况是这样的，图片来源：雪球

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIV8JqwC04lIck3O5RgSnSlQfmPvPqAXedBvmNfkYIY7zQzuxiaQVC7fImXtJluFo0KMib4lhvc8ZMA/640?wx_fmt=jpeg&from=appmsg)

可以看到2021年利润大幅增加，2022年利润略微下降

那么站在当时的位置，应该如何考虑？

当然是保守的考虑，假设2023年利润继续下降，那么下降多少呢？

毛估估，暂时认为降低到2020年水平，也就是降低大概40%，如果在这个情况下，会如何？

从上面逐年PB区间的图可以看到

2019年PB区间0.71到1.1

2020年PB区间0.61到0.86

2021年PB区间0.63到1.47

并且0.61是上市以来PB最低值

  

2023年4月18号当天的PB大概是0.8，相对最低点还有大概23%的空间，而且5月会收到每股0.58元的分红，如果不考虑一个月的时间，则相当于7.07-0.58=6.49元买入，实际并不到0.8PB

  

因此可以作为建仓的起点，考虑到仍然有可能继续跌，所以只买入了1.5%的仓位(总额一般不超过5%)

随后时间来到了5月，派息后继续跌，于是在5月22号到5月26号之间，再次买入1.5%的仓位

5月26号的PB到了0.7

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIV8JqwC04lIck3O5RgSnSl7QibH17rh2YILvtdfGVBWn82t4cg0noM6iaCNZydA0bFsIJicUIFgaSlg/640?wx_fmt=jpeg&from=appmsg)

剩下的仓位，计划PB到0.6附近再买入

买入后继续跌，6月26号达到最低，价格5.79，PB 0.68

没有到计划买入位置，继续观望。

  

另外一方面是股息，一季报出来以后，显示利润同比下降40.51%

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIV8JqwC04lIck3O5RgSnSlllmhVKL7rwcPiaHewOcFSfF92InGTF2ucr9UCvIF23KCrBHclUrzvUA/640?wx_fmt=jpeg&from=appmsg)

和我瞎蒙的差不多，那么如果全年按这个水平，则2023年的派息比例还是保持上一年的49.71%的话，则每股股息就是0.345(0.58*(1-40.51%))，股价6块则股息回报就是5.75%。如果股价是8块的话，股息回报就是4.3%

这个股息水平不算高，也不算低

考虑到PB估值已经接近历史极限，所以维持原计划不变。

  

目前按三季报，利润下降45.62%

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIV8JqwC04lIck3O5RgSnSlz3PWvs9rE9fJRK0a6lQb6CbdO1p664Rp6aCOdNXBicVpnS8naFR0efw/640?wx_fmt=jpeg&from=appmsg)

继续维持买入计划不变，但是近期股价相对5月的买入已经涨了超过20%，所以做了小幅减仓(不到1%)

剩下的继续观望

毕竟目前的PB只到0.9附近，从历年PB区间看，除了2023和2020年，其他年份的PB最大值都是大于1的。

  

####  更深入的研究

以上都是根据最基本的数据，进行的一些估计以及在这些估计的基础上做的决策

如果只想拿一个及格分，基本上就差不多了  ，在低位蹲着，等待市场的馈赠。

如果想要更深入的研究，可以到同花顺上查看最近一些年三张表的数据(利润表、资产负债表、现金流量表)

以及最近几年的年报，可能会发现更多的信息

比如企业更多的优势，或者一些缺陷等等，降低踩雷的风险。

不过对于低估分散的策略，加上国企性质并且长年派息，踩雷的风险本身已经很小了

继续深入研究能带来的提升空间比较有限，而需要花费的时间和精力代价却是成倍的，甚至是几十倍上百倍

因此目前并没有把深入研究作为重点。

  

低估区域毛估估筛选，股票组合整体上大仓位，属于主要矛盾，精确分析企业提高命中率降低踩雷风险，属于次要矛盾。

优先解决主要矛盾，有时间精力和兴趣，再去解决次要矛盾。
