#  持续稳定的现金流在股票投资过程中的重要性

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

本周A股市场三傻和白马都跌了不少，不少人的市值遭遇了明显的回撤开始坐立不安，甚至有人在心理压力下割肉，这个情况下
**持续稳定的现金流在股票投资过程中的重要性** 就体现出来了。

  

先来看一个老段子翻新：

>
> 有一个农民给地主打工，地主说我给你每个月一石米。农民却说：“我有一个大胆的想法：你给我第一天一颗米，第二天两颗，第三天四颗，第四天八颗...后面全部翻一倍。”地主想这农民傻了吧，要这么少，就答应了。后来农民坚持了十天终于饿死了。

这个翻新段子的原版是印度国王为了奖励国际象棋的发明人，对方要求在棋盘的第1个小格里放1粒麦子，在第2个小格里给2粒，第3小格给4粒，一直放满整个棋盘，最终结果是一个天文数字。

本来这个故事是用来说明复利的威力的，不过今天要说的重点不是收益率，而是现金流在股票投资过程中的重要性，也就是为什么段子中的农民坚持了十天终于饿死了。

段子中的农民，在一段时间后将获得巨额回报，但是由于没有口粮(现金流)支撑，没能活到那一天。

实际上打算在股票市场长期投资的人基本上都知道，在低估的时候分散买入优质企业并且长期持有，在较长的时间周期来看是能取得较好的正收益的，平均来看一年大概能获得10%~20%的收益。但是在整个投资周期的前半段由于本金少，能获得的收益额非常有限，就像段子中的农民前十天获得的粮食，根本不足以养活自己，一年的收益可能不如加薪一次来的多。

因此在投资的早期得靠工作收入的现金流维持生活，不要指望投资能带来多少收益额。投资早期由于本金少，因此收益的增量显得非常少，市值的增长缓慢，此时如果不断的持续投入效果将非常显著。假设在刚起步投入的本金是10W，连续3年每年增加投入5W，加上3年的收益市值大概能有30W(年化收益10%~11%)，但是如果没有持续的投入要达到这个市值按11%的年收益率计算则需要10年半，整个收益曲线向后推迟了7年半。如果年收益是15%，持续投入3年后的市值大约是32.5W，不持续投入要达到这个市值则需要8.5年，整个收益曲线向后推迟了5年半。

由此可见，早期如果现金流较为充裕，一方面可以维持正常的生活，另一方面可以持续的投入从而显著的加快整个投资曲线的推进，当然可能有很多人觉得这样的收益太低了而选择了更为激进的策略期望能在短时间内取得高额收益，但是事实上这样大反而很有可能会拖累整个投资周期，就如我之前写的《追求高收益是大部分人最终无法盈利的根本原因》，最终的结果可能是将前面几年的投资进度摧毁，白白浪费了几年。

另外一个方面，如我之前写的《能适应曲折前进才适合投资股票》，投资的过程不是一直向上的，会有反复会有波动，这个时候如果没有持续稳定的现金流(工作收入或者股息收入)，则会面临两个潜在的风险：

  * 如果需要用较大额的钱，很可能在股价低位被迫割肉，造成实质性亏损，整个投资进度倒退一部分。 

  * 本来指望用投资收益改善生活，看到本金出现了浮亏，不仅没有改善生活，还面临了巨大的心理压力，在压力下割肉，造成实质性亏损，整个投资进度倒退一部分。 

而如果有持续稳定的现金流收入，则能够很大程度规避上面的两个风险，需要用大额的钱时可以先刷信用卡然后用持续的现金流来还，需要改善生活则多积攒几个月的结余即可，这样才能保障投资进度不倒退。而所谓的用闲钱投资，就是用不需要动用的钱投资，没有持续稳定的现金流，闲钱很难说不需要动用。

在市场大面积长期阴跌的时候，往往很多人受不了市值的持续缩水而卖出股票，实际上这个时候很可能是低估加仓的良机，如果有持续的现金流，此时就能从容的持续买入，等待回归。当然这个回归的时间可能会比较久，短则一年半载长则3年5年甚至更久，由于有持续的现金流，这个过程并不会特别难过，而且由于可以用打折价买入股票，没准还能觉得很划算。

到了投资的后期，投资组合带来的现金流能够覆盖生活开销，并不需要卖出股价增长后的股票来改善生活，这将能保障市值进一步增长同时带来现金流增长，这个可以参考我之前的文章《假如有300万人民币的现金可以全职投资吗》《通过储蓄率来计算一下财务自由到底需要多少钱》。

因此持续稳定的现金流能在面对市值波动的时候减少心理压力，在投资的早期能显著的加快投资进度，在投资的后期可以用来改善生活或者替代工资收入而不需要卖出股票。

最后，祝大家投资顺利。
