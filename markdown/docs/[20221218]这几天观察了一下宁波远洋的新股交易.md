#  这几天观察了一下宁波远洋的新股交易

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

因为中了新股 宁波远洋，所以从新股上市开始就在关注，当然第一天是毫无悬念的涨停了，并且按照雪球网友的预估，涨停大概能维持到第3到5天

所以第二天和第三天我只是在竞价的时候看一下挂单和撮合的情况，不出意外应该都是涨停的，并且收盘前的封单还挺多

第4天封单稍微减少了一点，但是还是挺多的

前面几天逛了一下东财的股吧，里面投(赌)机(博)的氛围非常浓郁

各种什么“空狗又想骗筹码”，什么"庄家什么什么的"

挺有意思的

我一般只有在中签的新股上市的时候会在东财看一下挂单情况，顺便看一下股吧的帖子

之前并没有太在意，不过这次稍微多花了一点时间看，我觉得其实也不能完全无视这一块的生态，没准里面就有可乘之机。

到了第5天，竞价的时候挂单显示撮合已经非常充分了，未撮合的单子是卖单，而前面几天未撮合的单子是买单，这也意味着开板的可能性很大

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI6xkpAoicX8Yr8YxNbRAByMiabMlKUGo4dicgzoibYj4icfkA8elKFtYrdE4oJ85YY5xNDnKB3SWqia7ow/640?wx_fmt=jpeg)

于是我果断的挂了跌停价14.18，最终竞价17.08全部出了

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI6xkpAoicX8Yr8YxNbRAByMApQInRHPiavJpEWDbu5xOCO35utdIKzjt2bJChrIgSHwpoT4DPxyNUA/640?wx_fmt=jpeg)

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI6xkpAoicX8Yr8YxNbRAByMOAvJFIib7pzHI6T4ezu1fI5kIZJibuEnGDjjGicKhcfWp8nHguw9VIx2w/640?wx_fmt=jpeg)

剩下的就是看戏，果然开板了，不过很快就封板了

股吧里充满了欢乐的气氛

有卖出后恭喜别人发财的(我在雪球也发了一个贴恭喜有格局的网友继续赚钱)

也有破板买进去准备再吃几个板的

有封板后大骂庄家骗了自己筹码的

也有庆幸自己坚持没卖的

投(赌)机(博)的氛围非常浓郁，看着他们的帖子，我能够体会他们的开心与沮丧

我也能理解为啥那么多人热衷于短线炒作和打板了，确实是太刺激太带劲了

作为一个理性的投资者，我并不排斥赔率和胜率占优的投(赌)机(博)

这个我在《 [ 胜率、赔率以及价值投资
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484372&idx=1&sn=cfce52fd7ed957c435a3891ccaa30202&chksm=cf17b25ef8603b4867eb8586056711689d93f0899500c399be19aa534141acbc3d832543765c&scene=21#wechat_redirect)
》中提到过

> 对于我而言，投资就是赌马，在赛马场里，我在寻找一匹获胜概率是50%，赔率是一赔三的马，在股票市场中，我在寻找是标错了赔率的赌局，这就是投资的本质。 -
> - - - 芒格

而战胜过赌场的索普教授，则是这样的
![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfJGx7XzJPI0icuoUN45pNkK1y4fxvqKGbDicCliahM7JkHMdjQKpudaibpBc4ZkbcJjEVhNtHJQzRu9kA/640?wx_fmt=png&wxfrom=5&wx_lazy=1&wx_co=1)

但是开板后炒新，难度太大，短期内是纯粹的博弈互摸口袋，很难找到胜率和赔率占优的机会

不过如果在离预测开板还有2到3天的时候通过隔夜委托挂单，一旦买入就很有可能吃到大概2个板，这倒是可以考虑试一试，不过可能和抽签茅台差不多，聊胜于无。

第6天竞价相对前一天涨了4.21%，最终冲高到18.88后回落到水面以上，中午开盘后就跌到水面以下，最终收盘跌了7.16%

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfI6xkpAoicX8Yr8YxNbRAByMTWNcwbeHYqBTTtqGcTDWcibvr22OXibnr13FX4CD83FYJ9tOLNuabxEQ/640?wx_fmt=png)

有人庆幸自己高价跑了，也有人痛苦被埋。

第7天是这样的

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI6xkpAoicX8Yr8YxNbRAByMA4vJ3GLSuU0QlhzADicxadMSX1J8P21nUSvMRhJE4Al6xITfuNpR6KA/640?wx_fmt=jpeg)

开盘后不久的大绿柱意味着大量的筹码低价抛售，如果是前一天进的，则意味着割肉离场

这一波割完后，价格又回到了前一天的收盘价之上

最终尾盘的一个大红柱，又将价格拉涨停了。

可以看到第5天竞价卖出还是比较合适的

后面几天就是投(赌)机(博)者的游戏了，操作难度大，很容易被割韭菜。

如果没有过人的能力，还是不要参与比较好。
