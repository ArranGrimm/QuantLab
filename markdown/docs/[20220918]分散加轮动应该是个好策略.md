#  分散加轮动应该是个好策略

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

上周的《 [ 分散的收益可能比集中更高
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484882&idx=1&sn=ffd41706fa9d35c54be57fbb5cb84b85&chksm=cf17b458f8603d4ebf9aac2a979e8cc635ef3727af7e5796f2849a4141649b1fcace3e6873db&scene=21#wechat_redirect)
》中用几个例子做了一下计算来说明分散的收益并不会比集中差，甚至在很多时候会更好

里面的例子是一年轮动一次，很多投资的书籍里面的策略也是这样建议的，一年做一次轮动或者再平衡。

但是实际上我们可以轮动的更频繁一点，比如像施洛斯那样，效果可能会更好。

特别注意，这里更频繁的轮动是基于自己的规则，而不是去追逐热点或者情绪驱动的频繁交易。

很多人希望的可能是抓到一个牛股涨个3倍多，再抓到下一个牛股再涨一个3倍多，这样几年就能完成10倍的收益，这样的人不能说没有，但是肯定很少，并且里面或多或少有运气的成分。

作为一个凡人，我们还是不要抱有这样的幻想。

以我的组合为例，股票数量维持在40个左右(另外还有大概40个转债也在做基于我自己的规则的轮动)

这一周轮出了一些 保利发展 ，占比从9月4号的汇总表中的3.23%(价格17.02)降低到1.6%(周五收盘价18.05)，周五早上我是竞价18.8卖出的

我并不知道卖出后它会继续上涨还是下跌，卖出的原因仅仅是因为按照我的估值评价体系，它的性价比已经比较低了，打算卖出后买入一些其他的成员，而周五的下跌只不过是巧合。

下跌过程中，不少之前已经下跌的股票进一步下跌，已经到了我的买入区间

于是在16.13的位置买入了一些 柳药集团，在9.23的位置买入了一些 千金药业

同样的，买入以后它们还在继续下跌，收盘价分别是15.98和9.12

如果后面它们上涨到性价比比较低的位置，我也会分批的轮出后再轮入其他的成员

事实上，7月21号 柳药集团涨停(原因我不知道，也不打算花时间去了解)的时候就已经18.01轮出了，今天又轮进

而千金药业在今年6月派息前10块多轮出过一部分，7月底8月初9块左右又轮入过一部分。

接下来会轮到哪个上涨哪个下跌，我不知道也不关心，我只知道市场总是在过度乐观和过度悲观之间来回摇摆。

对手盘总是基于消息和情绪进行买入卖出，而我总是基于自己的估值体系进行轮动，这样不断的积少成多，最终就能获得较为可观的收益。
