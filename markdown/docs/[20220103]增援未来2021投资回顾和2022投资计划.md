#  增援未来 2021投资回顾和2022投资计划

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

在小说阅读器中沉浸阅读

2021年是我投资的第4年，再一次取得了正收益，已经连续4年取得了正收益。

这一年有两个重大的变化

一个是大幅增加了本金，从年初的不到30W的股票仓位，陆陆续续增加了接近80W的本金投入，年末股票加转债市值大约120W。

另外一个是我退出了职场，开始用投资的现金流回报代替工作收入。

这一年的投资成绩综合xirr为24.81%，高于预期

股票账户总体xirr为17.66%，其中新股贡献了大约7.5W，股票贡献了大约3.2W，股票获得了大约3.6W的股息，也就是说扣除股息和转债打新收入，股票本身是微弱浮亏的。

转债账户总体xirr为56.21%，回报非常可观(新债收益基本上可以忽略)。

关于收益率的计算方式，见之前的文章《收益率的几种计算方式》(链接在文末)

##  股票

股票依然是低估分散的格雷厄姆风格，并且偏重股息回报，基本上可以认为我的股票组合就是个人版的红利增强基金。

股票组合的前4大持仓平安万科保利农行有2个股票在这一年大幅下跌，

其中

平安从年初的86.98(不复权)跌到年末的50.41，跌幅42%

万科从年初的28.7(不复权)跌到年末的19.76，跌幅31%

在这一年股票行情并不好并且重仓大跌的情况下，股票组合在扣除新股收益后还能取得微弱的正收益，说明这个组合还是非常稳健的。

##  转债

这一年的转债收益主要源于2021年初的黄金坑，识别到系统性机会后投入资金，最终获得了非常可观的收益。

可转债组合采用的是类双低策略，和集思录的双低策略略有差异。

以价格为优先考虑条件，在黄金坑中挑选价格低于100元的转债，然后在这些转债中筛选溢价率低的，去掉PB过低没有下修空间的，去掉亏损的。

当然在这一年中，以持有封基老师为代表的一众高手，采用了低溢价策略，取得了更高的回报，非常惊艳。

但是我还是没能理解低溢价策略的逻辑，所以采用了类双低风格的策略。

另外基于强赎博弈的逻辑，我在10月8号买入了  海印转债、洪涛转债、济川转债，目前3个月时间，浮盈分别是
11.11%，11.1%，32.14%，等这3个转债都获利出清后，我再来写一篇回(chui)顾(niu)一下

##  上一年计划回顾

上一年的回顾和计划中有3个问题

>   * 单股持仓占比过高，平安占到了16%，后续如果没有特别优惠的价格将暂时不再买入平安，优先增加其他股票。
>
>   * 银行股占比过高，合计占比超过30%，后面考虑在合适的时候适当减少农行和交行，优先增加其他股票。
>
>   * 消费股占比过低，目前在观望双汇发展和伊利股份，会在合适的时候买入。
>
>

第一个问题，由于平安股价从86.98跌到目前的50.41，并且优先增加其他股票，目前平安的占比已经降低到7%。

第二个问题，银行股占比目前已经下降到23%，主要也是股价下跌并且优先增加其他股票的结果，没有主动减持农行和交行。

第三个问题，已经买入双汇和伊利，双汇买入价格偏高目前浮亏8.x%，伊利买入的价格还不错，目前浮盈20.x%

在2020年的总结中提到的2021年的计划

>   * 将理财产品和存款中到期的资金逐步增加到证券账户。上班族在难以提高收益率的情况下，增加本金能显著的推进投资进度，直到工资收入开始显得不重要。
>
>   * 调整上面提到的三个问题。
>
>   * 继续买入低估的A股和打折的港股。
>
>   * 增加可转债摊大饼，挑选20个平均买入，然后每个月调入更具投资价值与性价比的，淘汰持仓中投资价值和性价比最差的。
>
>

第一个，已经增加了大约80W的本金投入。

第二个，完成的很差，基本上是靠股价下跌被动完成的，如果是卖出股票主动完成，今年的收益将会更好。

第三个，大量增加了A股份额，少量增加了港股。

第四个，完成的很好，直接提升了整体的收益。

##  本年计划

  * 优化持仓的股票，一方面继续调整比例，一方面提升持股的性价比。 

  * 继续增加股票和转债的份额 

  * 有合适的可转债则继续轮动，如果没有合适的则逐步退出。 

  * 加强财报阅读的能力，提升排雷能力。 

2022年平安万科农行继续下跌的幅度我认为是极其有限的，因此在2022年股票(包含股息)取得高于2021年的收益率的可能性比较大，因此会继续保持现有风格，可能会进行略微的调整。

可转债的话，2022年几乎不可能取得2021年的收益率，除非再次出现黄金坑，但是这个可能性极小，因此考虑将价格上限调整为110元，当然如果理解了其他的策略，也会考虑采用。

2022年的主要收益来源应该是新股，目前已经将股票分散到多个账户用于打新，按照过去12个月的统计，打新的收益率还是非常可观的，虽然时有破发，但是对整体的影响有限，不知道打新的超额收益还能维持多久，但是至少这一年维持下去我认为可能性是极大的。
