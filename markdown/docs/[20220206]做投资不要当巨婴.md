#  做投资不要当巨婴

赚钱了自己牛逼，亏钱了股市垃圾。

春节前几天的连续大跌，网络上出现了一些很有意思的文章和评论。

如果只是调侃也就罢了，要是当真就有点傻了。

股票市场不应该本来就是波动的吗？

这一点点的波动就哭爹喊娘的，难不成进来股市就想提款？

哭爹喊娘没用的，骂市场也没用的，要骂可以骂自己，差劲的不是市场，而是自己。

做投资，不要当巨婴

看到涨了就发帖问能追吗，浮亏了就发帖问要割吗，这要是能问到答案赚钱那就见了鬼了。

什么内资不行，什么大A被外资抄底，大A被外资割韭菜，不要整天把目光盯着这些。

多想想自己为什么会亏钱，水平不行就想办法提高知识水平，心态不行就磨炼心态，虽然人类在本能上是不适应证券市场的(见之前的文章
《为什么股票赚钱那么难》链接在文末)。

不管进场投资是想赚一点零花钱也好，想赚一台车也好，想赚一个首付也好，这些都没有问题(虽然抱着这种期盼的人大概率亏钱)。

但是请为自己的决策负责，多从自身找原因，做一个成熟的投资者。

我认识和接触到的赚到钱的投资者，都有遭遇市值回撤浮亏的时候

我自己去年也在一个月内从浮盈7%跌到浮亏12%，今年1月份也从浮盈5%跌到浮亏0.5%

这些都是投资过程中必定会出现的，长期下来赚到钱的投资者在回撤时心情不佳的有，不在意的也有(甚至更多)，但是没有一个怪别人怪市场的。

早日摆脱巨婴思维，早日为自己的决策负责，早日走上盈利之路。
