#  让上市公司给我们打工

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

上市公司2021年的年报陆陆续续发布了，同时公布的还有分红方案

看了一下手里持有的上市公司，大部分分红都比上一年有所提升，甚感欣慰。

经过这几年的持股，这种上市公司给我打工的感觉越来越明显了。

这是一种幻觉吗？显然不是

比如，我随便举几个例子(利益相关，我都少量持有，不构成投资建议)

建发股份 2020年每股分红5毛，2021年每股分红6毛

威孚高科 2020年每股分红1.5元，2021年每股分红1.6元

兴业银行 2020年每股分红0.802元，2021年每股分红1.035元

中文传媒 2020年每股分红0.55元，2021年每股分红0.68元

等等

当然也有持平和降低的

总体来说，只要国家的经济在增长，上市公司的整体利润也在增长，那么分红也是增长趋势。

作为多家上市公司的股东(人话：散户)，这些上市公司工作了一年，到了每年年报发布后的几个月，就开始陆陆续续的给我发钱。

作为一个曾经的打工仔，我的职业发展并不算成功，并且我也很清楚我30多岁以后失业的概率越来越大，那些30多岁成功突破天花板进入下一个新的发展阶段以及成功上岸体制内的人，他们是幸福的，也是幸运的，我祝福他们

但是，一方面我的职业发展想要有所突破，难度不小，而上岸体制内，也并不容易

不过好在从小生活一直很简朴，即使这些年工作收入有了明显的提升，家庭的开销也没有显著的增加，于是略有积蓄

而在更早的几年前我就已经开始为应对失业做准备，又或者说我希望能在40岁以后过上自由的生活而不必为了赚生活费而被迫打工，那么学习投资就是其中的一项重要工作。

好在我从一开始就走在正确的方向上，并且通过大量的阅读别人的弯路和踩过的坑，帮我避免了很多需要交的学费。

而今年的行情是如此的差，以致于股息在6%左右的股票大量出现，既然这些筹码现在打折促销，那么我要做的就是筛选，然后陆陆续续的收集，这样明年我将获得更多的分红。

在可以预见的几年后，我持有的股票每年能带给我的分红将会超过十万，说多不多，说少也不能算太少。

而每年的分红达到30W，50W甚至更多，也只不过是时间问题。

当然轮动、套利和打新，还能额外获得一些收益，比如周五上市的纳芯微

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfKJIicsI8uylqzeLA8zbb0PBkGPFZxJsNITQqULSgSn8hmibpRDhichQic8p7zibbia8g1ChWKfIyicia1AtQ/640?wx_fmt=jpeg)

当然，这些都是市场送的甜点，而正餐还是这些上市公司经营带来的回报

不过如果哪天市场火热了，上市公司的估值提升了，市场一次性送来了好几年的正餐，那我肯定也会毫不客气的收下，这部分看上去是市场(对手盘)送给我的，而本质上是市场(对手盘)将这些企业本应该在未来几年给我的回报提前给了我而已。
