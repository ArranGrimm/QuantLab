#  什么是价值以及一个事物为什么具有价值

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

前几天和一个朋友聊天，内容是关于货币、价值、以及为什么一个事物具有价值等等。

聊的内容略微有点深入，并且含有一些逻辑。

能进行如此有深度的讨论是令人愉快的

具体内容就不展开了，这里主要想写一下这个过程中涉及到的我之前阅读过的内容和我想到的一些问题。

首先

**什么是价值**

百度百科显示有三条

第一条是哲学概念：是指客体能够满足主体需要的效益关系，是表示客体的属性和功能与主体需要间的一种效用、效益或效应关系的哲学范畴

第二条是经济学概念：价值，泛指客体对于主体表现出来的积极意义和有用性。可视为是能够公正且适当反映商品、服务或金钱等值的总额。

第三条是汉语词汇：意思是泛称物品的价格

哲学层面的就跳过，作为投资者，主要关注的是第二条以及物品的价格

那么先来一个问题，一个股票的价值从何而来？

价值投资者一般会说，股票就是企业的股权，买入股票就是买入企业的一部分，这体现了股票的价值。

同时股票可以在当下或者未来给投资者带来现金流，这也是股票价值的体现，这个在我之前的《股票的清算价值经营价值和波动价值》中写过(链接在文末)

再来一个问题，期货期权的价值从何而来？

期货起源于实业，比如一个工厂每年需要采购原料，在期货出现之前，工厂每一年的采购价格可能差异很大，会对工厂的生产造成显著的影响，而期货的出现，可以让工厂在合适的时候用合适的价格将未来需要采购的原料的价格确定下来，把重心放在生产经营本身。期货的出现将原材料价格波动的风险在企业主(需求方)和原料商(供应方)之间通过合约化解，给双方提供了未来交易价格的选择，这个就是价值所在。

当然随着期货的发展，市场上出现了纯粹的期货交易者，而这个交易发展到证券市场，标的物由原料变成了证券，就产生了期权，期权同样给交易双方提供了未来交易价格的选择。

然后一个很有趣的问题出现了

假设某个股票背后的企业价值为0，然后这个股票依然在证券市场上交易，每天的价格上下波动，这个企业或者说这个股票，有没有价值？

按照价值投资者的思路来说，这个股票是没有价值的，因为背后的企业价值为0

那么现在假设这个股票有一个这样的期权：期权约定在未来3年内每张期权可以用1元买入100股这个股票，期权的价格为1元。

这意味着某个投资者花1元买入一份这样的期权，未来3年内某一天这个股票的价格只要高于1.01元，这个投资者将可用100元买入100股，然后用市场价卖出得到大于101元，高出的部分就是投资者的收益(实际情况是行权的话会直接结算这部分收益到投资者的账户)

这就很有趣了，一个价值为0的股票，其衍生品却具有价值。

而关于钞票，《漫步华尔街》第12版里面有这样一段话：

> 一张美元钞票，其本身并没有任何内在价值。所有纸币都会遭受各种程度的怀疑，尽管纸币通常并不被人贬低为一种庞氏骗局。

大家习以为常的，认为代表了财富的钞票，本身没有什么价值，是它背后的信用带给它价值，而这个价值也并不是永恒的，比如国民zf的金圆券最终就没有价值，一张100万亿的津巴布韦钞票价值也约等于没有。

价值一直在变化，价值的范畴也很广。

比如在拿破仑时代，金属铝比金银更贵重。

小河边普普通通的一块鹅卵石几乎没有价值，而形状好花色精美的雨花石则比普通鹅卵石的价值要大的多。

如果继续深入思考，还有很多很多。

网上有一句非常流行的话是这样的：

**投资是认知的变现。**

那么，多阅读，多思考，提高认知促进投资。
