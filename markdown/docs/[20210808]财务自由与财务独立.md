#  财务自由与财务独立

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

财务自由不能解决所有问题，但是财务独立可以解决大部分问题。

每当说起财务自由，总有人说，500W不够，在北上深买一套房子都不够，1000W不够，在北上深买一套房子都不够。

那我就想问了，有多少人敢拍着胸脯说自己在退休(失业前)能赚到500W？

一说起财务自由就是500W不够1000W不够，一说起买房子就是不吃不喝多少年才够首付。

如果抱着这样的思想，那就永远只能在抱怨中加班，在生活的奔波中失业。

精神自由是目的，财务自由是手段。

精神不自由，多少钱都没用。

与其把目光盯着如同地平线那样的财务自由，不如把目光放在如山坡那样的财务独立。

在我们的视角里面，可以看到一些人已经在地平线，但是当我们到了那个地方，会发现地平线依然在遥远的前方。

财务自由是没有上限的，而财务独立是有上限的，就像一座山。

有的人的山高一点，有的人的山矮一点，虽然会有差别，但是都需要勤勉的攀登才能到达目的地。

这个目的地，就是被动收入覆盖生活开销，或者被动收入持平或者超过工作收入。

说500W，1000W不够啥啥啥的，不能解决什么问题的，其实是有逻辑错误的。

错误之处就是认为财务自由能解决所有问题，但是事实上这并不现实。

在病痛和生死面前，多少钱都无济于事，虽然可能有些疾病可能几百万能搞定，但问题是如果上班都赚不到500W，同时又说500W不能自由，这就显得很滑稽。

上班拿工资都不能解决的问题，还指望不上班能解决？

不如实际一点，把目标换成财务独立，解决本来需要上班才能解决的问题，把上班从生活的必备项变成可选项。

很多人应该有过这样的经历，非常饿的时候觉得自己能吃多少多少，但实际上吃了一两碗以后，没那么饿了，就会觉得还需要一点就能吃饱，等到吃饱了，就觉得之前觉得自己能吃多少多少显得非常可笑。

那些想着要三千万五千万才能财务自由的人，和饿肚子的情况是非常相似的。

真正靠工作积攒、靠投资积累了上百万的可投资资产(自住房产不算)的人，就像吃的半饱的人，再吃一些也行，就此打住也不是太要紧，毕竟已经半饱不饿了。

财务独立固然不足以支撑富裕的生活水平，但是能让自己免于失业的焦虑，还能促进身心健康。

有多少人早上上班是开开心心去的，而不是不得不去的？

有多少人在假期觉得很开心，希望每年能多一些假？

如果工作是一种负担，那么财务独立可以替你背负这份负担。

这个也是FIRE的核心理念：Financial Independence, Retire Early 经济独立，提前退休。

想要抖音那种豪华生活水准的财务自由(其实很多是假的)，很难。

想要身心健康，免于为生活奔波的财务独立，则是切实可行的。
