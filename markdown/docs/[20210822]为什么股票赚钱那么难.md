#  为什么股票赚钱那么难

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

因为人的大脑其实不适应证券市场。

其实不只是不适应证券市场，也不适应现代社会。

人类吃饱饭的历史很短，全世界范围内看最多大几百年的历史，对于中国甚至不足一百年，而饿肚子的历史却有几百万年。

几百万年来的生活经历和环境形成的习惯和脑回路，不适合现代社会，更不适合证券市场。

我们对高热量食物的偏爱就是其中一种表现，因为碰到高热量的食物尽可能的多吃，几百万年来都是对的，然而我们现在的社会食物太充足了，这种行为模式不再是正确的。

我们总是倾向于短期回报，不愿意做长期规划，因为几百万年来，我们的祖先每次出门干活(狩猎)，都有比较大的概率不能活着回来。直到农业社会，才开始了春生夏长，秋收冬藏。然而这样的历史也只有短短的几千年。

我们在证券市场的追涨杀跌、过度乐观、过度悲观，都是源于我们的祖先几百万年来形成的习惯，已经刻在我们的基因之中。

当我们的祖先组队出门发现了猎物，大家一起去追，追上来逮着了，大家回去饱餐一顿，谁要是中途跑路，这肉就吃不上了，要是最后猎物跑了，大家也只不过白跑一趟，没啥损失。

当一群散户发现一个股票开始涨了，大家一起去追，追上高位买了，大家都站岗亏钱了，谁要是中途跑路，这肉就他吃上了，要是最后对手跑了，大家就都亏钱了，损失很大。

看到没有，同样的行为模式，结局完全相反。

再来看看另外一种场景

在完成一次打猎后，大家都拎着战利品(兔子、分割的猪腿啥的)返程，突然听到了老虎的叫声，大家惊慌失措，纷纷扔掉手里的肉，拔腿就跑。扔掉肉，只不过这次没有赚到而已，并不会亏，如果不扔掉，可能连命都没有。

在完成一次建仓后，大家都看着浮盈(单个股票、或者账户整体)畅想，突然看到了市场的利空，大家惊慌失措，纷纷割掉手里的票，拔腿就跑。割掉票，是真的会造成亏损，会出现实际损失，如果不割掉，可能还会涨回来。

看到没有，同样的行为模式，结局再次完全相反，当然这里需要学会区分鬼故事会不会造成实际的负面影响。

所以祖先留给我们的经验智慧和思维习惯，在几百万年来，都是对的，直到我们进入了现代社会。

所以，为什么股票赚钱那么难。顶着一颗原始人的大脑，在高度抽象的金融市场，赚钱当然难了。

那么有没有什么方法解决这个问题？

当然有

一方面形成自己的投资系统，用系统驱动交易而不是用情绪驱动交易，这些需要学习和积累。

另一方面，阅读一些心理学和脑神经科学的书籍，认识大脑的缺陷，用自己的大脑控制自己的大脑，约束大脑的原始思维。

遗憾的是，真正这样做的人很少，而顶着一颗原始大脑妄图在证券市场中狩猎的人却很多。

所以想在股票上赚钱，对大多数人来说，很难。

除非让自己从大部分人中脱离出来
