#  回购注销约等于现金分红

几个月之前，伊利 和 中国石化 公告说要回购股份并注销

而回购注销约等于现金分红，不过这个逻辑可能很多人并不太理解

先说一下常规的现金分红

假设一个企业总股本是10亿股，年初的总净资产是100亿，经过一年的经营，获得了20亿的利润并且公司决定将10亿用于分红，也就是每股分红1元，另外10亿增加到公司账上计为净资产。

分红对应的两种情况

如果年初的股价是5元(对应每股净资产10元，PB为0.5)

那么分红后如果市场给的估值不变，则股价会平衡在5.5元附近(对应每股净资产11元，PB为0.5)，同时账上获得1元现金，市值6.5元，回报率30%(6.5/5-1)

如果年初的股价是15元(对应每股净资产10元，PB为1.5)

那么分红后如果市场给的估值不变，则股价会平衡在16.5元附近(对应每股净资产11元，PB为1.5)，同时账上获得1元现金，市值17.5元，回报率16.67%(17.5/15-1)

而如果是用10亿来回购股票并注销，对应的两种情况

5.5元的价格，10亿可以买入1.818亿股，注销后总股本降低为8.182亿股，每股净资产13.44元(110亿/8.182亿股)，如果市场给的估值不变，则股价会平衡在6.72元(对应PB0.5)，回报率34.4%(6.72/5-1)，如果是按6.5的价格回购，10亿可以买入1.538亿股，注销后总股本降低为8.462亿股，每股净资产13元(110亿/8.462亿股)，如果市场给的估值不变，则股价会平衡在6.5元(对应PB0.5)，回报率30%(6.5/5-1)

16.5元的价格，10亿可以买入0.606亿股，注销后总股本降低为9.394亿股，每股净资产11.71元(110亿/9.394亿股)，如果市场给的估值不变，则股价会平衡在17.56元
(对应PB1.5)
，回报率17.06%(17.56/15-1)，如果是按17.5元的价格回购，10亿可以买入0.571亿股，注销后总股本降低为9.429亿股，每股净资产11.66元(110亿/9.429亿股)，如果市场给的估值不变，则股价会平衡在17.49元(对应PB1.5)，回报率16.6%(17.49/15-1)

可以看到在市场给的估值不变的情况下，企业进行分红，或者是用现金进行回购并注销两种情况，对于投资者来说获得的理论回报基本上是一致的(回购注销带来的理论收益率比直接分红略微高一点)。

并且我们可以看到，在同样的ROE水平上，低PB的股票进行分红后投资者获得的回报将远大于高PB的股票

有兴趣可以计算一下，如果ROE是10%的话，回购注销后的理论收益率分别是多少
