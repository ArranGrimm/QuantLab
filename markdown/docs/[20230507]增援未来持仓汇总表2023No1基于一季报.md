目前股票持仓的加权平均PB 0.83，加权平均PE 8.18，加权平均ROE 10.2%，加权平均股息 4.78%

主要变化

银行的占比升高到27%，主要原因是股价上涨次要是小幅加仓，国有5大行年初至今都上涨了20%左右，其中最离谱的是 中国银行，涨了29%

大幅加仓金地集团，目前金地的持股数量接近年初的3倍  

清仓了 中文传媒、南方传媒、长江传媒  

清仓了 千金药业

高位清仓了 威孚高科，低位接了一部分回来  

减仓了一部分 大秦铁路、中国石化、中国建筑  

新增了一部分破净的钢铁股  

由于减仓的部分股息率较高，叠加金地和其他部分股票减少分红，目前股票组合的综合股息回报为4.78%(如果当前持仓的股票分红都和上一年一样，则综合股息回报为5.56%)

清仓和减仓的股票，都收获了较大的涨幅收益，不过实话实说，传媒全部卖飞了。

##  收益情况

年初至今持仓股票获得了一定的涨幅同时减持了一部分，目前股票持仓市值接近年初水平，转债部分除开兴业转债外略有增加

年初至今  沪深300(510300)加上分红涨幅为3.45%，中证红利(SH000922)涨幅为12.71%

我的整体收益率是10.47%，大幅跑赢沪深300，小幅跑输中证红利，主要是作为现金替代的兴业转债占据了较大份额，如果去掉这部分，则收益率接近中证红利。

接下来就是等各家公司给我分红了，分红以后填权不填权都可以接受，然后就是阅读财报，筛选股票和转债，继续加仓增加仓位。
