#  柚子割韭菜的时候散户可以躲开，甚至还能顺势赚点钱

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

周二早上(2023年9月5号)，我正在看书，看了一眼手机，发现可转债组合中的 瀛通转债
大涨到130的位置，至于原因我也不清楚，当然我也不关心，后来才看到大概是因为华为概念。

本来我挂的条件单是价格高于132回落3%后卖出，然而当时才早上10点半，它的成交额已经高达大约30亿，而它的总规模才3亿，这种巨量成交通常会伴随着柚子割韭菜

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI0RK4wDKTgkfLUh26G16VWx4vcKYCf93Eyb1oIleA56rbZorU1eRjCU2JGd4vib3NkoQG4NfphiaJQ/640?wx_fmt=jpeg)

转债大涨是由正股瀛通通讯带来的

正股前一天(周一)放量涨停，换手率13.72%，而平时的换手率才2%左右

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI0RK4wDKTgkfLUh26G16VWXJL2yj6JRehfhN7DT9jT7ZHmAUiaiamPpeL0okmuTeX7cianjr0xZFXNg/640?wx_fmt=jpeg)

而周二的涨停板，卖单非常少，观察了一下，一分钟才几手买单而已，因此全天的成交量也很低，换手率只有3.69%

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI0RK4wDKTgkfLUh26G16VWqQAqeJUekqxM3Nq3WhsWF9GdxJribXic9aKMDqyJIROIYdTL8lIuC2OA/640?wx_fmt=jpeg)

通常来说，第二个板成交量很小的情况下，下一个交易日要么继续封板，要么封不住板同时成交量急剧增加

那么对于我来说，问题就是要不要卖出转债。

如果下一个交易日继续封板，那么可转债存在继续上涨的可能，但是130左右的瀛通转债溢价率已经接近100%了，如果第二天正股涨停，转债不涨的情况下，溢价率依然高达90%左右

作为一个保守的，基于价值的投资者，我决定卖出转债，于是挂单129.8卖出了全部的瀛通转债，然后就看书去了。

半个小时以后，瀛通转债从最高138.5回落，我的条件单触发了，挂了一个134的卖单，但是我已经手动卖出了，条件单提示委托超出数量

嗯，开小差手动加戏失败，少赚了几百块。

于是我开始思考柚子会如何操作，我的设想是这样的

第二天柚子将正股拉涨停，然后趁转债冲高的时候卖出转债，再把涨停板的单撤掉，留下有缘人在可转债和正股的高地站岗。

然而柚子显然要比我高明  得  多。

周二当天转债冲到138.5后就回落，收盘时  的  价格只有120.868

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI0RK4wDKTgkfLUh26G16VWKbpjrSaFFODhtKgzwc5h5BgHEW1u3c6BDXibiakPeybtS6fZcAibA6C9w/640?wx_fmt=jpeg)

正股涨停收盘，价格14.27

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI0RK4wDKTgkfLUh26G16VWiaalV62jiaMuyYfFItlaamiaD1vOQoLzTvE139hMib5pwqFySib1kbQcNMw/640?wx_fmt=jpeg)

而周三的时候，正股高开6.5%，最高触碰涨停板15.7元，随后就回落，全天换手率高达42.59%，收盘14.41

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI0RK4wDKTgkfLUh26G16VWrfwkxZibnWibgib8Xia7ndQ3lue89Tib0w7kcps43ecZcun5f6WHt6pp5QA/640?wx_fmt=jpeg)

而转债则从当天最高124.2回落到114左右，收盘114.46，成交额只有11亿

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI0RK4wDKTgkfLUh26G16VWZDFquxffvv9iaQOoQu5sncHgKadzBJ9ibsy8SCbVOucc1iaW5aCV0c5zw/640?wx_fmt=jpeg)

周四

转债在114左右徘徊，成交额只有1.91亿

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI0RK4wDKTgkfLUh26G16VWRwgUke5nLbttiaZZWfd8gibvXmHRQ5eUCrmjw6TEopFt3sd6kdS7eKQw/640?wx_fmt=jpeg)

而正股最终跌停，换手率22.25%

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfI0RK4wDKTgkfLUh26G16VWAbr1aLowYUqvv35yg5PiaTlXib6YK4y5fcIYETzXKoBUoocicxic8poO7A/640?wx_fmt=jpeg)

我原本以为，柚子会周三  在可转债上完成收割，没想到他们在周二就完成了可转债的收割，周三又在正股上完成了第二轮收割，然后留下站岗的散户相互伤害。  

这个手段颇为高明

而作为一个保守的，基于价值的投资者，我没有被收割，甚至还顺势赚了一点钱

我的瀛通转债是2021年1月，可转债被市场砸出黄金坑的时候买的，买入均价93.75

我上周发的持仓汇总表里面可转债持仓汇总表的第10行就是瀛通转债，有兴趣可以看一下，《  [ 增援未来 持仓汇总表2023-No.2(基于中报)
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247485388&idx=1&sn=6d514ea3ae069bff002f53867c1c1e39&chksm=cf17b646f8603f50628c1f56ec0d1869515c69b6f84bfcf9dfa19cf4d8bf58d9a8b55aab84ce&scene=21#wechat_redirect)
》，  童叟无欺。

两年半了，终于在柚子大爷的宠幸下，让我赚了接近40%

虽然这个收益率也算不上高，但是一笔一笔这样平平无奇的收益积累下来，还是非常可观的。

柚子大爷割韭菜的时候，手里没货的话就躲开，手里有货就顺势赚一点，最好不要冲上去PK，虎口夺食。

作为一个散户，正确  地  认识自己，给自己一个合理的定位，才能在市场长久  地  活下去。
