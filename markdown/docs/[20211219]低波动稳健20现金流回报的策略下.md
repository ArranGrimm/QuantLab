#  低波动稳健20%现金流回报的策略(下)

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

接上一篇《 [ 低波动稳健20%现金流回报的策略(上)
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484489&idx=1&sn=45f23656d26ada1a7269c02229fd70d1&chksm=cf17b5c3f8603cd57a1b4af409805c4caf5564044974579fb611b3d838a0ded9bff05b91b2b3&scene=21#wechat_redirect)

上一篇写到，情况肯定不会这么理想，这个是肯定的。

以下内容提及的股票均为举例，不构成投资建议。

动员全家开足6个账号，已经非常不容易了，如果起步是100W，策略运行两年后理论上会有大约140W，平均一个账户23.3W，打新的理论回报已经降低到12.x%，加上股息回报，一年的现金流大概在18%，这个时候就需要继续想方法了。

要么继续增加证券账户，在合法合规的情况下，可以动员小舅子小姨子等等开户，然后"指导"他们买入门票，教会他们打新股，中签后借钱给他们缴款，卖出后赚的钱怎么处理，就看本事了。

如果投资者不能获得家族成员的认可，这个策略基本上就到此为止了，守着100W的本金，每年20W的现金流，也不错了，也可以在高估的时候分批减仓，保持市值和现金流收益的同时还能赚点票价，如果不高估了，再逐步加回来，被动实现高抛低吸。

还有一种方法可以合法合规的增加账户数，但是对绝大多数上班族不适用，就不展开写了。

接下来的增强策略，难度要比之前吃息打新的躺赚策略稍微高出一点。

核心就是 等待系统性机会、市场的错误报价以及顺势轮动

比如今年的系统性机会就是可转债

年初可转债的黄金坑，我投入的仓位大概是股票的1/4，到目前收益率大约为40%，其中一部分达到卖出标准后已经出清，已经落袋的利润大概有25%，这一部分也成了现金流，那么综合起来的现金流回报就是
18%*0.8+25%*0.2=19.4%，剩下的可转债可以补充下一年的现金流。

然后就是市场给出错误报价的时候，比如我在《 [ 股票跌出破产价怎么办？
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484294&idx=1&sn=f38fcb99b0c5d982ef64a71c7e395eb8&chksm=cf17b20cf8603b1a54ff178f6ca5262daf545fccb48667cc923b16ed9368b0081429661a401a&scene=21#wechat_redirect)
》中写到的9块多的保利发展，即使在10块多用5%的仓位增持保利发展，按14.5元卖出增持的部分，也能够给全仓带来 40%*5%=2%的额外收益补充现金流。

再来看一个顺势轮动的例子，这个我在《 [ 胜率、赔率以及价值投资
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484372&idx=1&sn=cfce52fd7ed957c435a3891ccaa30202&chksm=cf17b25ef8603b4867eb8586056711689d93f0899500c399be19aa534141acbc3d832543765c&scene=21#wechat_redirect)
》中写到的兖州煤业(现在叫 兖矿能源)，从8块多一路吃到35.6元，卖出利润也给持仓带来了一些现金流。

兖矿能源的例子看起来是一个偶然，但是对于分散的持仓组合来说，这种偶然出现的概率是非常高的，因为市场总是在波动，情绪或者资金来了就会出现这样的情况。

比如我持仓中提到最多的3个门票，伟星股份重庆水务千金药业。

其中重庆水务和千金药业在《 [ 增援未来 持仓汇总表2021-No.3(基于三季报)
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484461&idx=1&sn=d7b889d28a7c0a618be1b8056e817182&chksm=cf17b5a7f8603cb102616368cb72539c24ab50c82fce9152b97b960e77b7f6cce1726d42c0f5&scene=21#wechat_redirect)
》中显示持仓占比分别是4.79%和4.58%，当时的价格分别是6.23和8.66，而我加仓的价格分别是5.57和8.65，这个在《 [
投资总仓位加到了100W
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484412&idx=1&sn=2cd38e82b04c7327dd16d81964ecd405&chksm=cf17b276f8603b6035f960c5c7b645e3670b13216831cba9ea542c9a13324a042835aafc3f2d&scene=21#wechat_redirect)
》中有写到，而目前的价格分别是6.61和10.59(本周五)，我已经减持了一部分重庆水务，千金药业看情况再决定。

另外一个伟星股份，我在《 [ 股性与牌友
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484425&idx=1&sn=eaf37e9954d56eacb30d8cbde9404e52&chksm=cf17b583f8603c95cfbb4f9951629c60d50dc2cde0242da9d3f96975a990318655ed403ab7d4&scene=21#wechat_redirect)
》中有写到，我已经持有比较长时间了，并一直在来回做T，目前从年初的6.82(不复权)，涨到现在的13.1(本周五)，12月10号我在13.39元的位置减持了一部分。

兖矿能源涨几倍、伟星股份翻倍、以及重庆水务和千金药业大涨的例子，并不能说明我选股厉害，而是说明市场的波动是始终存在而又无法预测的，很多人希望在有明显的信号后才追进去，但是我感觉这个难度非常高。

但是构建一个分散的低估组合，将被风刮到的轮动出去，换入新的或者增加旧的，在不考虑组合整体盈亏的情况下，用差价增加一部分现金流还是大概率能做到的。

如果牛市一直不来，那么守着低波动的组合用20%的现金流维持日常开销或者增加筹码，并不是太困难，而一旦牛市来了，还能顺势将低价的筹码分批高价还给市场，岂不美哉。
