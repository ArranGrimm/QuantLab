#  散户也有机会割机构的肉-以某企业的债券为例

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

今年(2020年)9月的时候，网上流传出一份某地(恒)产(大)公司的文件，大意是说资金紧张，恳请批准重组，不然资金链断裂会引起金融风险云云。

这个文件引起了这个地产公司股票的大幅下跌，同时这个企业发行的债券也大幅下跌。后来我知道有一个资深的个人投资人在这次债券下跌的过程中成功的割了机构的肉，一起来看一看是如何做到的。

这个企业在A股交易的公募债券有两个，其中155406回售的时间更近，9月26日155406的收盘全价是76.456元(含息)，按2021年5月6日回售扣除利息税后可以得到100+6.27
*  (1-20%)=105.016元，收益率为37.35%，周期为7个多月。

等等，为什么会有这么高的收益率，是不是有什么地方不对？是的，确实有个地方不对，那就是这个债券有违约的风险。

说到违约暴雷，很多人第一反应是钱拿不回来了。其实那些都是野鸡平台的故事，这种正经证券市场的违约很少到那个程度，比较常见的是：

  * 延期兑付，也就是到期了没有兑付过一段时间再兑付。 

  * 本金兑付，利息不兑付。 

  * 本金兑付一部分。 

那么问题来了，要是真的暴雷了怎么办？来看一下前车之鉴，康美债。

去年康美药业爆出财务造假，账面上300亿说没了就没了，引起了巨大的反响，康美药业一路跌，而15康美债也是暴跌。那这个康美债最后怎么处理了？来看一下《康美药业股份有限公司关于“15康美债”公司债券2020年第一次债券持有人会议决议的公告》

> 利息全额支付、面额50万（含）以内部分的本金全额兑付、面额50万以上部分的本金按25.666%的比例支付

也就是说持有康美债连本带息在50W以内的话资金是没有损失的。

所以回到这个债券的问题，到这里有没有看出什么特别的地方？这个特别之处在于，如果债券顺利兑付，那么能获得37.35%的收益，而如果暴雷后处理方式和康美债一样，只要投入总金额在50W以内，也能获得37.35%的收益。损失的全是大户，而这些大户多半是机构投资者，也就是说损失被机构承担了，收益被个人拿走了，散户割了机构的肉。另外，传闻出来以后的下跌可能是有部分散户恐慌抛售引起下跌引发机构风控触发止损进一步下跌，那么抄底的散户吃到的就是机构割的肉。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKlxcScxEOqCtj1XjRglXbJHGMDXgMkBFjGRaM31Ihu2ub1p4fkDZfoYVndTELbzYVI6KuZVcapLA/640?wx_fmt=png)

这里关键的地方在于，本息不要超过50W，另外如果总仓位在50W以内的投资人不要梭哈，万一出了一个黑天鹅呢？不过实际情况比预想的要好很多，155406在11月10号达到了90元以上(不含息)，也就是说2个月不到的时间获得了约18%的收益，折算年化收益率那是相当的可观。  

这里还有最后一个问题，需要开通债券合格投资者权限才能交易这个债券，开通的要求是500W资产。所以，大家继续加油吧。
