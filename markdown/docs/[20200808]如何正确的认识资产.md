#  如何正确的认识资产

先看一下资产的定义：

> 资产是指由企业过去的交易或事项形成的、由企业拥有或者控制的、预期会给企业带来经济利益的资源。不能带来经济利益的资源不能作为资产，是企业的权利。

上面这个是企业视角，从这个定义来说，黄金不能算资产，在会计上，公司购入的黄金会记为存货。

按照《富爸爸，穷爸爸》里面写到的，给你带来现金流的是资产，这个是个人角度。那么有哪些我们普通人能接触和拥有的资产？

##  汽车

大部分上班族的车并不是资产，因为不能带来现金流。少部分上班族工作需要的车是生产工具，能够因此带来现金流，则可以算资产。所以，摆摊神车肯定是资产。

##  企业的股权  

在民营企业工作的朋友们可能觉得民营企业有这样那样的缺点，不过民营企业也有民营企业的优点。比如一些比较小的民营企业，会给骨干员工股份，然后每年能拿到企业利润分成。拿到企业部分股份的骨干员工则不再是常规意义上的员工而是老板，当然这个老板属性的含量不是100%而是股份份额的比例。这种机会在小型民企相对容易获得，毕竟小企业吸引力不大，上升空间小，愿意去的人不多，有一定能力加上机遇，可能性还是很大的。对于大型民企，想要获得企业的股份则难度大得多，当然也有人在大型企业工作以后去小型民企，这样就回到了刚才说的。

拿到股权的骨干员工获得的利润分成的这一部分，不是劳动所得，而是由股权带来的现金流，这部分股权是真正的资产。即使不劳动，也是能拿到这一部分的收益的，当然前提是这个股份有法律保障而不是凭老板的嘴巴说，比如要在工商注册信息上体现。

##  体制内工作

可能很多人没有意识到，体制内的一份工作也是资产。首先看一下资产的定义：能带来现金流。

工作能带来现金流，这个好像不管什么工作都一样，毕竟需要付出劳动换取报酬。但是注意了，一个人在民营企业工作并且没有股份，在不能再产出工作成果的时候还有没有报酬？显然是没有的。但是一个人在体制内工作，即使生病休息完全没有产生任何工作成果的时候依然能拿到工资，那么在这个时候，体制内工作的资产属性就充分体现了。

《流浪大师病假二十多年，工资照发》

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfImoHGGAUnOT4Z3yLhqE1l1ibF0jU0a5Te6rSb5ateYJ33tswziaZIqtdiboBGTYz0raibPl50mhntoiag/640?wx_fmt=jpeg)

其实可以稍微扩展一下思路，所有在体制内工作的人，都是拥有股份的。所有的体制内工作都属于国家，拥有这个工作的人就拥有了国家股份的一部分，而且是终生制的，当然要是违纪被开除或者自己离职，就相当于把这部分股权交还给国家了，这样是不是就好理解了。  

##  股票基金

不少人买股票基金要么是看到别人赚钱了，要么是听说可以赚钱，要么是觉得不能落伍。然而很多刚开始投入的人并没有理解股票基金。这里基金指的是股票型基金而不是货币型基金。

很多人觉得股票要涨了，赶紧买，股票要跌了赶紧卖，等等。这个是从股票的交易层面理解和操作的。

股票既是股又是票。

股，就是股权，上面也有提到，股权是能带来收益的。

票，就是票据，是拥有股权的凭证，是有价证券，是可以交易的，从这个角度，股票和房产证有相似之处。

从票的角度来说，股票的收益来源于低吸高抛，赚的是对手亏损的钱。

从股的角度来说，股票的收益来源于企业利润的分成，体现为股票的分红。另外一个部分就是企业利润留存在企业增厚净资产扩大经营提高利润而带来的股权价值的提升。

##  房产

这个就不必多说了，早年购入的房产，既得到了房价的增值又得到了房租的收入。  有人觉得房贷消耗了现金流，  所以  房产不是资产，这个理解是有偏差的。

房产如果出租则获得了现金流，如果自住，则减少了房租开销。月供的房贷并没有消失，而是用来减少了贷款余额，至于房贷大约5%的利息，简直是国家提供给群众的福利，如果能够全款买房也最好贷款，余下的这一部分资金做理财投资，只要收益率高于5%则能够覆盖利息开销。

当然现在房价已经比较高了，而且不同城市的情况各有不同，以后房产还是不是优质资产就难说了。

##  结语

多积累资产，少背负负债，让资产的雪球滚起来吧。
