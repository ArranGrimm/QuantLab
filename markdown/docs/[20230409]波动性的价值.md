#  波动性的价值

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

波动不是风险，波动蕴含价值

假设有两个企业，过去几年的利润增长都是10%上下(意味着基本可以排除业绩大降的可能)

当前估值也不相上下(意味着当前价格下性价比差不多)

其中股票A在过去几年中的波动较小(假设上下波动大概在5%)，股票B在过去几年中的波动较大(假设上下波动大概20%)

那么，抛开其他因素，哪个股票更值得投资？

我想除了喜欢刺激的投资者，通常来说选择股票A的会更多一些。

因为毕竟长期来看两个股票的年化收益率都在10%左右的水平，而股票A的波动小意味着回撤更小，持股体验更佳。

从这个角度看，确实如此，但是却忽视了一个很重要的因素

那就是， **波动性是有价值的**

我知道很多价值投资者(俗称夹头)是不屑于谈论波动性的，因为谈论波动性就会涉及到交易，而谈论交易就好像不是一个正宗的价投

但是别忘了，价投眼中的偶像巴菲特，并不是不交易，而是不频繁交易

他在估值低投资价值大的时候会买入，在估值高投资价值低的时候会卖出

虽然频率低，周期长，但是本质上还是在利用波动性，并且他不频繁交易，是因为他无法频繁交易

因为他的钱实在是太多了，在他钱没有那么多的年轻时代，他的交易频率要高的多。

另外他的大师兄施洛斯的交易频率就高的多，因为他管理的资金规模很小，在2002年清盘时规模才1.3亿多美金(每年会将利润大比例分红给客户)

所以对于散户来说，合理的、有计划的交易，对投资是有益的

回到前面假设的两个股票

如果是我，我会选择用多个类似B股票这样的标的，构建一个组合

这样的方式，可以通过多个标的涨跌不同步进行对冲，让组合整体运行更平稳，同时可以根据适当的条件做仓内轮动，获取一部分超额收益

原理并不复杂

假设有两个股票B1，B2，当前价格都是10元，各持有1W股，则当前市值是20W

经过一段时间，B1跌了15%，那么卖出700股B2，买入800股B1

则B1有10800股，B2有9300股，现金200元，合计市值18.5W

有经过了一段时间，B1涨到了10.5元，而B2跌到了9元，卖出1400股B1，买入1600股B2

则B1有9400股，B2有10900股，现金500元，合计市值19.73W

经过一年，企业利润增长了10%，在利润的推动下股价上涨了

当两个股票都回到之前的估值时，两者都上涨到了11元，此时的市值是22.38W

而如果没有中间的波动，市值是22W

以上只是一个例子

实际操作中对股票的价值中枢进行估值以及根据条件来轮动比这个要考虑的因素要多

但是本质上没有什么差别，都是通过轮动交易将波动性的价值兑现为收益

事实上我的投资也是这样执行的

比如我持仓中的 千金药业、中文传媒、金地集团

都经历了几轮从9块左右到12块，14块，甚至16块，又回到9块左右的过程。

而在这个过程中，我也根据自己的估值锚，进行仓内轮动，收获了一部分超额收益。

波动性的价值不仅体现在股票上，也体现在转债上。

有些转债的波动就比较大，在一两年内，可以从债性状态(110块左右或更低)，到触碰强赎线状态(130块左右)，到股性妖性状态(150块以上)，来回几次

我持仓的转债中就有几个转债因为这种波动，一条鱼吃了几次

这看上去好像一点也不价投，但是我认为这种方式是可以算价值投资的

因为投机分子买入的理由通常是未来有人出更高的价格接盘，卖出的理由通常是可能会继续跌先止损

而我买入的理由是投资价值很大，即使后面没人高价接盘也不要紧，有新的资金我还要继续买入

卖出的理由是投资价值已经不如其他标的了，即使后面再涨我也不关注

市场先生总是时不时的发神经，给市场带来波动性

我们可以顺从市场先生，将波动性的价值兑现为收益收入囊中。
