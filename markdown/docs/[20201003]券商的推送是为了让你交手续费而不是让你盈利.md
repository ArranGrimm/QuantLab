还有类似什么， 上涨的原因找到了，下跌的原因找到了之类的。

对于还没有形成自己的投资系统的人，这些推送会不断的挑动读者的神经。资本市场的预警怎么看？仓位重的要减仓，我要不要减？半导体的名单先收藏起来没准哪天用的上？北上资金潜伏了要不要跟？北向资金加仓的股票要不要抄作业？小而美的股票名单都给出来了，要不要抄作业？

如果一直关注这些消息，跟着这些消息走，就会整天神经兮兮的，一有消息就要看，生怕遗漏了什么重要消息。而有时候没有什么消息却大涨了，大跌了，那么不久后就会看到类似”上涨/下跌的原因找到了“之类的消息，这个我亲测过，
有一天中午吃饭的时候看了一眼指数，大跌，再看消息推送，果然有一条"下跌的原因找到了"，如果是一直跟消息的投资者，则会马上点开看看，看完后，”哦，原来是这样，哎，早知道应该怎样怎样“。实际上，这些找到的原因可能看上去很有道理，逻辑也没问题，只不过有一个最大的问题，就是无法在事前推送。为什么不能事前推送？嘿嘿，且不说券商有没有能力事前得到准确的消息和逻辑，假如真的能事前知道，会告诉别人，让别人赚钱吗？

所以我们需要想一想，这些资讯到底有什么用？实际上，这些资讯对散户基本没有什么用，如果有用，那谁还上班，直接看消息炒股就可以了。需要明确一点，这些消息都是券商希望我们看到的消息，不希望我们看到的消息，我们一定看不到，毕竟媒体在他们的手里。

所以这些消息，只对券商有用，有什么用呢？只要能鼓动散户交易，券商就能获得佣金，这个就是这些消息的作用，至于散户能不能赚钱，谁在乎？跟着这些消息操作的抄作业行为，基本上是无法盈利的，原因可以参考《
