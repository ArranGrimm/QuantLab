#  熊市应该做些什么

目前是熊市，大部分人今年是亏钱的，那么现在应该做些什么来改善未来的处境。

割肉？

这个显然不太合理，现在割肉止损，等市场回暖了再进来，多半会加钱才能买入。

先卖掉等更低了再买入？

如果真的有这个能力，之前就不会买入导致现在套牢了。

那么现在应该做的主要是4件事

* * *

**  
**

**1 梳理手里的持仓，调仓或者加仓**

将手里的持仓逐个查看财务数据，确认没有明显的问题，再核对一下历史估值区间和目前的估值百分位，确认都处于历史低位附近。

如果有明显问题或估值不在低位的，可以考虑调仓，换成更合适的个股。

不要因为浮亏而不调仓，举个例子，假如持仓的某股票目前浮亏10%，并且之前的买入逻辑已经不再坚实，或者处于历史中高位，那么继续持有，可能大盘涨10%它才涨了5%，因为它处于历史中高位，很可能比市场整体涨的少。

而如果换成其他低估个股，可能大盘涨10%，它就能涨10%甚至20%

市场上的资金总是将一个板块的估值拱上去，然后流入到其他低估的板块，就像前几年的赛道股，这两年都腰斩再腰斩，而低估的个股在大盘没涨的情况下也大涨了，比如农业银行从去年收盘的2.91，本周收盘3.63(2023年10月27号)，加上7月派息0.2222元，收益率高达32%，这要是放在前两年肯定没人相信。

类似的还有大秦铁路，从去年收盘的6.68，本周收盘7.2(  2023年10月27号  )，加上7月派息0.48元，收益率高达14.97%，也是相当可观的。

如果手里现金储备还有富余，也可以筛选后买入估值低潜在回报高的个股，这样能提高未来的收益。

  

**2 攒钱增加现金储备**

把一些大额支出适当延后，增加手里的现金储备，如果在现金充裕前，市场就已经回暖了，那自然是好事，股票赚钱了，手里攒的钱也多了，双倍的开心。

而如果攒钱到手里现金储备充裕了，市场还在低位徘徊，那么就可以适当加仓。

很多人喜欢在看到市场火热的时候冲进去，期望首付进去全款出来，绝大部分情况下都会落空并且亏钱。

而如果在低位的时候进去，假以时日，大概率是可以赚钱的。

不管是手里的现金增加了，还是低位加仓后浮亏比例降低了，都能在一定程度上减少心理压力。

  

**3 看书学习**

查看个股的财务数据，查看个股的估值水平，都需要一定的基础知识，这就离不开学习。

当然会有很多人说，市场不讲道理，看书没用。

但是我不这么认为

在职业发展中，看书学习提高自己的工作能力，当然是有用的，不能说有些人靠运气踩到了好机会或者靠关系升职加薪了就否定努力工作和学习提升的作用。

同样的在投资市场，有混乱的地方，也有符合规律的地方。通过学习，可以识别哪些是混乱的，哪些是规律的，然后在规律的地方，通过知识识别机会进而获利是完全可能的。

比如将手里的持仓合理调仓或者加仓，然后在未来获得回报

另外，一些经典书籍里面也包含了世界主流国家证券市场的历史，里面有历史规律，比如这个

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIMR6Z3lvp21cSHYzrJxN7Nsa1dgcIgg4ZFnRP4QUSkKHjXJEaepdibzp5ESmCgELDibeljwMLgaKtw/640?wx_fmt=jpeg)

主流国家的股票长期回报是扣除通胀后大约在7%上下，这个就是历史规律，即使这个过程中经历了各种危机和战争，最终的结果依然如此。

另外，A股目前的种种乱象和无法理解的涨涨跌跌，在几十年前的华尔街都上演过，A股只不过是在重播而已，太阳底下并没有多少新鲜事

关于这些内容，可以阅读《漫步华尔街》 作者 伯顿 G. 马尔基尔

  

**4 锻炼身体，等待回报**

如果投资的时间超过一定的年限，一定会遇到熊市，也一定会遇到牛市

而如果时间拉的更长，就会遇到很多次熊市和牛市。

这是市场规律的一部分，既然一定会遇到熊市而又不可能每次都躲过，那就坦然面对

在做好前面的3点的基础上，我们应该锻炼身体，保持健康，等待回报。

不然等以后赚到钱了，这也不能吃那也不能吃，玩也玩不动，那赚钱还有啥意义。

  

熊市会持续多久无法预测，能做的大概就是这些，剩下的就是等待。
