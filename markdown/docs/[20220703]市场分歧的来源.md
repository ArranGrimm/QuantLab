市场总是有分歧

有时候分歧的双方，有一方是错的

而有时候分歧的双方都没错，只不过是自身的其他因素不一样

关于分歧的双方一方正确一方错误，这个没有太多值得说的，无非就是错误的一方买单而正确的一方将自己的认知变现。

而双方都没有错的情况下的分歧，更有趣，值得展开说一说。

先用现实生活中的场景来看一看

比如，手机的颜色

水果手机过去几年的新品都会增加新颜色的款，而每次最受欢迎的都是新颜色的款，甚至于在黄牛渠道，特定颜色会出现溢价。

那么对特定的颜色的溢价，在不同的人之间就存在分歧。

不支付溢价的人会觉得，为了颜色支付溢价实在是不值，毕竟手机拿来用，啥颜色不是一样用。

支付溢价的人觉得，我就是要让别人知道我用了新款，就是要和别人不一样，多出来的几百块，用在哪里不是用。

好像双方都没有毛病，但是双方为同一个东西(除了颜色不同)，支付了不同的价格。

再  比如，手机的到手时间

新机发售经常没货，但是可以通过黄牛加价拿到。

不支付加价的人觉得，为了早一个月拿到手机支付加价实在是不值，等一个月有啥不可。

支付加价的人觉得，我就是要让别人知道我用了新款，就是要比别人早一个月，多出来的几百块，用在哪里不是用。

好像双方都没有毛病，但是双方为同一个东西(除了到手时间不同)，支付了不同的价格。

  

再来看看股票

  

假设两个投资者，一个对股票的成长属性更为看重，而另一个对股票的股息回报更为看重。

那么当某一个高息股跌到某一价格的时候

前者会觉得这个价格不值得买入，而后者会觉得这个价格值得买入

而当一个成长股涨到某一价格的时候

前者会觉得这个估值水平依然值得买入，而后者会觉得这个估值水平根本不值得买入

  

假设两个投资者，一个对波动的耐受程度高，而另一个对波动的耐受程度低

那么对于两个股票，同样是长期看向上增长，一个波动更大同时增长也更大，另一个波动小同时增长也相对小。

那么前者会更倾向于买入波动大增长也大的股票，而后者会更倾向于买入波动小增长也小的股票。

  

假设两个投资者，一个是趋势风格，一个是价值风格

当某一个股票最近一段时间没有涉及热门题材也没有资金进入，价格跌了一段时间了，估值相对便宜。

前者可能会止损离场换股，而后者可能会趁机买进

  

类似的分歧还有很多，就不一一列举

分歧带来交易，交易让股价波动，而我们需要考虑的是哪些分歧对我们有利。
