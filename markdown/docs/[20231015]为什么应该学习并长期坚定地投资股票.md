#  为什么应该学习并长期坚定地投资股票

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

几年前有一段时间猪肉涨价，然后有个段子

> 早上去买油条，居然涨价了！
>
> 我就问：“大爷，怎么昨天才1块今天要2块了？”
>
> 大爷说：“因为猪肉涨价了。”
>
> 我就郁闷了：“猪肉涨价关油条什么事？”
>
> 大爷来一句：“因为我要吃猪肉！”
>
> 说得好有道理，我竟无言以对。

  

这个段子反映了两个事实

1 人力资源是抗通胀的，卖油条的大爷依靠卖油条养活自己，随着物价上涨，他的油条也涨价，这样就能维持自己的生活水平。

2 作为一个经营实体，在没有其他因素影响的情况下，营收和利润会随着通胀自然增加，比如这个油条铺的营收和利润就随着通胀增长了。

所以在一个运行正常的经济体里，人力资源是天然抗通胀的。  

只要一个人可以一直工作，那么生活就能一直维持。

那么问题来了，一个人基本不可能一直工作。

抛开失业裁员不说，我想应该没有几个人希望一直工作。

如果我们手里有一个油条铺，又不用我们自己亲自炸油条，那就可以让油条铺一直工作，不仅可以抗通胀，甚至还能提高生活水平，比如卖什么健康油条，利润更多了，就可以用多出来的利润去买高档水果，去旅游，这样生活水平不就提高了吗。  

想开一个经营稳定的油条铺难度很大，而买入一家企业(的一部分股权)来获得类似的回报难度就小  得  多。  

当然，投资股票也有投资股票的问题，比如很多企业上市就是为了大股东套现，市场上很多柚子割韭菜，股票波动大因此每年的回报不稳定，等等

投资股票赚钱并不容易，但是开一个油条铺也不轻松，即使学会了炸出合格油条的手艺，每天还得凌晨就开始准备，天没亮就得开始炸油条卖油条。

即使是有些行当不用早起，比卖油条轻松一些，也只是轻松一些，但是必须自己干活或者守着，想当甩手掌柜并不容易。

而通过投资股票来当甩手掌柜就是天然的，毕竟上市公司也不会听我们指手画脚，我们只需要在合适的时候买入，在合适的时候卖出，剩下的时间坐等分红就行了，经营管理交给公司的管理层就行了，我们只看看不说话。  

唯一需要做的就是掌握基本的投资原理，筛选合适的股票。  

这固然需要学习，但是干啥不需要学习，炸油条也得学习

学会了炸油条还得每天早起炸油条

学会了投资股票后并不需要每天操作股票

有很多懒人策略，基本上一个月或者一个季度调整一次就行了，勤快的每周调整一次，平常只需要稍微看一下甚至不看也行，这不比炸油条轻松得多？

很多人误以为投资股票需要学很多的财务知识，商业知识，经济学知识。  

曾经的我也是这样以为的，直到后来我搞清楚了投资股票收获回报的原理，发现其实并没有那么复杂，如果仅仅是入门并预期获得年化10%的回报，需要学习的东西并没有多少

以我自己的实践经历来看，如果有明确的学习路径，理解基本原理以及学习相关的知识，顺利的话应该
不需要超过100个小时，相当于每天3小时一个月就够了，理解能力强的话可能50个小时就够了，不过如果误入歧途，走进死胡同或者钻进牛角尖，那就不好说了。

掌握了基本的原理和方法，剩下的就是按方法实践和执行了，这将是一个漫长的过程，就像学会开车只需要一个月，但成为老司机至少要几年。

一边不断完善投资组合，一边不断地将收入结余投入其中，让投资组合成为帮自己赚钱的“油条铺”，最终让自己过上既富有又轻松的生活  。
