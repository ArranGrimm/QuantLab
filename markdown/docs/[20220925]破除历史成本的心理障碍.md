#  破除历史成本的心理障碍

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

惦记历史成本是投资的心理障碍，那么如何破除这个心理障碍？

先来看两种情况

第一种情况

10块钱买入A股票1000股，6块再买入1000股

此时持仓成本为8块，账面浮亏4000块，浮亏比例 -4000/16000=-25%

然后当A股票涨到9块时，如果全部卖掉，则盈利2000，盈利比例2000/16000=12.5%

相信很多人都干过类似的事情，在下跌过程中，加仓，然后回升到均摊成本之上以后卖出。

然后又9块钱买入了B股票2000股，涨到10块卖出，盈利2000块

合计盈利4000块

第二种情况

10块钱买入A股票1000股，然后6块再买入C股票1000股

然后一段时间之后A股票也跌到了6块

此时两个股票的综合持仓成本为8块，账面浮亏4000块，浮亏比例 -4000/16000=-25%

然后A股票和C股票都涨到了9块

这个时候C股票浮盈3000块，浮盈比例50%，但是A股票浮亏1000块，浮亏比例-10%

这个时候卖出C股票(但是舍不得卖出A股票)，然后9块买入B股票1000股，涨到10块卖出，盈利1000块

C股票盈利3000块，B股票盈利1000块，A股票浮亏1000块

合计盈利3000块

  

第一种情况卖出A股票大家都能接受，毕竟卖出价高于 ** 均摊成本  **
，而第二种情况下，即使AC两个股票的均摊成本和第一种情况是一样的(都是8块)，但是很多人无法接受卖出浮亏的A股票，最后导致只买入了一部分的C股票，最终盈利比第一种情况少。

这个在心理学上有一个专有名词叫 损失厌恶

**损失厌恶是指人们面对同样数量的收益和损失时，认为损失更加令他们难以忍受。**

这种心理让人只想把亏损的部分扳回来，哪怕因此错过了其他本应该得到的收益。

那么如何破除这个心理？

**计算盈亏时，把不同持仓当成一个整体看待，而不是关注单个股票**

关注整体的盈利，避开关注单个亏损，同时在调仓的时候，对于单个股票不要看盈亏比例，全部当成保本状态

如果在保本状态下，会考虑加仓或者减仓，那么就去执行，而不能根据是否盈亏来决定加仓或者减仓。

  

这里会引发第二个问题，很多人无法接受加仓价格高于持仓成本(或者减仓价格低于持仓成本)。

同样也是分两种情况

情况1

买入时间较早，价格比较低，企业发展良好，业绩驱动股价上涨

比如几年前10元买入了1000股，每股净资产12元，每股收益1.5元，净资产回报率12.5%，每股股息5毛

企业经过了几年的发展，目前每股净资产24元，每股收益3元，净资产回报率12.5%，每股股息1元

现在的1股的净资产、收益、股息，相当于几年前的2股，现在的股票的价值是几年前的2倍(不考虑其他因素)，而现在的股价是18元，那么买不买？

**如果几年前买入的逻辑依然成立，那么就应该买**

因为现在的1股相当于过去的2股，而价格却只有原来的1.8倍，投资价值更高了。

类比一下就相当于，几年前买入一个火锅店需要100W，现在买入一个两倍面积两倍营业额两倍利润的火锅店，只需要180W，买不买？当然要买

当然，前提是需要合理的评估投资价值。

事实上我经常干这样的事情

比如 保利发展，我目前的持仓成本是
-8.29元(没错，是负数)，但是在我的评估体系以及交易计划里面，如果在今年内跌到16块以下，我会再次买入，毫无疑问，买入后我的持仓成本会增加，但是这并没有什么关系，因为这个价格我觉得是值得的，当然也可能继续跌，甚至再一次跌到10块以下也并非不可能，但是很显然按目前的情况，如果它跌到16块以下同样有机会再回到18甚至更高。

再比如 建发股份，我目前的持仓成本是
-4.69(也是负数)，但是如果价格回到11.5附近，我会开始分批买入，同样的，买入后我的持仓成本会增加，而在2020年我建仓的时候，买入的价格是8块上下。这里需要感谢一下8月24号高管被查引起的市场砸跌停提供的加仓机会。

再比如
千金药业，我目前的持仓成本是6.77(占比大约4.3%)，如果价格回到8.8左右，我会继续买入，显然买入后我的持仓成本增加了并且浮盈总额也降低了(因为股价跌了嘛)，而当股价回到现在的位置，我的浮盈总额就增加了(因为股数增加了嘛)，而如果持仓占比低于4%的话，现在这个价位我就会买入。因为按我的评估体系，现在这个价位就值得买入，只不过占比已经接近5%的上限，因此要等待价值更高的买入机会。如果再次出现去年千金藤素引起的非理性大涨，我肯定会分批卖出。

所以可以看到，上面3个例子里面，买入的时候根本就没有考虑当前的持仓成本，而是盯着这个股票目前的价格是否值得买入(或者是否需要卖出)，以及持仓比例是否过高。

当然，我的手里也有持仓成本高于目前价格的股票，也就是账面浮亏的。而对于这些股票，我的评估方式和交易计划的制定，和上面这3个大幅盈利的股票一样，只看是否值得买入以及是否应该卖出，而不关心历史成本和浮亏金额，因为那些是过去的决策造成的结果，不应该和现在的决策产生关联。

当然，也有割肉彻底离场的和盈利清空的，两者的逻辑是一样的，即当前的价格不值得持有，卖出后换入别的股票，至于是亏损离场还是盈利离场，都已经是历史，不应该影响未来的交易计划。

我们在证券账户里面可以看到每个股票的盈亏情况，但是当你把钱从证券账户转出来，你能分得清哪一张是哪个股票赚的吗？
