#  散户其实并不需要恐惧机构量化交易，甚至还能占点便宜

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

本周一(2023年8月28号)在4个王炸利好的情况下，上证指数竞价高开5%，但是最终收盘只涨了1%，不少竞价追进去的人被埋。  

后来有不少人说是量化交易惹的祸，至于是不是，我也不知道，反正早上交易软件也卡了，雪球也崩了，那一天我没有交易，竞价的时候全仓涨了不到十万，然后午觉起来，浮盈就只剩下小几万了，就当早上没有高开，好歹还浮盈了不是。

不过今天又有新段子了

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIBPpF97ukE8QB2CIWStv7dGqwuxvyYBPUpjkVCDaFYVROm9icybT90lAUIpA0dJIaDgScjE1QtHPw/640?wx_fmt=jpeg)

只能说段子编的不咋地，柚子说量化程序没有意识无法判断公司价值，别逗了，柚子割韭菜的时候会在乎公司价值？

段子看看也就图一乐，要是当真那就真的输了。

其实要我说，这些量化程序搞他们的，散户也可以自己玩自己的，想想教员的思路：你打你的，我打我的。

就算量化程序交易下单快，计算快，日内交易有优势，要是我不参与这个游戏，阁下又该如何应对呢？  

如果我采用最简单的策略，始终保留至少20%到30%的现金仓位

对于股票，筛选一批股息回报5%左右同时利润增长5%左右的股票做一个组合，综合预期年化收益就是10%，要是涨了，比如分红5毛的股票从10块涨到12.5元，股息回报降低为4%，我就卖掉换其他的，如果跌到8块，我就加仓，要是不涨不跌，我就躺平不动。

那么阁下又该如何应对呢？

对于转债，筛选一批价格110到115左右，到期收益为正的转债，剔除问题公司，做一个组合，如果涨到130左右或者当日脉冲大涨，我就卖掉换其他的，如果跌了我就再加，如果不涨不跌，我就躺平持有到期。  

那么阁下又该如何应对呢？  

只要我的策略足够简单，躺的足够平，任你是量化交易也好，柚子也好，任你大盘如何上蹿下跳，都割不到我。  

甚至还能在大涨或者大跌的过程中，占一点便宜。  

那些花里胡哨的各种线，各种热点题材分析，各种产业分析，都是浮云，里面各种柚子和量化程序嘎嘎乱杀，艺高人胆大的可以去pk试试，比如我就知道某网友把交易软件设置了各种快捷键，做起日内交易来堪比电竞打比赛，实在是令人叹为观止。

我这种艺低人胆小的就不掺和了，用最简单的策略，用最躺平的心态，把平均年化10%搞到手就可以了，当然碰到市场额外赠送，那自然是不会拒绝的，比如截止到8月结束，我今年的收益率就超过10%了，躺平就完事了。

是不是有点像那个  **重剑无锋，大巧不工？**

至于我的持仓汇总，可以找找我之前发的，也可以等本周天，我会发基于中报更新后的股票和转债持仓汇总表。

令我意外的是，证监会和交易所，竟然也表态了 **
