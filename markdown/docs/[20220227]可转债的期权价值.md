#  可转债的期权价值

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

自从去年1月在可转债的黄金坑创建可转债组合到现在13个月，我的可转债组合取得了接近50%的收益

这里面既有能力的成分，也有运气的成分，能力的成分是对可转债期权价值的初步理解(真的只是初步理解，理解更深刻的投资者在这一轮取得了大约70%到100%的收益)，运气的成分是没想到价值兑现的如此之快。

要理解可转债的期权价值，首先要理解期权

但是市场上的投资者知道股票基金的很多，知道可转债的有一些但是了解可转债的少，而了解期权的就更少了。

期权有看涨和看跌，以及买和卖，由此构成了4种组合，但是要理解可转债的期权价值并不需要这么复杂。

可以用生活中的实际例子来理解(有些细节是为了便于说明，在现实中可能不会存在，不要纠结这些细节)

假设现在有一个楼盘在卖期房6个月后开盘，所有户型朝向面积啥的都一样全部200W一套(不要在意这些细节)，有一个人想买一套，于是他到售楼部和销售签订了一份特别的合同

合同约定他可以在开盘以后到6年内，用200W的价格买入一套房，并为这份合同支付了1W元(不要在意销售会不会签这样的合同)，同时这份合同可以转让，转让后同样有效。

如果开盘后到6年内，这个楼盘的一套房的交易价格在200W以下，那么这个人想要买房就可以直接买了，这份合同并不会起到什么作用。

但是如果开盘后房价在涨，或者有涨的预期，那么这份合同就会体现出价值

假设开盘后，这个楼盘的房价涨到了205W一套，现在这个合同的价值是多少？

对于这个买家而言，他只需要支付200W就可以买入目前市场价205W的房子，对于其他买家而  言
，如果能用少于5W的价格将原买家的合同买过来，然后拿着合同去200W买入这个楼盘的房子，显然是比用205W买入要更划算(忽略交易成本)。

那么现在这个合同的价值可以认为是小于5W，而显然大于1W。

一年后，这个楼盘的房价涨到了210W一套，那么按照前面的推算，这个合同的价值可以认为是小于10W。

然而，真的小于10W吗？

假如两年后，这个楼盘的房价涨到了220W一套，那么只要买入这个合同的价格小于20W都是值得的，基于这种假设，在第一年结束时不同的人会对未来有不同的预期。

最乐观的人会认为3年后房价会涨到230W甚至更高，那么他会愿意现在就花小于30W的价格买入这个合同，到时候要么花200W履行合同买房，或者将合同卖给别人。

那么这个合同，可以认为是一份看涨期权，最初的买家花了1W块买入，而随着房价的上涨(以及上涨的预期)，这个合同的价值迅速增长。

接下来在上面的基础上说可转债的期权价值，相较于上面的例子，可转债提供了更多的保底。

假设某家公司的股票价格是100元，发行了可转债，可转债的发行价格是100元，每张可转债可以在6个月后到6年内兑换一股公司的股票(转股价格100)，并做了如下约定

1 在可转债兑换之前每年给持有人发一点点利息

2
如果在最后两年里，公司的股票在一段时间内(比如连续30个交易日)低于转股价格的一定比例(比如70%，也就是股价跌到70块以下)，持有人可以按原价100块将可转债卖给公司。

3
如果在发行的6个月后，公司的股价在一段时间内(比如连续30个交易日中有15个交易日)低于转股价格的一定比例(比如80%，也就是股价跌到80块)，公司可以将转股价格下修到一定价格(有一定的约束条件)，假设下修到80块，那么一张可转债可以兑换1.25股(1.25股对应的市场价格是100块)。

4
如果公司的股票在一段时间内(比如连续30个交易日中的15个交易日)，高于转股价格一定比例(比如30%，也就是股价涨到130块及以上)，公司可以强制按100块加上当年的利息回收这些可转债，也就是说持有人可以将可转债兑换成价值130元以上的股票并卖出股票获利(当然也可以直接将可转债卖掉，因为这个时候的市场价和能够换到的股票价值差别不大)，如果持有人既不兑换成股票，也不卖出，那么最后被回收了就只能拿到100块加上少量的利息。

由此可以看出，100块的可转债，相当于免费送了一份看涨期权

如果发行转债的公司的股票跌了，持有人要么每年拿利息，6年后拿回本金100块

或者跌到一定的程度，提前拿回100块及利息

或者公司下修了转股价格，可转债又可以兑换100块的股票了

如果发行转债的公司的股票涨了，那么可以按转股价(低于市场价)兑换成股票，持有股票或者卖出，或者直接卖出可转债。

那么天底下真的有这种稳赚不赔的好事？

答案是真的有，只不过没有这么简单。

首先，可转债发行价是100块，需要申购并不是想100块买就能够买到

其次，存在违约的可能，虽然到目前为止还没有出现违约的情况，但是以后一定会出现。

违约的问题，并没有什么问题，因为即使出现违约也不是大面积违约，并且也不会一分钱拿不到，可能只是延迟拿到，或者拿不到利息，或者本金只能拿回一部分。最坏的情况下本金只能拿回一部分的话，如果是同时买入多个转债，那么这个损失是可以由其他转债的收益覆盖的，所以这个风险并不能消除，但是可以消化。

那么如何买到？

很遗憾，虽然了解可转债的投资者很少，但是还是有的

因此100块买到可转债(以及附带的免费看涨期权)的机会非常少，甚至由于预期收益较大，很多投资者会用高于100块甚至高于110块的价格买入转债，更有高级玩家买入130块甚至150块以上的可转债。

当然，也有很多人高位接盘，这个是另外一个话题。

机会少并不代表没有，去年1月份，这样的机会就出现了，只不过机会经常看上去和风险很像，在这样的机会出现的时候，甚至有一些可转债投资者割肉并发帖说受不了了。

在市场上的可转债大面积跌破100块的时候，可转债附带的看涨期权已经不是免费了，已经是倒贴钱了。

倒贴的看涨期权经过一年的时间兑现了很大的价值，不知道当初抛售的人现在作何感想。

最后来看一个例子，感受一下期权的价值。

去年10月8号，我在112.136的价格买入了洪涛转债，彼时距离到期还有9个多月，如果到期赎回，我将亏损大约3.7%(忽略利息税)，而如果到期之前股票大涨，我将会获利

也就是说我为这个可转债的看涨期权支付了大约4元，2月23号伴随着洪涛股份涨停，洪涛转债大涨
