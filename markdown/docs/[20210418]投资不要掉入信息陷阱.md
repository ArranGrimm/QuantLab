#  投资不要掉入信息陷阱

这些都不是今天要说的重点，因为我既不持有顺丰控股，也没有了解顺丰控股的财务数据，既不看多也不看空，主要是前几天看到这样一段文字

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfLRg2fmeymPjLzEUJccpvFuXF5EafexiaOF7VThBQvtbd3U3VzSXojuvkJzsQ7bKZsTibF7tL82Bib5g/640?wx_fmt=png)

当我看到最后一句"时代抛弃你往往是毫不留情的"的时候，我笑了。

这种句式太熟悉了，比如：

《大润发被收购的背后，是时代毫不留情的抛弃！》

《数字经济全面来临:时代抛弃你,连招呼都不打》

《你的同龄人,正在抛弃你》

等等。

这样的文章，标题挑拨情绪，引起读者的注意，给文章带来流量，作者倒是获得了收益，但是读者得到了什么？得到了焦虑吗？

单就这段文字，我觉得应该深入思考以下几个问题：

  * 顺丰原来的快件中发票的占比是多少，现在由于电子发票导致这一部分减少到多少？ 

  * 发票快件减少的同时，其他业务有没有增长，增长了多少？ 

  * 有没有新的快件品种(比如拼多多上卖手机的快件)，如果有，增长了多少？ 

  * 有没有新的业务类型产生？ 

  * 顺丰目前的估值算高还是算低，未来的增速预计在什么范围？ 

很多投资者每天会看很多信息，但是如果看的都是这样的信息同时又不思考，完全被文章带动情绪，被情绪带动交易，那想不亏钱真挺难的。

这些都是信息陷阱，掉入陷阱就成了别人的猎物。

想要真正在投资市场赚钱，就需要避开这些信息陷阱，寻找优质高效的信息来源同时形成自己的投资系统和方法论。

这样才能在市场先生发癫的时候有正确的应对，如果市场报了低价，那我们就多买一点，如果市场报价显著高了，我们就出让一点。

让市场先生为我们服务，而不是被市场先生牵着鼻子走。

在市场先生不正常的时候，热衷于寻找各种解读文章安慰自己，这样的投资者来市场大概就是来送人头的。

远离信息陷阱。
