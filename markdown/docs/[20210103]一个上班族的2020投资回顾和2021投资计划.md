#  一个上班族的2020投资回顾和2021投资计划

__ _ _ _ _

在小说阅读器中沉浸阅读

2020年已经结束，先回顾一下投资成绩，再说一下2021年的投资计划。

2020年股票账户总体xirr为15.05%，其中可转债打新贡献了2400多元(家里人的账户可转债打新收益不计入在内)，股息7千多元，货币基金套利的部分比较少没有单独计算，其他为股价增长。可转债打新和股息在不卖出股票的情况下带来了一些现金流，另外还有一部分资金在支付宝的存款产品上，平均利息大约为4.x%，这部分也带来了几万块的现金流。

目前持仓如下(占比高于2%的部分)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfJ9Riao9yU8yQibkHX4b8apb0zeCiasCWRKkFZDCroOLoJ6bKqyd1fLMzqcwZ91Z1lj6Xrsgqa3Md31Q/640?wx_fmt=png)

加权平均PE 9.98，加权平均PB 1.39，加权平均ROE 14.13%，加权平均股息率 4.75%。

由于我的持股会超过一年，因此即使我卖出A股股票也是免利息税的，港股通买入的股票利息税20%虽然不能免但是由于占比很少就没有在表格中扣除。

目前持仓三傻占比较高，整体低估，在没有白酒消费医疗芯片等好赛道股票的情况下取得了15%的收益，个人觉得还是符合预期的。由于三傻目前还是低估，明年有望小幅回归，有可能继续取得和今年相当的成绩，长期看15%的年化收益就是五年一倍，这个收益率是我的预期目标。

目前持仓的问题：

  * 单股持仓占比过高，平安占到了16%，后续如果没有特别优惠的价格将暂时不再买入平安，优先增加其他股票。 

  * 银行股占比过高，合计占比超过30%，后面考虑在合适的时候适当减少农行和交行，优先增加其他股票。 

  * 消费股占比过低，目前在观望双汇发展和伊利股份，会在合适的时候买入。 

目前可转债打新已经出现了连续破发，可转债无脑打新的阶段可能要过去了。可转债市场的整体回调使市场上已经有超过50个转债  价格
在100元以下，新的投资机会出现了，那就是可转债摊大饼。

2021年的投资计划：

  * 将理财产品和存款中到期的资金逐步增加到证券账户。上班族在难以提高收益率的情况下，增加本金能显著的推进投资进度，直到工资收入开始显得不重要。 

  * 调整上面提到的三个问题。 

  * 继续买入低估的A股和打折的港股。 

  * 增加可转债摊大饼，挑选20个平均买入，然后每个月调入更具投资价值与性价比的，淘汰持仓中投资价值和性价比最差的。 

由于需要花一些时间阅读与分析每个转债的资料，账户目前只买入了6支转债，剩下的将会逐步买入。

目前已经在EXCLE中记录了持仓股票2020年的收盘价，  忘记持仓成本，
将收盘价作为2021年持仓的成本，毕竟2020年已经过去收益已经计算完毕，当前的持仓成本对2021年已经没有参考价值了。

期待2021年市场的回报。
