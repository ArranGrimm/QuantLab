#  重大不寻常事件情况下的投资思考：以粤高速A为例(一)

在小说阅读器中沉浸阅读

熟悉历史、了解社会、相信常识，投资的功夫在诗外。

7月15号，粤高速A公布了业绩预报

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcPwORq9zCpRibCr03vaSdzGAnCU0gvmfMYJBt2fJriaUC7UskoYZ4ylLDw/640?wx_fmt=png)

净利润7.64亿~9.34亿，业绩同比增长3004.3461%~3693.9886%。

如果是新手，看到这个预报的第一反应可能是：卧槽，牛逼。

如果是有一些经验的投资者，可能会看一下过往的业绩对比一下

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcPEkX0eajNbvokt3Kfn1icmoNoRU2w80dG0IFHz2gkv2vcf7RDic5icjWhA/640?wx_fmt=png)

这样看是不是就正常多了。

抛开2020年中报，

如果利润是7.64亿，则相对2019年略有增长，相对2018年略有下降。

如果利润是9.34亿，则相对2018、2019有较大的增长。

那么值得思考的就是2020年的情况了。

2020年初的疫情让整个社会的生产生活受到了严重的影响，后来国家宣布了高速公路免费，于是这些高速公路公司一下子就少掉了很大一块收入来源，这将直接导致利润下降。

那么在这个情况下，会有一部分投资者认为粤高速A的基本面发生了变化，会卖出离场。

而卖出意愿强于买入意愿的情况下，将导致成交价格下降，于是就形成了趋势投资者眼中的下跌趋势。

结果也确实是这样，股价一直处在慢慢下跌的过程中。

而高速免费最终也确实导致了粤高速A的利润下降，于是2020年中报显示利润为负数

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcP1gEbj3cHAnqBcuwhqUtdhA1QMYS8v8iccK4aW5N7LEqqoMllkk1LybQ/640?wx_fmt=png)

于是股价继续下跌，2021年1月29号发布的业绩预报显示年报业绩大幅下降

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcP6g75sV1UCTSt1GuScFxDtdG3DW6oOY0tcxy5rE482YDMxHicC6Ekx7w/640?wx_fmt=png)

于是股价也在这一天又达到了最低点

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcP93Uh3Yym17QPceoUrysauNibPOrbu2ATtSeLspj25pLe7IdXYdG1HjQ/640?wx_fmt=png)

最终2020年年报显示利润只有往年的大约一半。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcPQdNueDxWHeMjrjn6ywqBvEcLx1EKT7GFP7useHdoiaJC4JXIIVE6ictA/640?wx_fmt=png)

其实我们可以再来看一下中报，2018年7.79亿，2019年7.36亿，2020年 负583万。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcPic3QeQZj53g8DN38W8ou8eQpurdWeJib7wc0ooQPdofJKG8NyItdP9NQ/640?wx_fmt=png)

对比一下年报

2018年16.77亿，相对中报增加8.98亿

2019年14.69亿，相对中报增加7.33亿

2020年8.68亿，相对中报增加8.73亿

看出什么信息没有？

2020年中报到年报之间的这一段时间，利润已经超过2019年，接近2018年水平，看上去已经正常了。

然后再看看官方的通知

>
> 根据交通运输部4月28日发布的《交通运输部关于恢复收费公路收费的公告》，经国务院同意，从2020年5月6日零时起，经依法批准的收费公路恢复收取车辆通行费。
>
>  
>
>
> **以下车辆可以继续享受免收车辆通行费政策：**
>
> 4、鲜活农产品运输车辆，进行跨区作业的联合收割机、运输联合收割机(包括插秧机)的车辆;

那么可以看出，虽然疫情还没有结束，但是社会秩序和生产生活已经基本恢复正常，高速的收费和利润也回到了正常水平。

当然，当时很多人依然对疫情后的社会生产生活恢复没有信心，那么我们可以来回顾一下历史。

先来看一下和这次疫情颇为相似的非典

2002年发生后，到2003年中逐渐消失，时间跨度大约一年半。

现在的社会发达程度相对于非典时期又有了很大的提高，人员聚集和流动都更强了，因此传播可能更快，但是也因为发达程度更高了，医疗水平和物质条件都有所提高，而这次我们展现出的惊人的动员组织能力，让疫情得到了很好的控制。

因此非典的时间跨度就有比较大的参考价值。

然后看一下更早一点的西班牙流感

> 西班牙型流行性感冒是人类历史上致命的传染病，在1918~1919年曾经造成全世界约5亿人感染。

时间跨度不到2年。

因此也可以看出，流行传染病，在100年前也能在大约2年左右平息。

最后看一下 欧洲中世纪大瘟疫

>
> 十四世纪四五十年代，对于欧洲来说，是一个极为悲惨的时期。从1347至1353年，席卷整个欧洲的被称之为“黑死病”的鼠疫大瘟疫，夺走了2500万欧洲人的性命，占当时欧洲总人口的1/3。

时间跨度7年。

这个情况稍微有一点不一样，当时整个世界的科学水平和民众素质都不高，当时注重个人卫生的犹太人很少被感染，这一情况并没有被社会做为正面例子分析，反而使犹太人成了替罪羊。

也就是说如果是在现代社会，根本不会持续7年，估计也就是1年半载的事。

因此，我觉得疫情并不会对粤高速A造成长期的实质性负面影响，于是我在2020年下半年到2021年1月之间一直在小幅买入粤高速A

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcPf8tASCvzUzNx9FF3wgTVfSzvOw3f84bbOumZLPrgntqU5AsPJ7VctA/640?wx_fmt=png)

由于我在今年春节之前在股市的投入还比较小，而且持仓有30多个股票，所以每次加仓根据不同的股票金额在1千到两三千不等，相当于每个股票加仓的话一轮下来就能增加5万的市值。

而我现在每一次加仓的金额在5千上下，股票加可转债的标的数大概在70个，相当于全部加一轮需要35万左右。

粤高速A最低加仓价格是5.77元，目前的价格是6.82元，加上7月14日的每股分红0.291元，相当于这一笔买入的收益率为(6.82+0.291)/5.77-1=23.24%，时间大约为6个月。

当然有人可能会说不可能加在最低点，这个确实。

但是6.82元也不会是粤高速A的高点，毕竟6月15号价格冲到了7.7元(不复权)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcPr2u8hVib9dy6RJd7LH7mnXiaf7WzibiaicmP6BiaucUGgnodLjLTGGA8nD9g/640?wx_fmt=png)

而2020年1月 6号曾达到了8.54元(不复权)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfILQgXQlbNDKEFcEGK9rWcPAwibf6qLFmAvZPJ7nUibBIZvM1oWGibqZcUicKNsgv1zloRss34ZhmBmlw/640?wx_fmt=png)

所以即使是在6.5元买入，在价格达到7.5元的时候，加上分红0.291元，收益率也会达到(7.5+0.291)/6.5-1=19.86%。

因此潜在回报还是比较可观的。

由此可见，熟悉历史将有助于我们在投资中识别机会。
