#  做胜率赔率占优的交易-以威孚高科获利为例

本周三在第三个涨停无法封住的情况下，清仓了 威孚高科，价格在24.3附近

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIBq5j567jxJdIxrZyz7Ksicpb8esZIvWzrqYpyqic740gUVX6Nibe7cjR0rkKvacxx7BuicOotiaDC4rQ/640?wx_fmt=jpeg)

本来打算留一手观察仓，后来想想没有什么实际意义，于是当天把留的那一手也卖出了

威孚高科首次建仓于2021年10月22号，价格21.1，见《 [ 投资总仓位加到了100W
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484412&idx=1&sn=2cd38e82b04c7327dd16d81964ecd405&chksm=cf17b276f8603b6035f960c5c7b645e3670b13216831cba9ea542c9a13324a042835aafc3f2d&scene=21#wechat_redirect)
》(同期建仓的还有双箭股份)

建仓不到半个月价格涨到24块多，于是少量T出了一点，然后在21块到20块附近陆陆续续买了一些，2022年6月每股分红1.6元，分红后在18到17块附近又陆陆续续买了一些

2023年2月20号条件单触发在20.3出了一小笔，然后就是本周三(2023年2月22号)清仓了

总体持仓时间1年多一点，回报率大约为27.5%，这个回报率我是满意的。

实际上威孚高科目前的估值也并不算高，为什么我要清仓。

因为在概念的加持之下连续3个板(第三个没封住)，并且成交量急剧放大的情况下(三天的换手率分别是4%，8.5%，14.51%，而在此之前的换手率基本上是在0.5%)，很大可能是柚子借概念进行的一波流割韭菜

那么大概率在接下来的一段时间里面价格会开始回落，大概率会回落到启动之前的20块以下

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIBq5j567jxJdIxrZyz7KsicGkNkicCgpxjsHd4IJwnDE3gRYQ4BRdadndjUEia9OYexQA5hKeGiakicdw/640?wx_fmt=jpeg)

而且假设不回落的话，目前市场上还有很多估值偏低的股票可以选择，完全可以用回笼的资金买入其他的股票

既然如此，选择清仓以后在20块附近再接回来(或者换入其他的股票)，就是一个胜率赔率占优的操作。

当然，操作并不是每次都那么令人满意

经常是要么卖飞，要么没卖出，刚好卖在高位的次数实际上很少。

但是，股票不停地轮动，资金不断地接力，不断地积累小赚，最终下来结果也不会差

施洛斯就是这样干的，不断地买入低估的股票，估值回升到一定程度(涨了一定幅度)之后就卖出，再买入新的股票，经常卖飞，但是这并不妨碍他在48年的投资生涯中取得年化20%的回报

而我卖出威孚高科后获得资金，第二天买入了两个钢铁企业的股票，一个PB0.61，一个PB0.74

4月30号所有持仓发布年报后，我会发布持仓汇总表，如果到时候这两个股票的占比大于1%则会出现在汇总表里面

这两个企业，如果在两年时间PB能回到1，则可以分别获得61%和35%的回报，再加上股息收益

这依然是两笔胜率赔率占优的交易

当然，如果仅仅只是如果，数学期望也暂时只是期望。
