#  通过储蓄率来计算一下财务自由到底需要多少钱

上周我写了一篇《假如有300万人民币的现金，可以全职投资吗？》，文章设定300W的目标是按我自己家庭的消费水平和消费习惯来推算的，但是考虑到不同城市的收入水平和消费水平差距较大，因此今天换一个角度，从储蓄率、消费水平满意度、财务自由度这3个角度来计算一下财务自由大致需要多少钱。

首先定义一下标准，财务自由度为:被动现金流/年开销，如果被动现金流能100%覆盖年开销则认为财务自由度是100%，如果被动现金流能覆盖50%的年开销则认为财务自由度为50%。

##  储蓄率与财务自由需要的年限

依然以权益类资产按《假如有300万人民币的现金，可以全职投资吗？》中提到的，每年4.5%的股息，2.5%的打新收益，5%-10%的资产增值，取8%来计算。总体看就是每年资产带来的现金流7%，除权后资产年平均增值8%，合计平均年化15%。

因为考虑到不用全职上班后，很多开销会减少，比如淡季旅游的机票酒店折扣很大，甚至可以秒杀0元机票，这些都是上班时享受不到的，另外工作日的时候很多餐厅刷卡消费5折都是常规操作，因此不用全职上班后开销大概能打八折。

物价和工资平均上涨3%，相当于平均两年获得一次6%的薪水提升。计算一下家庭已有不同程度的积蓄的情况下，不同储蓄率情况下，需要通过多少时间积累多少资产。

以家庭年收入30W为例计算。实际上年收入的多少并不影响计算结果，影响计算结果的因素是储蓄率和通胀率。

计算过程

    
    
    total=income*1 #已有积蓄等于年收入的倍数，下文分别计算0，1，2，3的情况  
    savingRate=0.1 #储蓄率10%  
    inflationRate=1.03 #通胀率3%  
    for i in range(30):  
        remain=income*savingRate  
        cost=income - remain  
        cashflow=total*0.07  
        print "第",i+1, "年 工作收入:", income, "开销:",cost, "上一年年底总资产:", total, "上一年资产产生被动现金流:",cashflow  
        if (cashflow>=cost*0.8):  
            break  
        total=total*1.15+remain  
        income=income*savingRate

##  消费水平满意度与变化

上面的计算结果的前提是消费水平不变，开销只随着通胀而增加。

如果当前的消费水平满意度很低，那么在积累了一些积蓄或者工资增加以后，消费水平会提升，这个时候储蓄率可能会有变化。

另一种情况是生活发生了变化，比如结婚了、买房了、有小孩了，这个时候储蓄率也会出现变化。

当然还有一种情况，就是大家喜闻乐见的大幅加薪了，这个时候储蓄率也会出现变化。
