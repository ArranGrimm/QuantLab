#  回购注销约等于现金分红(2)

在小说阅读器中沉浸阅读

接上一篇

《 [ 回购注销约等于现金分红
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247485048&idx=1&sn=0790c6d5b97892b6a84b7366909e8ec7&chksm=cf17b7f2f8603ee4cd8a57402039011927eb639947d60710a0f50d4721b0d715a27d86a168e8&scene=21#wechat_redirect)
》

上一篇分别用一个PB0.5的股票和一个PB1.5的股票做了一点理论计算，设定的ROE是20%

但是PB0.5，ROE20%，分红比例50%的机会是极其罕见的，因为仅仅是股息率回报就高达20%

上一篇留了一个问题

**如果ROE是10%的话，回购注销后的理论收益率分别是多少**

还是假设一个企业总股本是10亿股，年初的总净资产是100亿，经过一年的经营，获得了10亿的利润并且公司决定将5亿用于分红，也就是每股分红0.5元，另外5亿增加到公司账上计为净资产。

分红对应的两种情况

如果年初的股价是5元(对应每股净资产10元，PB为0.5)

那么分红后如果市场给的估值不变，则股价会平衡在5.25元附近(对应每股净资产10.5元，PB为0.5)，同时账上获得0.5元现金，市值5.75元，回报率15%(5.75/5-1)

如果年初的股价是15元(对应每股净资产10元，PB为1.5)

那么分红后如果市场给的估值不变，则股价会平衡在15.75元附近(对应每股净资产10.5元，PB为1.5)，同时账上获得0.5元现金，市值16.25元，回报率8.3%(16.25/15-1)

而如果是用5亿来回购股票并注销，对应的两种情况

5.25元的价格，5亿可以买入0.952亿股，注销后总股本降低为9.048亿股，每股净资产11.6元(105亿/9.048亿股)，如果市场给的估值不变，则股价会平衡在5.8元(对应PB0.5)，回报率16%(5.8/5-1)，如果是按5.75的价格回购，5亿可以买入0.869亿股，注销后总股本降低为9.131亿股，每股净资产11.5元(105亿/9.131亿股)，如果市场给的估值不变，则股价会平衡在5.75元(对应PB0.5)，回报率15%(5.75/5-1)

15.75元的价格，5亿可以买入0.317亿股，注销后总股本降低为9.683亿股，每股净资产10.84元(105亿/9.683亿股)，如果市场给的估值不变，则股价会平衡在16.26元(对应PB1.5)，回报率8.4%(16.26/15-1)，如果是按16.25元的价格回购，5亿可以买入0.307亿股，注销后总股本降低为9.693亿股，每股净资产10.83元(105亿/9.693亿股)，如果市场给的估值不变，则股价会平衡在16.245元(对应PB1.5)，回报率8.3%(16.245/15-1)

可以看到在市场给的估值不变的情况下，企业进行分红，或者是用现金进行回购并注销这两种情况，对于投资者来说获得的理论回报基本上是一致的(平价回购注销带来的理论收益率比直接分红略微高一点)。

并且我们可以看到，在同样的ROE水平上，低PB的股票进行分红后投资者获得的回报将远大于高PB的股票

同时对照上一篇可以看到ROE对预期回报的影响是巨大的

而如果ROE能维持的话，低PB的企业(通常是稳定的傻大黑粗型重资产企业)应该多分红多回购，高PB的企业(通常是高速增长期的企业)应该少分红少回购适时增发做大蛋糕。

不过这里可能有人问，要是市场给的估值变化了怎么算？

那么可以看一下我之前写的《 [ 看透股价的幻觉才能赚钱
