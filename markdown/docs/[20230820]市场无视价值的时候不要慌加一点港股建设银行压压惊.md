#  市场无视价值的时候不要慌，加一点港股建设银行压压惊

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

本周(20230814至20230818)所有账号市值微跌大约一千块，忽略不计，周四加了一点 港股建行，周五加了一点 道恩转债

先说一下建行

我之前持有A股建设银行比较长时间了，小部分是3年前买的，大部分是2年前陆陆续续加仓的，整体平均持仓时间大概不到一年半，看了一下整体收益率大概是17%

一方面是吃了两次分红，去年每股分红0.364，今年每股分红0.389，按本周的收盘价5.96，股息就带来了14%+的回报，加上去年11月16号5.38加仓了一笔，今年2月15号5.57又加仓了一笔，于是综合收益率就到了17%

所以股息对投资收益的贡献还是比较显著的，另一个方面就是要买的便宜。

而这一周 建设银行略微上涨，但是港股建行却跌了不少。

看着这种跌法，有便宜货还是要收一点的

于是周四早上看到卖1价格4.07有2千万多手，买1价格4.06也有2千多万手，我就挂了一个4.06的买单，还特意看了一下成交明细，4.06的买单一分钟大概成交40万手，估摸着一个小时后就能成交，结果到了快12点，我的买单还没有成交，而4.07的卖单只有一千二百万手左右了，估计不少原本和我一样挂4.06买单的人主动吃掉了4.07的卖单

到了下午一点，好家伙，连续几个几十万手的4.07和4.08的大买单，最大的一笔有139万手，瞬间卖1买1就变成了4.09/4.08，然后我改挂了4.07的买单，想着能成就成，绝不上头加钱，结果没多久就成交了。

为啥买港股建行？当然是图便宜了

我看了一下，港币从8月1号到8月14号最高对人民币升值大约1.75%，而港股建行的价格从4.54到4.07，跌幅10%+，股息回报已经10%+了(港股派息和A股金额一模一样，而且都是人民币)，就算有20%的利息税，拿到手也还有8%+，涨不涨的已经无所谓了，主要想躺着吃利息。

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLBBehqvgiaA8XFnpxcaDwcxHaUMibq6QO3AZOV8kPEJaH7G7UPMRibxvR4Uky7bvlVfwiaU9J5hIlJtw/640?wx_fmt=jpeg)

当然也不是说真的就不会涨，市场会短时间无视价值，但是总有重视的时候，而且一旦重视起来也挺吓人的，比如今年上半年的中特估，建设银行在5月9号最高涨到了7.28，比本周的收盘价高22%，而港股建行5月9号最高5.7，比本周收盘价高39.7%，算上汇率差异，也比本周高32.4%

所以在这个位置买不了吃亏买不了上当，下一次市场啥时候重视无法预测，假设3年后回到5月9号的水平，股息加涨幅合计能拿到50%到60%的回报，这已经很可观了。如果5年内回到五年前的水平

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLBBehqvgiaA8XFnpxcaDwcx6zzm6G0SicMvCtaDCQ9MSppicGvvHBxcH8a5TqMVk08B6NZicyW2nnxyA/640?wx_fmt=jpeg)

累计股息回报40%+，股价涨幅130%，扣除汇率影响，股价涨幅也有100%

所以当市场无视价值的时候，不要慌，挑选一些值得持有的股票，买入并等待，剩下的交给时间。

再说一下道恩转债

2021年3月97块多买入后，2022年6月130块卖出吃了一波，今年3月10号117.29又买入了，周四逛雪球看到有大佬提到，决定抄个作业加一点。

周五早上按前一天116.6的收盘价挂了买单，最终开盘116.15成交。还有两年多到期，到期118元加两年税后利息2.8元，不违约的情况下，最差的结果是持有到期，年化收益大约1.37%，而中间如果涨了，就可以卖出了，就像上一轮130卖出一样，而且它的历史价格是这样的，我130卖出的那天收盘161，第二天最高200块，严重卖飞

之后又出现过两次价格超过130

所以看到合适的作业，抄一抄也无妨，不丢人。  我也时不时抄点作业，但并不是无脑抄，也得看看作业的质量和自己持仓的情况 。

当然我的持仓也是公开的，要抄作业随意，赚了是你的能力，亏了和我没关系哈。

本周就这样了，反正上证指数跌了2.35%，沪深300跌了2.58%，估计很多人的账号都是亏钱的

既然已经在这个低估的区域了，手里还有钱的话就看着合适的买一点，手里如果没钱的话躺平就行了，再跌也没多少空间了。

而一旦涨起来，那也是气势汹汹，上一波中特估的暴涨犹在眼前，不是吗。
