#  清仓精达转债，5个月获利56%

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

6月15号下午，精达股份发布了《精达股份关于不提前赎回“精达转债”的提示性公告》

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfJ2kTiauGSCAJKAD07ibOJzP0jhzqq1OibfRQtzksmqTib1g6JKnM66vNwicL1CKmMEQoUicwCuQLx37MEA/640?wx_fmt=png)

由于当天工作很忙，晚上回家没有注意看公告，直到16号晚上到集思录看了一下强赎榜，发现精达公告不强赎

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfJ2kTiauGSCAJKAD07ibOJzP07brSze42Fnj6Ftw7USJfMJZbjUWXUK0a2xkaAFo0My0MNpica4S95dg/640?wx_fmt=png)

资料显示精达转债上市时间为2020年9月21日

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfJ2kTiauGSCAJKAD07ibOJzP0wv5D4xjb7XE1HdAKfcvrUlOTpDntydsomqmjw71FibzsQEibIUfttoUA/640?wx_fmt=png)

上市到现在的时间为9个月就达到了强赎标准，算是比较快的。发转债的公司基本都是不打算还钱的，因此基本上会抓住机会启动强赎，然而现在精达股份显然是还不想强赎。

既不想还钱，又舍不得用股票强赎，大概是想这样耗着，先占几年市场的便宜吧。

于是第二天早上直接挂单卖出了，整个周期142天，获利56%，折算年化214%，算是比较理想的回报了。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfJ2kTiauGSCAJKAD07ibOJzP0vTJM4Kl8uhrqkVV45P2aGswXI1HMJMjp5DEo6cjZCu6sueLnZEQSWA/640?wx_fmt=png)

然而周末我再回顾的时候看到，在我卖出的第二天  又  涨了10%

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfJ2kTiauGSCAJKAD07ibOJzP0fUFCdu187B85wrwvyCHfOoibu0sNhfqNUW4l7vxYibNyVcRmseJCPtnQ/640?wx_fmt=png)

之前在5月31日，价格也到达过164.88元。

当然，我肯定是没有能力卖在最高点的，如果有这个能力，估计我很快就能登上福布斯。

所以能做  的  就是多回顾历史数据，多理解关联因素，尽量获得较好的回报。

清仓精达转债后，按之前的筛选标准，我买入了正川转债。

这一段时间经常有人看到我的文章后问我，现在还能买转债吗？

对于这个问题我是这样看的，相对于今年1月，转债整体都上涨了不少，像年初那种遍地便宜货的情景现在已经没有了，我的转债组合整体浮盈14%，全市场的涨幅我没有统计，估计应该在10%左右。

也就是说按我年初建仓的平均价格94元的话，现在同样的组合可能成本要到103元左右。

按照溢价率小于等于40%，价格在90到105之间的转债仍然有62只，价格在100到105之间的有47只。

也就是说仍然能够构造一个可转债组合，只不过成本可能在100元~103元之间。如果说我之前在《双低可转债组合构建完毕预期年化15%》中写到的预期年化为15%的话，那么现在构建的组合的预期年化收益率大概在11%的水平。

当然最终的收益可能会更高，因为在《双低可转债组合构建完毕预期年化15%》中按130元强赎，计算的预期收益是接近15%，按140元强赎则预期收益是18%，按平均下市价166元强赎则预期收益为26%。

这次精达转债的卖出价格是146.22元，而上一个上岸的森特转债的平均卖出价格是145元。因此我的组合最终的收益率很可能会高于15%，同样的如果现在构建组合则最终的收益率也很可能高于11%。

但是要特别注意，预期年化收益11%并不代表买入后就慢慢涨到每年有11%的收益，因为这个过程不是线性的，中间会有起伏，也可能建仓完成后就遇到回调，我在年初建仓的时候就遭遇了5%的账面浮亏。

另外一个需要特别注意的是，虽然市面上的可转债的平均存续期为2年多，但是投入的资金也要做好持有到期的准备。
