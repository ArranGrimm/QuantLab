#  横盘阶段做T娱乐

在小说阅读器中沉浸阅读

做T不仅仅是娱乐，还能一定程度增强收益。

这周比较有意思的是 千金药业，继上周五涨停后，周一继续涨停，我顺势T了一小部分

周一晚间公司发了公告说公司没有千金藤素产品，同时另外几个千金藤素概念公司也发布了一样的公告。

然后周二的时候 大理药业10点就开始跌停并维持到收盘，这企业亏损，跌停不冤枉，之前涨停才是不正常。

而千金药业就好一些，开盘11.11，然后还最高涨到了11.2，我在开盘价挂了一点卖单就没管了，后来发现成交了。

周三更有趣，最低跌到9.88，最高触及涨停，不知道谁在割肉，谁在站岗。

这一波小比例的T，给我的全仓带来了大概0.157%的增强收益。

看起来是不是微不足道

但是

如果这样的操作一年能做30次，则可以给全仓带来4.71%的增强收益

如果这样的操作一年能做50次，则可以给全仓带来7.85%的增强收益

是不是觉得不太可能

如果全仓一个股票，这样的T，一年做30次50次，那确实不太可能

但是如果，全仓有几十个股票呢？

我的全仓有40多个股票，千金药业只占全仓的5%不到而已，如果一个股票一年做2次，只要15个票做到了，那就有30次了，就能给全仓带来大约5%的增强收益

这样一看，可能性是不是就大了很多？

当然这并不代表一定能做到

来看看前一段时间我的持仓经历的波动

金地集团2月28号12.53小额增持后，3月3号13.97小额T出，4月15号15块多没有操作，5月12号12.33接回

长江电力 5月9号 22.31小额增持，5月20号23.72T出，如果到22附近会接回来

葵花药业
4月13号14.45小额增持，4月22号15.52、4月25号17.07、4月28号17.91、4月29号19.03，4次T出，如果到16以下会接回来

那么天底下真的有这么好赚的钱吗？

在回答这个问题之前，首先要了解网格交易的几个局限性

局限1 如果单边上涨，很快就T完了，极大可能会踏空整个牛市

局限2 如果单边下跌，很快就没钱了，遇到更便宜的价格，无法买入

局限3 如果一直小幅震荡，闲置资金效率低

那么如何解决这3个问题？

增加边界条件，比如

对于局限1 如果某个股票涨了一定幅度，比如10%后T掉一小部分，再涨10%再T掉一小部分，再涨10%(或者20%)后T掉一小部分，然后暂停，防止错过翻倍行情

对于局限2 如果某个股票跌了一定幅度，比如10%后小额加一笔，再跌10%再小额加一笔，再跌10%(或者20%)后再小额加一笔，然后暂停，防止腰斩深套

对于局限3 用分散来解决，如果组合中有30个股票，在横盘阶段涨跌不一，今天这个涨了T掉，明天那个跌了加仓，左手右手来回倒腾，相当于这一笔机动资金一直在滚动。

上面的幅度只是举例，实际上对不同的股票我设定的幅度是不同的，需要根据股票本身的因素(估值水平，历史价格区间，股息等)来估计价格区间，并且需要在长期的网格交易过程中不断修正，类似牛顿迭代

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfJawSnR2CcMql1qBjdfXXbJRpDic3TpibOdjBU8RlBBiblLvuC3n8ibnXlhcQj5aqhZ639NKRP9X4k6vw/640?wx_fmt=jpeg)

不断的调整x的值

当然，这个x不是固定不变的，因为不同的股票利润的增长有高有低，不能一概而论。

那么这样的网格交易，有没有代价？

显然也是有的，比如某个票涨上去后就不下来了，那么这部分就接不回来了。

例如，持有某个股票1000股，股价10元，市值1W

涨10%T掉100股，股票市值9900，现金1100

再涨10%T掉100股，股票市值9680，现金1100+1210

再涨20%T掉100股，股票市值10164，现金1100+1210+1452

合计市值13926，如果不操作，则市值为14520

而分散，可以在一定程度上消除这个问题，一个由30个股票组合的组合，基本上不太可能在一年内所有的股票都上涨40%多，即使是在一个上涨趋势中，中间也会有反复

如果真的都没有接回来，就当是股票起飞的燃料吧。

而这个股票T出后的资金，可以用来给下跌的股票增加份额，然后等涨上来后再T。

假设一个股票长期看年化涨幅是15%，那么这个15%肯定是不均匀的，可能11个月都是正负5%的波动，最后一个月涨了15%，又或者23个月都是正负5%的波动，最后一个月30%的涨幅。

而对于这个股票，如果拿出10%做网格交，只要一年内吃到2个7%的差价，最终要是接不回来也并不亏，因为这部分做丢的份额已经提前获得了14.45%的收益，资金还可以给别的票来做网格

如果没吃到差价，则表示这个票在短时间内已经显著上涨，要么等价格下来了接回来T飞的部分，要么继续涨了以后再次考虑T，特别是无厘头的上涨。

市场总是时不时出现无厘头的上涨和下跌

肝炎新闻来了，葵花药业无厘头涨停，千金藤素新闻来了，千金药业无厘头涨停，等等。

最后一个问题，这部分增强收益的来源是什么？这个问题是整个逻辑的核心

来源是： **没有耐心的对手盘，市场中的绝大部分散户都没有耐心，而且喜欢追涨杀跌**

这些追涨的散户，给了我们T出的机会，而等到大家一哄而散的时候，又给了我们接回来的机会。
