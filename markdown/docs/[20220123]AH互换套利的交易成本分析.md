#  AH互换套利的交易成本分析

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

AH互换套利，指的是通过在A股和H股(港股)的同一个标的上来回交易来获取收益的一种方式。

因为在两个不同的市场，价格会有差异，而差异的比例并不固定，因此就产生了套利空间。

比如某个股票在A股现在是10元，在港股目前折算人民币是8元，某一天A股价格跌到9元，而港股价格跌到人民币6元，则卖出10000股A股股票可以买入15000股H股股票。然后某一天，A股价格回到了9.5元，而港股价格回到了人民币6.65元，则卖出港股得到99750元，可以买入A股10500股。

相比持股不动，增加了500股。

当然，这里有一个前提，就是想要进行这项操作的人本来就打算长期持有交易标的，否则这种套利操作就演变成了波段操作，属于另外一个范畴。

我在很早之前就知道这种操作，但是之前一直在上班，所以没把注意力放在这块

但是前几天在雪球上看到 @流水白菜 老师，又进行了一次AH互换，获得了8个点的收益

我觉得有必要开始根据自己的实际情况，制定AH套利的操作计划了，毕竟我也是长期持有 中国平安，以及另外一些AH同时上市的股票。

AH互换的操作有两种方式

一个是直接在港股通里面操作，相对比较方便，但是港股通的红利税是20%

另外一种就是在A股和港股两头操作，在港股市场直接买H股，资金进出不方便，但是红利税是10%

今天只说通过港股通交易的成本

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfKBtibe760GhfzhrnG2S8qNkdkcZ0m9OBoMDYFwkANJ4Wa71C5P7YL63RtUic9CzoUpiah0Rwy0YN2FQ/640?wx_fmt=jpeg)

港股通买入汇率0.8393卖出汇率0.7905，1 - 0.7905/0.8393 ≈ 5.81%

所以很多人会发现港股通买入港股会马上就显示浮亏大约6%，实际上这个是港股通在结算之前预先保留了两个3%作为汇率波动的缓冲。

1 - 0.814/0.8393 ≈ 3%，1 - 0.7905/0.814≈ 2.89%

买入的股票结算后如果股价没有变动会显示浮亏3%，也就是买入时保留的3%已经没有了，还保留了3%作为卖出时的缓冲。

如果看交割单就会更清楚，刚好我一年前在港股通短线交易了一次中国移动，可以通过交割单看一下交易的成本

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKBtibe760GhfzhrnG2S8qNkj7icRiamRWKydicicj0rO19ZfTJI2vhBRvfJy4FWqukEZIUc8cfJ2icsb6w/640?wx_fmt=png)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKBtibe760GhfzhrnG2S8qNk2fxu8DW7tLmnTLWBOsz1vtwVp2bhOrUAQ4IhfRbOLIzuQ0fmYjA2kw/640?wx_fmt=png)

可以看到买入和卖出的结算汇率分别是0.83257，0.83547，结算日期T+2是2021年1月12号和2021年1月14号，汇率小幅波动，所以在不考虑汇率波动的情况下，汇率带来的成本大概应该是这样的  

比如购汇结汇价格如下(手机银行的app截图)

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfKBtibe760GhfzhrnG2S8qNkRsaEXYlXLkb6dVc3qHzHrSv8wIfhhvzxOEokicJZibkZ6D0NzucHgtCA/640?wx_fmt=jpeg)

则买卖各一次的成本为 1 - 81.25/81.58 ≈ 0.4%

印花税双向0.1%，其他七七八八的费用 (12.24+1.66+0.87+0.41+0.47)*2/17483 ≈ 0.18%

综合下来港股通买卖一次的成本约为 0.4% + 0.1%*2 + 0.18% = 0.78%，这个佣金能不能降低我不确定，我的客户经理回复我的是不能降低。

但是要注意，港股的印花税从2021年8月1号开始由0.1%上调至0.13%，那么交易成本就变成了 0.84%

然后A股如果按万2计算，印花税千分之一，则买卖一次的成本为 0.1%+0.02%*2 = 0.14%

一次完整的AH互换套利，包含从原市场卖出，在新市场买入后卖出，然后在原市场买进，4次操作交易成本约为0.98%

至于AH的差价在什么水平的时候适合做互换套利，这个就难以统一了，因为涉及到A股的打新加成，账户里面市值不同，打新的理论收益率是不一样的，另外套利空间多大的时候进行操作也属于个人偏好，无法一概而论。

另外有一点需要注意，A股股票持有不足一年卖出，如果持股期间有分红，是需要扣红利税的，这个我之前写过，这里不单独写了，参考《股票分红科普与原理分析(2)》中后半段
