#  其实A股给普通散户提供了大量的机会

乐观者看见甜甜圈，悲观者却只见孔中深渊。

A股当然存在很多问题，长期以来定位为融资市，也就是为企业提供融资服务，回报股东并不是重点。

问题还有很多，因此很多人亏钱就会吐槽市场

然后喊出了：相信国运，定投纳指。

这句话热度最高的时候大概是7月中下旬，也就是那个时候纳指创下年内新高，如果真的在喊的时候买入，到现在又亏钱了。

吐槽市场没有问题，不过在吐槽的时候不要忘了识别市场中的机会，毕竟A股其实给散户提供了大量的获利机会。

比如2021年8月前后，2021年11月中下旬，2022年3月中，2022年7月，PB不到0.8，股息回报接近8%的大秦铁路，这么多次买入的好机会

比如2022年8月到10月，PB0.7到0.8，股息回报超过7%的中文传媒(对应价格9块上下)，在2023年给了12块到19块多的卖出机会。

类似的例子很多

很多人会说A股不规范，定价混乱，垃圾股票猛涨，有价值的股票不涨。

这些都是客观事实，但是另一方面，这不是恰好给散户提供了可乘之机吗？

垃圾股票猛涨，那意味着如果在低位不慎买入了垃圾股票，依然有高位卖出的机会。

有价值的股票不涨，那意味着可以长期低位买入有价值的股票，  通过股息回报和复投，  最终获得可观的回报。

当然跟风交易，那相当于给其他人提供获利机会，自然是要亏钱的

比如基金抱团行情的时候，眼看着抱团股一直涨，最后喊一句：“打不过就加入”，然后冲进去。

这个时候市场其实已经给出了机会，就是那些因为不受基金青睐的股票由于没有资金追捧而跌入低位，估值那么低而股息回报那么高。

当然买入后会面临抱团股继续涨而持仓股继续跌的局面，这个是无解的。

因为想要获利，就需要承担这个局面，正因为承担了这个局面，承担了这些不爽，才能获得回报。

市场上没有既让人爽又能获得回报的事，“打不过就加入”，加入的一瞬间之前的煎熬都烟消云散，心情爽了，但是后面就需要为这种爽买单了。

比如最近报道的几个基金经理  卸任，留下一个大额亏损的基金。散户在基金中亏了不少钱，基金经理的管理费却不能少。

基金公司和基金经理当然难辞其咎  ，但要命的是在熊市发基金未来回报高就是没人买，在牛市发基金未来亏损大但是买的人多。

个别的良心基金经理可以在估值高的时候限购甚至卖出后分红，但是如果基金经理在公司的地位低的话，这样干可能会被公司开除，毕竟影响公司赚管理费。而且在市场火热的时候这样干，不仅会被客户骂：“为什么不让我申购，影响我赚钱”，还会被同行骂：“你能不能别说话，你不想赚钱别拖累同行”  

我觉得一位前辈写得很对：

> 这是个对跟风型散户不友好的市场，这是个对学习型散户很友好的市场。
>
> 风险和收益匹配，那是总量的匹配。
>
> 四顾周围，多少人承担了大风险，却只挣点小收益甚至亏损。也正说明有许多的高收益，需要承担的风险并不大。

A股中跟风的人如此之多，以至于市场经常给出不合理的报价，提供了大量风险小而潜在回报可观的机会。

就像甜甜圈，虽然中间有个空洞，但是周围一圈还是很香甜的，不是吗。
