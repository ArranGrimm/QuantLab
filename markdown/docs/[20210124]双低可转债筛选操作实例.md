在之前的文章《 [ 一个上班族的2020投资回顾和2021投资计划
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483944&idx=1&sn=f702f29d9cdd111fd3a9e52bf1d1200a&chksm=cf17b3a2f8603ab43dca4a10fe365d95fedd8ff05e675709f146482324fb46bfc5aea8b99236&scene=21#wechat_redirect)
》有提到2021年的投资计划里面会增加双低可转债摊大饼，在《 [ 双低可转债摊大饼策略简要分析
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483949&idx=1&sn=919077321db56f33fa01ac35b6581927&chksm=cf17b3a7f8603ab1ef1c5d29cbd3acc5e7503ba5a7f99531da17d1d739d61cfcaad85ce2f3f0&scene=21#wechat_redirect)
》中说明了双低可转债摊大饼的原理和操作策略，今天来说一下操作实例。

目标：价格小于100元，正股PB  ≥  1.3，溢价率≤40%，企业处于盈利状态。

最终目的：  达到或超过130元以后卖出换入新的双低转债，如果没有则等待强赎公告后卖出或转股卖出。

打开集思录网站的可转债板块  https://www.jisilu.cn/data/cbnew/#cb

在溢价率一栏填入40，点击筛选后，点击标题栏的“现价”，按价格从低到高排序，如图

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIZNwm60PdibW77w8rRp29dVDeIH3enHRLh4b084FOXGTvjCSZ40rsVgvHwPXIpibDO9Ez9Nib6Jaibew/640?wx_fmt=png)

在结果中淘汰不符合条件的，挑选符合条件的20支，等量买入，淘汰与选择举例：

  * 广汇转债 PB0.54，淘汰。 

  * 本钢转债 PB0.56，淘汰。 

  * 鸿达转债 PB1.09，淘汰。 

  * 孚日转债 PB 0.98，淘汰。 

  * 新星转债 PB 1.55，价格 88.87元，溢价率30.44%，正股深圳新星，2019年利润9.4千万，2020年三季报利润2.8千万，入选。 

  * 永安转债 PB1.07，淘汰。 

  * 花王转债 PB 1.55，价格 90.79元，溢价率20.3%，正股花王股份，2019年利润9.7千万，2020年三季报利润6百万，入选。 

  * 搜特转债 PB1.29，淘汰。注意，搜特转债的正股前一段时间PB是大于1.3的，对应正股搜于特最近跌了一些因此PB降低了，另外搜于特处于亏损状态，也是要淘汰的。 

  * 威唐转债 PB3.43，价格91.58，溢价率27.8%，正股威唐工业，2019年利润3.6千万，2020年三季报利润3.6千万，入选。 

以上几个范例说明了如何淘汰与入选，按这个规则，挑选出20个转债，等份买入，每个转债的占比大约为5%。

按过往的历史推算，可能会有1个转债到期还钱(也可能没有)，剩下的强赎。  

看一下集思录上显示的当前处于强赎阶段的转债的情况。页面上点击“强赎”或者链接
https://www.jisilu.cn/data/cbnew/#redeem  ，点击标题栏“强赎天计算”排序。

目前已经公告强赎的有7个转债，其中除了晨光转债之外，其他的6个转债价格都是高于130元的。

晨光转债发布强赎公告是1月14日，当天晨光转债最高价131.7元，最低价127.5元，按《投资要义》和《财富工具》里面论述的策略，当公司发布强赎公告以后最好马上卖出可转债以免夜长梦多，因此当天卖出将能收回每张130元左右。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIZNwm60PdibW77w8rRp29dVh1dIiabicaNCBFfBXe6VddJElMnMPvkblC5BjzNtcp2RIlvTjlQuich5A/640?wx_fmt=png)

再来看看满足强赎条件但尚未公告的可转债

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIZNwm60PdibW77w8rRp29dVQggmMYywkWKHmaDpJfqsWjWpOtvwEc0AWp8OUvhyicGra6VTDCtlIiaw/640?wx_fmt=png)

目前有十个，最低价152.66元，最高价334.6元。

其中我看到了一个眼熟的名字：巨星转债。

这个转债我中过，上市当天130元卖出了，因此可以看出，可转债打新和双低摊大饼，是两个不同的盈利策略。  

双低可转债的盈利预期来源如下：

摘录一段《投资要义-增订版》的内容：

> 截止到2017年6月
>
>   * 国内市场历史上多数可转债的收盘价格都超过了130元，平均收盘价格为166.3元。
>
>   * 国内市场历史上可转债最高价格的平均值为200.9元。
>
>   * 国内市场历史上可转债最低价格的平均值为102元。
>
>   * 国内市场历史上可转债的平均存续期限为2.3年。
>
>

以上历史数据显示，以目前的情况来看当前的可转债市场是存在显著的获利空间的。

摘录一段《财富工具》(2020年6月出版)的内容：

> 从结果上看，历史上发行并且已经到期的可转债中，没有成功实现转股的比例只有不到5%。

也就是说，买入20支可转债，可能会有1支可转债最终执行了回售或者到期赎回而另外19支最终在130元以上执行了转股(当然也可以直接卖掉)。

需要说明的是，双低可转债策略的最终目的是等待强赎(或者卖出130元以上的转债换入新的双低转债)，企业可能通过等待正股上涨来达成强赎条件，也可以通过主动下修转股价来达成强赎条件，但是在这个过程中，可转债的价格是有可能继续下跌的(也可能上涨)，例如我现在已经买的转债中，既有买入后继续下跌的，也有买入后上涨的。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIZNwm60PdibW77w8rRp29dVlZQbohbyKfPnt3dwLElVZ2Sr2XVMfMMwKkPlkDic3QCFsaiao1hwSogg/640?wx_fmt=png)

买入后涨的最多的是这个

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIZNwm60PdibW77w8rRp29dVd7iaoKqNRRy3TbIrIaibD1vGa2rcMdmlyeY0mcB1QfibDZlu7yp69O2Ww/640?wx_fmt=png)

为啥上面那个图显示的单个转债数量是30，下面这个图的转债数量是20？

因为我是分批次买入的，一次买入20个转债，每个买10张，第二次再次买入第二批，到我买入第三批的时候，斯莱转债已经不在选中的20个转债之中了，因此这个组合最终的结果将是大部分转债的占比为5%，少部分的转债占比小于5%，总的个数略大于20个。
