#  能适应曲折前进才适合投资股票

本周五(2020年9月18日)，股市大涨，大部分人的账户收获了幅度不等的浮盈，我的账户一天的涨幅超过了上个月一整个月。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIOQjIA6YV5wqYJHhYpxibicoGK9uNcGlB7n4lfFL4cgxD8DAFkVDwwh5p2Np2BZfXCRPmb7uW1PAMg/640?wx_fmt=png)

网络上一片欢天喜地，完全忘却了之前的唉声叹气。我倒是还好，情绪没有波动，为什么？因为我知道我自己的组合大概率会走向何方，即使今天涨明天跌，那都是市场的正常起伏，就像长江的波浪，风小的时候波浪就小一点，风大的时候波浪就大一点，唯一不变的就是最终都会流入大海，既然大概率知道了最终的结局，也就没什么好激动的。

看看7月6号那天真刺激，一天涨那么多，不过我那天也是下班后才看到的。可能有人会说，涨了那么多，为什么不先空仓，保住利润躲过后面的下跌。那么，我先把后面的数字遮住，谁来告诉我哪天应该空仓？

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIOQjIA6YV5wqYJHhYpxibicoGkbCwsMCmB2Rv9vcIgkibX6VfuUZseBW24BTqmIiaibTVPNFHBfMOerIg/640?wx_fmt=png)

连续涨了2天，要不要空仓？如果周5空仓2500块的涨幅没有，如果周一继续空仓又有1W的涨幅没了，周2再进来，duang，1900块的跌幅，周3赶紧空仓，duang，接近1200的涨幅又没了。

9月18号空仓，duang，一个月的涨幅没了。

* * *

以天为周期看，每天起伏不定，最后一看这几个月都是浮盈的，恩，5月是浮亏的。

以月为周期看，每个月也是起伏不定的，1月到3月好像有小幅浮亏记不清了，最后一年下来，大概率是盈利的。

以年为周期看，每年的收益也是起伏不定的，可能是5%，也可能是20%。但是以10年20年为周期，如果中国的GDP维持在5%左右，那么我将大概率获得平均年化15%左右的收益。

2月3号3000股跌停，我趁打折买了一些。我的逻辑是这样的：

  * 如果疫情几个月搞定，社会秩序恢复，这些优质公司的利润会受到一定的影响，但是不影响以后的业绩，那么现在打折卖，后来折扣没了，那就很划算。 

  * 如果疫情要1年2年搞定，这一年企业的业绩受影响，过2年才恢复，打折买也不会吃亏。 

  * 如果疫情一直搞不定，整个世界都乱套了，那买了也不亏，因为钞票股票可能都没用了，得用粮票了。 

所以不论从正面还是负面看，好像都是机会大于风险，于是我就买进了一些。当然要说我是不是有100%的把握？那肯定是没有，所以我只买了一些。

* * *

股市的每一笔交易，就像封闭系统中的气体在做无规则的布朗运行，既可以通过一笔交易让某个股票的最后成交价上升，也可以通过一笔交易让某个股票的最后成交价下降。但是业绩良好的企业的股价最终会慢慢抬高，就像布朗运动最终的结果是速度快的分子运动速度降低，速度慢的分子运动速度提高，最终整个系统达到平衡。当然，股票市场这个大容器，可能永远也没有平衡的那一天。

* * *

股票价格经常会偏离价值，价值在变，股价也在变，但是不管怎么变，股价总是倾向于回到价值附近。

只要不在显著高估的时候买入，大概率是会最终盈利的。你要是问什么是显著高估，那种按当年利润要100年才能回本的，就是显著高估，如果非要说高估的是不是一定不能赚钱，那也不一定。
比如按第三种情况，在第一个高估的时候买入，到了第二个高估的时候，账面是盈利的，但是要在第二个高估的时候卖出，才能真正有盈利。第一个高估的时候买入的人，在第二个高估的时候如何做到卖出？

有机会大于风险的企业，为什么一定要选风险大于机会的企业？

* * *

今年剩下的3个多月，我的市值是会增长还是下降抑或是持平，都不要紧，要紧的是持仓的企业利润在涨同时股息在涨，那就够了，手里的资产价值含量在增加，市场的短期报价其实无关紧要。作为一个普通的上班族，投资股票如果总是导致情绪起伏，影响工作影响生活，是一件极其不值得的事情。只有理解股票本身的正常波动，适应曲折前进，才适合投资股票。

最后，祝大家投资顺利。
