#  股市萌新可以考虑从高息股开始入手

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

这周(2020年11月22日~2020年11月27日)三傻(银行保险地产)涨了不少，工商银行更是在周五当日上涨5.89%，实属少见，当然后面也还是很有可能会回落，不过这种小碎步的上涨方式，估计又开始让不少人蠢蠢欲动了。同样的戏码在年初也上演过，只不过那个时候的主角是医疗白酒科技。如果总是看到涨幅然后开始入市，恐怕容易事与愿违，对于股市萌新，我建议从高息股开始入手，为什么？

股市是一个庞大的市场，里面有很多生态，各种人群不同的策略。每个策略都有其使用范围和局限性。股市中每年也不乏造富神话，很多人被赚钱效应和造富神话吸引，在没有进行多少学习的情况下就投身其中，这个和还没有学会游泳就一头扎入海中打算捉鲨鱼一样，很大概率会成为代价。而从高息股开始入手，则像在齐腰深的浅水区摸鱼摸虾，一来可以熟悉这个领域，二来多少能捞点小鱼小虾，即便是摔跤踩坑，也不至于伤筋动骨，等到经验丰富了，能力提高了，再慢慢向深水区进发。

举几个例子，其中涉及的股票我都持有或者曾经持有，不构成投资建议。

万科2019年8月15日每股派息1.045元，如果对标理财产品4%的利息，如果万科的股价是26.13元则相当于买入后能获得大概4%的股息，当然这个也不是100%稳定的，比如2020年8月14号每股派息1.0166元。我在2019年开始买入万科是26.02元，最低买入的一次是2020年3月19号，价格24.73元。后来25块26块27块陆陆续续买了一些。买入的理由其实挺简单，接近4%的股息，业绩也不错，估值不高，未来还有增长空间，就算跌了，每年还是有股息，总不能一直低估吧。当然为了应对可能出现的风险，也不能单只股票占太大比例。2020年11月27日收盘价31.61元。

兴业银行2019年6月19日每股派息0.69元，如果对标理财产品4%的利息，如果兴业银行的股价是17.25元则相当于买入后能获得大概4%的利息，而到了2020年7月15日每股派息0.762相对上一年增加了不少。
我  在  2020年2月3号三千股跌停那一天开始买入兴业银行价格为16.81，到了3月19号以15.02的价格又买入了一些。
不知道2021年派息能派多少，还是挺期待的。  2020年11月27日收盘价20.61元。

另外的农行在3.3x也买了一些，在3.17也买了一些。价格在3.17元的时候，对应的股息率为0.1819/3.17≈5.74%。而工行我最低的一笔买入是2020年9月2日价格4.92元，对应的股息率为0.2628/4.92≈5.34%。农行2020年11月27日收盘价3.32元，工行5.39元。

上面这几个都是表现比较好的，再来说两个表现不太好的。

鲁泰A从2015年到2019年，连续5年每年每股派息0.5元，我在2019年派息后开始少量买入，后面跌跌涨涨的过程中做过一些网格交易，结果2020年每股派息0.1元，简直了。由于买入的很少，到现在浮亏小几百元。

海利得2018年和2019年每股派息0.2元，我在2019年派息后开始少量买入，价格是3块出头，结果2020年每股派息0.1元，简直了。由于最近股价涨到了4块多，到现在略微浮盈。

从上面这些例子可以看到，这些曾经稳定派息的股票，通常在不远的未来都会继续派息，大部分的股息是缓慢增加的，也有少部分的股息会减少。
这个就像在浅水区摸鱼，小鱼小虾能摸到一些，如果股价涨了一些也能吃到一  些  肉，碰到派息减少或者股价下跌，也不至于伤筋动骨。

萌新进入股市，进入了一个陌生的领域，首先不要妄图赚大钱，先在这个领域站稳不要被那些浪卷走成为别人的食物。先在浅水区摸小鱼小虾，熟悉规则积累经验锻炼心态，然后再逐步向深水区进发。

另外上面提到的股票只是我的组合中的很小的一部分，我的组合也有一定比例的成长股并不全是高息股，不过所有的股票的股息都在2%以上。

高息股组合，不是最好的组合，但是却是比较合适股市萌新的组合之一。这里我引用一个我很钦佩的投资人对一个没有基本概念年年炒股亏损的人说的一段大白话：

>
> 大多数人买股票，都想找涨得多的，其实这个思路完全反了。每个做股票的人都遇到过牛市，一到牛市几乎所有股票都能翻倍，甚至翻好几倍，但结果下来90%的人都是亏的。为什么呢？因为他们的过程是这样：10块钱买了一只股票，先跌到3块，然后牛市一来涨好几倍，终于回到10块了，刚解套。如果再贪心一点想赚钱，牛市一结束，又回三块了。股市每过三五年总有一个牛市，既然牛市的时候随便什么东西都能翻倍，我们要做的就是找到那些在牛市来之前，不会跌太多的东西！不要去追逐利润，控制好风险，利润会来找你！
