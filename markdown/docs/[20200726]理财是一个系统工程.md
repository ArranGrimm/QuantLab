#  理财是一个系统工程

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

理财最终的目的是能够持续的增值以及获得一定的现金流，同时弹性和抗风险能力也是必不可少的。就像一个系统，各个部件各司其职完成最终目标，也需要定期维护调整，修补损坏的部分，替换不合适的部分。

##  理财投资三要素

流动性、风险、收益这3个要素，不可能同时在一个品种上满足，也就是说不存在一个品种具备流动性好风险低收益高的特点，任何品种都存在对一个或者多个要素的妥协。

##  流动性

个人或者家庭的理财系统，需要具备一定的流动性，也就是需要一定量的可以随时动用的现金类资产，例如活期理财以及信用卡加短期理财的组合。因为信用卡有最长大概50天的免息期，因此遇到比平时开销更大的额外开销或者应急开销可以这个月刷卡，下个月提取一个月期的理财产品还信用卡。

如果投资品种里面有定期派息的品种例如支付宝上面每30天每90天每180天定期派息的长期存款产品，也会大大的改善流动性，每年派息的股票可以参考公众号 增援未来
之前的文章 《 [ 如何正确的认识股息  》
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483661&idx=1&sn=4aea859a3fbdbcbad45a9d5ad10bb4df&chksm=cf17b087f86039918ab00a9f2d1c1967c21f6a58992377feb50f48eade8011beea4a28240283&scene=21#wechat_redirect)
。

用活期以及短期理财的组合保障流动性的策略，可以参考公众号 增援未来 之前的文章《 [ 试试准备开始用利息来支付开销  》
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483691&idx=1&sn=565eee7bb300d12808cd73ac24fba5e6&chksm=cf17b0a1f86039b73a47fcc7e4609bc18afaad69029f8a39f3c6e0025d6b27a436e74214669e&scene=21#wechat_redirect)
。

##  风险

风险的来源有多个方面，需要做一些甄别。按目前的市场情况，收益大于5%的品种都可能存在本金波动的风险。本金波动的风险，可能是浮亏也可能是永久亏损。

以P2P为例，市面上的很多品种就存在本金永久亏损的可能性，因此目前我是远离的。

另外一个风险是由期限错位导致的风险，例如在急需用钱的情况下却只剩下长期理财可以动用，提前赎回将损失即将获得的收益，如果不能赎回则可能需要借短期有息借款承担利息成本。

在低估区域买入的股票，其价值回归的时间可能很长，如果在回归之前由于各种原因被迫卖出将会造成本金的永久亏损。

##  收益

理财组合的收益是组合中各个品种的综合，另外合理使用金融工具可以在一定程度上提升收益。例如在有套利空间出现的时候动用活期部分做一个短期的套利，这个可以参考公众号
增援未来 之前的文章《 [ 货币基金无风险套利实例  》
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483679&idx=1&sn=8ae73323966c93bcb75fff9f82ed1794&chksm=cf17b095f8603983c5e699f20861555bd9ff8746465cf8e766187f62f283e840b741bee3a2dc&scene=21#wechat_redirect)
，也可以让家里人都开通帐号打可转债，目前的情况是一个帐号一年预期2-3千，这个可以参考公众号 增援未来 之前的文章《 [ 可转债无风险套利实例  》
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483687&idx=1&sn=6c803d12c0a31f83a42095b2f91fc066&chksm=cf17b0adf86039bbc58bcab51f6096f27b7ae62eebe24bca6695255984f774b07d38f206b9f7&scene=21#wechat_redirect)
，还有一个就是打新股，如果股票市值较大的时候可以分帐号比如一个帐号30w深沪大概4比6的比例，这样可以提高打新的中签概率，当然要先考虑开通各个权限。

##  结语

总的来说，理财不是无脑买入某个品种就完事了，而是要用不同的品种组合起来，让一部分品种提供流动性，一部分品种抵抗风险，一部分品种提高收益，使整个系统保持动态的平衡，在面对各种情况时可以根据实际需求按层次动用资金，动用后需要再次补充使整个系统再次回到平衡的状态。同时也需要根据自身情况调整比例，剔除不再合适的品种，添加新的品种。虽然需要投入的时间并不多，但确实是一个需要长期维护的系统工程。
