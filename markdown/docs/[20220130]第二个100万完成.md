#  第二个100万完成

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

上一个春节前完成了第一个100W，时隔一年，这次春节之前完成了第二个100W。

当然并不是一年增加了100W的积累，因为上一个春节前完成100W可投资资金的时候是超过了100W的。

这一年的积累是自工作和投资以来金额最大的一年，主要部分来自于家庭收入的结余，次要部分来自投资收益。

这一年算是典型的双轮驱动的一年(双轮驱动指的是 收入结余投入和资产增长双轮驱动)。

在《第一个100万完成》一文中，我原本预计市值达到300W即可实现不上班的自由，但是实际上并不需要那么多，并且我现在也已经没有上班了(见《退出职场全职投资》)。

当然这会在一定程度上影响双轮驱动的效果，因为我没有在上班了，所以2022年家庭收入中的工作收入部分会减少，带来的结果就是结余会减少，而投资收益并不确定。

第三个100W估计在下一个春节前是完成不了了，除非股市大涨，但是这个可能性非常小。

对于我来说这并不是什么坏事，因为我还在持续的买入，在这个过程中，我是希望股市跌的，至少不要涨，不然我就没那么容易买到便宜货了。

至于股市什么时候涨，对我不是太重要，更重要的是维持好现金流，维持好生活水平，股票的价值总有兑现的那一天，不是通过股价兑现，就是通过分红兑现，这个我在《用数学计算来支撑投资计划(二)》写过，所以涨或者不涨，都行。

接下来的时间我会伺机加仓，今年计划年中之前增持大约50W的股票，因为年中还有几个账户满两年，可以开通科创板，届时将增持的股票匀过去即可开始打新股，希望今年的新股收益不要让我失望。

祝大家在虎年里投资如虎添翼，账户长红不断新高。
