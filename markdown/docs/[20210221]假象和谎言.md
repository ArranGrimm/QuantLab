#  假象和谎言

价格充满了假象和谎言，我先说一件小事。

有一天我去一个文具店打印照片，这个文具店是一个夫妻店，我问老板娘打印照片多少钱一张，老板娘停顿了大概0.5秒说3元，我当时有点疑惑她的反应，进到店里面打印的时候是她老公在拷贝打印等，然后我特意又问了一下老板打印一张多少钱，老板回答我2元，最后我也没有多说什么，按2元的价格结算了。

这事情其实还有几种可能，第一个就是我没有问然后按3元结算了，第二个就是老板听到了也给我说3元然后我按3元结算了，第三个就是价格确实就是3元(即使别人家只要2元)我按3元结算了。所以这个店里打印的价值可以认为是2元，而价格却是不确定的，如果我没有多问一句，我就接受了3元的价格。

这个和股票有什么关系？当然有关系。

上周有个大V第三次卖出茅台价格接近2600元，几乎清仓。在他的文章中说明了他按照之前设定的预期目标按计划卖出了茅台，并写到茅台现在估值很高。然而不少人回复说他卖了就唱空，甚至还有谩骂。

这个就很有意思了，一个个人投资人，卖出了茅台并说高估，对市场能产生啥影响？显然没有任何影响，但是很多(持有茅台或者白酒基金并希望茅台继续涨的)人却见不得有人说茅台高估，仿佛说了就会跌一样。

实际的情况是茅台确实跌了，但是这个也不能说明后面就不会涨，哪怕今年明年茅台涨到3000多也并不稀奇。然而问题在于，假设那个文具店说打印一张照片需要10元，明显报了一个高价，是不是一定没有人接受呢？不一定，并且即使报价15元，可能也会有人接受。但是这个价格确实有点高了，为什么不去问一下其他的文具店。

再说另外一个事情，港股低估值股票。

几个月前港股的股票很多极度低估，我趁机买入了一些。抛开那些老千股不说，那些在A股和港股同时上市的企业以及在港股上市的大陆国企，这些企业的业绩虽然不亮眼，但是经营没有任何问题，然而市场却给出了破产价，为什么？

从公开信息中能看到的理由和分析非常多，什么流动性问题，什么低估总有低估的理由，等等，然而这些信息是真相还是假象  抑或  是谎言？

其实这并不重要，最近港股开始大涨，这些理由开始慢慢的消失在信息的洪流中，取而代之的是什么价值洼地，什么新的方向，等等。

然而这些信息是真相还是假象  抑或  是谎言？  这并不重要，重要的是你在假象和谎言中是按高价结算还是按低价结算。
