上周开通了港股通，前天(12月22日)以港币10.5的价格买入上海实业控股。

不构成投资建议，请勿全仓梭哈。

![](https://mmbiz.qpic.cn/mmbiz/xd7OSHTJMfLbBuOTeXVNblK6uPPdJZyXtaQtlYteHOJCjy1DTZnpvfSacQwkcmkZTkFgh3kPay4mkg5qH5O5KQ/640?wx_fmt=other)
![](https://mmbiz.qpic.cn/mmbiz/xd7OSHTJMfLbBuOTeXVNblK6uPPdJZyXU1yvicBRM0WChFqKlLUGGroOAibEJ5Hib8fXlMQGOO3ZvrF9WfpMMdgXw/640?wx_fmt=other)

汇率折算后人民币价格为8.854。

买入以后会发现浮亏了6%左右，这个是预扣了汇率结算的浮动。
![](https://mmbiz.qpic.cn/mmbiz/xd7OSHTJMfLbBuOTeXVNblK6uPPdJZyXCy31Z2P7kCRoXAyNibtP7XSiaW0e4WtibnAQbhySiaEB8wOQgoaYag0jEg/640?wx_fmt=other)

第二天会还回来3%，剩下的3%会在卖出后还回来。
![](https://mmbiz.qpic.cn/mmbiz/xd7OSHTJMfLbBuOTeXVNblK6uPPdJZyXfPUoWUuL9twXia9ibHTSjFtsKBwk6mBljgOzliaswyia5fLWIQnFHnz2Pw/640?wx_fmt=other)

今天涨了一点，所以加上扣除的手续费浮亏不到3%了。

说下为什么买这个股票，原因就是经营情况没有问题，低估。

![](https://mmbiz.qpic.cn/mmbiz/xd7OSHTJMfLbBuOTeXVNblK6uPPdJZyXagFucdyLAWTx6Px7jxgt01eFPNHwHdNNXPhOPl6DicetR2dDuWiaP1ow/640?wx_fmt=other)

股息7%，扣除20%的红利税依然还有5.6%，能提供现金流，比理财划算。

PB0.29，相当于用114.81亿可以买下这个净资产为397.6亿的公司。

pe5.39，相当于买下整个公司后在利润不增长的情况下5.39年能获得114.81亿的利润，一部分作为现金分红，一部分变为公司的净资产。

![](https://mmbiz.qpic.cn/mmbiz/xd7OSHTJMfLbBuOTeXVNblK6uPPdJZyXFw7EaIIicsLjJ4cmCcgYcibncAfvCV9OYqmWd91WZicnVF7EgFqexmE5w/640?wx_fmt=other)

净资产回报率约8%，以净资产397.6亿为基准，如果维持这个回报率，净资产会慢慢增加。
![](https://mmbiz.qpic.cn/mmbiz/xd7OSHTJMfLbBuOTeXVNblK6uPPdJZyXDRNvKRK0H1jTbjHDWjQtUC585az8C3PBvrSbvMnksSRYnrDxWc0zFQ/640?wx_fmt=other)  

利润逐年小幅增长。
![](https://mmbiz.qpic.cn/mmbiz/xd7OSHTJMfLbBuOTeXVNblK6uPPdJZyX2CibUxwvNrLiaf6M6j7qUrI3yYyxoqTC4jibxEv76aUAkgQlNI8OGQBYQ/640?wx_fmt=other)

上海国企，未来几年内倒闭的风险几乎为零，业绩倒退的可能性也很小。
![](https://mmbiz.qpic.cn/mmbiz/xd7OSHTJMfLbBuOTeXVNblK6uPPdJZyXXBp4R1PNoT7HIg9QHiapCibELwricKg0HY9mgYXutLq7S4oyyCc65gePQ/640?wx_fmt=other)

15年4月，市场给出了330亿的报价，5年过去了，净资产增厚了，市场却给出了114亿的跳楼价。

总体而言，是一个大概率能带来稳定回报的企业，尤其是现在市场给出了一个跳楼价。

引用一个大佬的话：

> 问：港股是不是成估值陷阱了啊？
>
> 答：这就是价投的逻辑如此简单，却能在过去几十年持续赚钱的原因——低估的时间稍长一点，大家就不信这一套了。
