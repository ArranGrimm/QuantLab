#  双低组合第一个转债上岸折合年化1147%

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

我在2021年1月31日发过一篇《  [ 双低可转债组合构建完毕预期年化15%
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483985&idx=1&sn=519e07bedae90b4d631d9e64643a4239&chksm=cf17b3dbf8603acd8f547fa6a5c63a6202d6cd3e2ffc163660572846a90655f8c091c95f820e&scene=21#wechat_redirect)
》，森特转债就是其中的成员之一。

本周五(2021年3月5号)，森特转债开盘直接到了130元。

这么快就有一个成员达到上岸标准，确实出乎我的意料。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfLtzS0KoibtO6yVJ0LQCPyxnOoZvJPPnbdicvWsVy8ib8YOORUIERxFBibPM9IUiccyTyTQxZ5nFfPBQEg/640?wx_fmt=png)

对于双低可转债组合，如果没有出现黑天鹅级别的意外，预期年化15%是大概率能实现的，这个我在《 [ 双低可转债组合构建完毕预期年化15%
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483985&idx=1&sn=519e07bedae90b4d631d9e64643a4239&chksm=cf17b3dbf8603acd8f547fa6a5c63a6202d6cd3e2ffc163660572846a90655f8c091c95f820e&scene=21#wechat_redirect)
》有过论述。  

当时是分批次每次买入一组可转债，森特转债的买入成本为94.83元。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfLtzS0KoibtO6yVJ0LQCPyxnib6EzC36QAjDG0GSEHOAKJZf8jCFe0iaNv5DuQUEZYKQHOJDKf63100Q/640?wx_fmt=png)

按中间时间1月20日计算，相当于一个半月的收益率为37.08%，折算年化收益率1147%。  

当然这只是一个偶然，而且森特转债涨到130元，也只占我的可转债组合的4.27%，而可转债只是我的投资组合的一小部分，还有很大一部分是股票。

森特转债暴涨的原因是正股森特股份涨停，而森特股份涨停是因为隆基股份收购森特股份27.25%的股权。

其实这些我并不关心，因为转债就是带有看涨期权的债券，一个转债的正股由于某个原因大涨带动转债大涨，这个本来就是预期内的事情，没有什么值得花精力去关注的。

虽然森特转债短时间达到上岸标准纯属偶然，但是双低可转债组合中的大部分成员最终都将上岸则是必然，区别只不过是时间问题，而按照历史统计显示这个时间是2.3年。

那么可能今年下半年还会有成员上岸，如果运气差一点，那么明年有成员上岸将是大概率事件。

按照之前的计划，如果市场上还有符合入选双低组合的可转债，那么应该将价格超过130元的成员卖出，换入新的成员。

不过这次我只将50张中的30张按130元卖出了，剩下20张。因为

> 截止到2017年6月
>
>   * 国内市场历史上多数可转债的收盘价格都超过了130元，平均收盘价格为166.3元。
>
>

  

> 从结果上看，历史上发行并且已经到期的可转债中，没有成功实现转股的比例只有不到5%。(统计至2020年6月)

既然历史数据给出了这样的结论，我打算把剩下森特转债，其中10张挂条件单160元以上卖出，另外10张等强赎公告出来了再卖出。

一个转债积累3个经验，也是极好的。
