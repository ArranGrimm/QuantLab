#  发挥散户的优势割了机构的肉

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

在特定的情况下，散户对机构是存在优势的。

比如之前写过的一篇《 [ 散户也有机会割机构的肉-以某企业的债券为例
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483884&idx=1&sn=d48ffd5630f0042f488ea07c8b2e97d1&chksm=cf17b066f86039704a314546ead86576b6c867161d26e8a5f47888bb81c36e280699920b1253&scene=21#wechat_redirect)
》，而这次我则抓住机会割了一次机构的肉，写这一篇的目的主要是记录与回顾，以便下次再次抓住同样的机会。

这个事情的起因是1月8号在雪球看到一个帖子提到一个套利机会  (  操作窗口期已过  ，请勿照搬  )  ，总结一下要点就是：

  1. 中国移动被纽交所强制摘牌，美国资金(例如股指ETF)要被迫卖出。从11月12日特朗普的总统令到现在，中国移动已经从53.95港币跌到42.2港币，跌幅约22%。 

  2. 根据美国的行政命令，美国人员不得在2021年1月11日上午9:30(美东时间)以后从事某些交易。 

  3. 在总统令之后存在大量的做空资金，需要在1月11号之后，结束做空，买入股票。 

  4. 按现在的价格中国移动估值已经非常便宜了，PB0.7，税前股息高达7%。 

因此在看了这个帖子之后，我看了一下中国移动的财务数据，在目前的价格之下，安全边际很充足，即便是买入后价格不会马上回升，长期拿着也是挺好的，毕竟估值低股息高。

胜率很高，赔率尚可，即使不成功长期看也是划算的，因此我决定搞一把。

于是在8号早上直接下单买入，不浪费时间看行情，毕竟功课做在前面。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfI2kZVmicgysDCpxic2301IicGUic50QDc3mE9I07jXn34ZDjVHKwLs2OCktYlIicbt9aXa4duibmQI9lOw/640?wx_fmt=png)

实际上当天最低价是39港币，而我的买入价42港币算不上特别好。

随后周一(1月11号)就开始回升了，不过当天上班忘了这事，到了周二想起来了，一早就下单卖出了，同样不浪费时间看行情，在44.75港币卖出，当天最高价47.45，最低价44.25，我基本上卖在低点(失误贴错图无法修改)。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfI2kZVmicgysDCpxic2301IicGOfX0bFcuznibcO1tXkddVJ5rhXByykIjkUAjG7kNsibibKuZz2z8AwKXw/640?wx_fmt=png)

不过算下来，扣除手续费和其他损耗，每手(500股)获利1300多港币，也是挺不错的，毕竟看帖子看财务数据下单，前后3个交易日合计耗时15分钟收益率约6.5%，性价比高出天际。

作为一个上班，我的投资风格是低风险不费神，花极少的时间在投资上然后长期持股，不过碰到有套利的机会也会搞一把，这样既赚到了钱又获得了乐趣，就当是投资历程中的娱乐。
