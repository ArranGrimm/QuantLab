#  在15.94元卖出最后两手中文传媒

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

4月3号先卖了10手均价13.3x，当天一度涨停，涨停板13.64又卖了10手，还剩10手

本周三(4月12号)早上大涨，又卖掉8手，均价14.41，后来涨停直到收盘

还剩下最后两手在另外一个账号，因为挂了15.48卖出的条件单，所以就暂时没有管

周四(4月13号)竞价直接涨停，条件单触发挂了15.48的卖单，不过因为价格笼子限制成为废单，于是手动挂了涨停板的卖单

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfKPqo5AmP3WtOWMNoG2TTQicYADBzaxia2Bu12pibJKIOzjfagblPUQW6J1827qCMZzB2hZEv0a902kA/640?wx_fmt=jpeg)

至此，卖出了所有的中文传媒。

这一轮建仓开始于2022年3月，第一笔买入价是11.1

后面10块多9块多8块多都买过，最低买入价是2022年9月26号的8.57元

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKPqo5AmP3WtOWMNoG2TTQicCbjNaLvcGrv2YexYaZZf8l2gMcxMIZ8ibg0Z3YZzyFQKicXkEWD3RwSw/640?wx_fmt=png)

图片来自我的2022年度总结中的持仓汇总表

之前10块左右卖了一点，然后就是最近12块左右一路卖出

14个月的时间收益率34%，这个收益率我是满意的

最开始买入依然是基于我一贯的投资逻辑：低估高息

只不过没想到第一笔买入后能跌那么多，最后一笔买入相对第一笔买入跌了接近23%

跌的过程中也有不少负面评价

比如什么出生人口减少了，以后买教辅的少了

比如什么游戏业务不行，什么什么的。

要说游戏业务不行，还能算个像样的理由

说出生人口少了，我觉得实在是扯的有点远

2016年是最近几年的出生人口高峰，这一批小孩在2022年和2023年刚进入小学，明明是消费者数量增加了，要说降也是2023年和2024年从高点降下来，反应在财报上也要到2025年了

然而说这些理由的人，有几个人持股会超过一年？

而且这个理由的逻辑也是非常脆弱的，小学生如果增加了课外阅读书目呢？

小学生现在学校要求多少指定阅读书目，我多多少少还是知道的。

当然这个不能作为投资的指导，因为这个因果的关联度有多大，逻辑链条有多长，我能力有限无法判断。

不过我可以判断的是，说小学生减少影响业绩，同样也是不靠谱的。

既然无法得到一个靠谱的逻辑，还是按最简单的逻辑来吧

那就是 **低估高息**

至于最近传媒热门，资金都在追捧什么的

我既无法提前知道，也不知道市场的热情会在什么时候消散

那么我只能用简单一点的方式，在性价比相对其他持仓的股票变低同时单股持仓占比偏高的时候，开始分批卖出

网上看到不少人在等16块以上甚至20块以上卖出，每次看到类似这样的帖子基本上预示着这一波差不多到头了

去年千金药业的一轮上涨，我计划在17块卖出大部分，最终最高价只到了16元，那一天前后的换手率大概是平时的两倍

而中文传媒周五(4月14号)的换手率是前几天的两倍，是之前不温不火的时候的几十倍，大概率是在派发筹码了，并且估值和股息回报已经没有太大的吸引力了，该是离场的时候了
