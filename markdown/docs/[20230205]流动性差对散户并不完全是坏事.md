#  流动性差对散户并不完全是坏事

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

通常散户对流动性差的股票的印象是：成交不活跃，没人关注，容易长期阴跌

而且这样的股票通常市值比较小，遇到一点利空很容易大跌甚至跌停

但这并不完全是坏事，甚至可以反过来变成有利条件

比如 双箭股份 就是这样的股票，我在2021年底7块多的价格开始买入，并在2022年5月写过一篇《 [ 双箭股份的买入逻辑
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484739&idx=1&sn=51655edfdb972b497ff9f73c521c0c83&chksm=cf17b4c9f8603ddf7cdf77da2b9acbbc6cefb1beed93fe653534987fbe413d004d379e1130f2&scene=21#wechat_redirect)
》当的价格已经跌到6.15

它的成交量很小，通常一天的成交量 也就是2万手到3万手，有时候一天不到2万手(金额不到一千万)

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIEPObkn3GnmL7bRmHib4xdp30ESwUnRn7KTUhEzJTUr43onQukv8YO5P2nicyNWn8d4tcgoGIia9TbA/640?wx_fmt=jpeg)

经常连续几分钟没有一笔成交

因为市值小(目前不到30亿)，成交不活跃，一旦遇到大盘整体跌的时候基本上就是跟跌，而大盘涨的时候却不一定跟涨，以致于很多散户对此非常嫌弃。

但是如果反过来想，这样的情况，是不是非常适合低吸？

流动性很差的股票由于流动性差，波动性比其他股票会大很多，而如果已经有了估值之锚(  比如上周我写了一  篇《 [ 用实例简述我的估值之锚及交易
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247485079&idx=1&sn=1db9eba82fef9dc794f85c1e75c611cd&chksm=cf17b71df8603e0b8b9b21c0767ee124de6e4055e04563626efd548f1cf5b7dd30a383955b5b&scene=21#wechat_redirect)
》  )，那么就可以在波动中遇到自己的目标价。

比如我建仓的时候价格是7.5左右，2022年4月下半月上证连续大跌的时候，它也不例外的大跌，于是4月29号我在5.58的价格加了一笔

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIEPObkn3GnmL7bRmHib4xdpxiaBooquOWl4D1YtQfwIYDl1kTxiaCc2VcLTpMTvthLbuYshIjQEMiaMg/640?wx_fmt=jpeg)

而2022年9月下半月上证连续跌的时候，它同样不例外的跌，于是9月30号我在5.58的价格又加了一笔

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIEPObkn3GnmL7bRmHib4xdpMicFW8uwTKkTJhPuOwjiaibkHt7Hhd7STULBg3zSIZUfqlWfBrpwgrGIg/640?wx_fmt=jpeg)

可以看到，大盘跌，它通常就是跟跌

而在这个过程中，它由于养老概念，大涨几次甚至涨停

比如2022年12月19号那天涨停，于是我在涨停板6.82卖出了一部分，当天破板又封板

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIEPObkn3GnmL7bRmHib4xdpOHsiaX7Y6mKx8E7Zkjr9pj4CDtZCd4n3U3wqnPXsQo3UQHiciaUY3WibRw/640?wx_fmt=jpeg)

成交量大约是平常的10倍，随后几天又大跌了，于是12月23号我在5.91的价格又接了一部分回来

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIEPObkn3GnmL7bRmHib4xdpiaRtib31cja1g6YGdlrq8vs3yDwgRNlWQeBQkk5pG0xiaa6xvSezG4Ocg/640?wx_fmt=jpeg)

而2023年1月18号，我又在6.88的价格减持了一部分

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIEPObkn3GnmL7bRmHib4xdpzFN522qXhps6qIiaIffMb4EJm3ep3yu1pbg1OQtXC1xzu1adnbcloLg/640?wx_fmt=jpeg)

经过几次做T，加上2022年5月派息

我现在的持仓成本是6.52，持仓占比大约是3.7%

流动性差导致的跟随大盘的大跌，给了我低吸的机会，题材概念引起的大涨给我了高抛的机会，让我从建仓时的成本7.5降低到如今的持仓成本6.52，这都是流动性不足带来的机会
，不过前提是需要有足够的耐心同时资金期限也足够长，如果是短期资金可能就会吃亏。

当然，并不是说流动性不足的小市值股票就一定值得关注和投资，这些都是表象，最关键的还是企业本身。

而关于为什么要买入双箭股份，《 [ 双箭股份的买入逻辑
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484739&idx=1&sn=51655edfdb972b497ff9f73c521c0c83&chksm=cf17b4c9f8603ddf7cdf77da2b9acbbc6cefb1beed93fe653534987fbe413d004d379e1130f2&scene=21#wechat_redirect)
》已经写过

而我关心的是这个

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIEPObkn3GnmL7bRmHib4xdp6WpNATt8FtezCqwKQE1xMtLWKicoqKBA5eokicVDxvib1Bagmz4MmpE5w/640?wx_fmt=png)

期待它在扩产后，遇到一个原材料价格下降的机会，经历一次利润与估值双重提升的戴维斯双击，走出一个类似 伟星股份、梅花生物 这样的翻倍或者3倍之旅。

当然，为了控制风险，我依然只分配给它5%的仓位上限，如果接下来的时间还有大跌(预计今年年报利润不乐观，分红可能也不太行)，我会伺机加仓。
