**“没有的幻想躺平，拥有的不会躺平。”**

而且实际上每个人对躺平的定义也不一样

有些人是把钱存银行，用利息维持最低生活

有些人则要有房有车，然后还要时不时能旅游

不一而足，众说纷纭。

作为一个已经脱离了工作岗位的个人投资者

我现在做的最多的事情就是通过指标筛选上市公司的股票和可转债，然后翻阅相关数据

虽然没有上班，但是每天还是在做一些事情，我也不知道这个算不算躺平

为了能将这种状态持续下去，我的投资收益肯定是要跑赢通胀一定程度的

而我在之前关于跑输通胀两个点的理论计算中发现了一点有趣的结论，今天稍微扩展一点

前面那一篇中通过理论计算，发现这样一个结论

**初始资金1000W，第一年开销10W，开销随着通胀增长，如果收益跑输通胀两个点并且通胀率不超过20%的情况下，这些钱能维持的时间在55年到59年之间**

    
    
    # -*- coding: utf-8 -*-  
    def func(remain, cost, inflationRate, rateOfReturn):  
        year=0  
        while remain>cost and year<100:  
            remain-=cost  
            print "第",year+1,"年花销 ", "%.2f" % cost,"万，本金剩余 ","%.2f" % remain,"利息收益 ","%.2f" % (remain*rateOfReturn),"合计剩余 ","%.2f" % (remain*(1+rateOfReturn))  
            remain*=1+rateOfReturn  
            cost*=1+inflationRate  
            year+=1  
      
      
    func(1000,10,0.02,0.00)

func(1000,10,0.02,0.00) 表示 初始1000W，第一年开销10W，通胀率2%，收益率0

通胀从2%到26%，收益率从0%到24%(跑输2个点)的结果如下

通胀2%，收益0，55年后钱不够继续维持生活

通胀5%，收益3%，56年后钱不够继续维持生活

通胀10%，收益8%，57年后钱不够继续维持生活

通胀15%，收益13%，58年后钱不够继续维持生活

通胀21%，收益19%，59年后钱不够继续维持生活

通胀26%，收益24%，60年后钱不够继续维持生活

**所以我们能够发现，重点不在于通胀率到底是多少**

**而在于我们的收益率和通胀率之间的差异**

**如果收益率能和通胀率持平**

**那么不管通胀率是0%还是20%，都可以维持100年的生活**

而作为一个投资者，跑赢通胀是最起码的要求

那么如果跑赢通胀2个点，结果会怎样呢？

按起步1000W资金，起步开销10W，显而易见钱肯定是花不完的

并且还有一个非常有趣的结果，那就是大概在第57年，剩余资金会达到当年开销的200倍

既然钱花不完，那么就并不需要1000W那么多

杰里米·西格尔教授在《投资者的未来》  一书  中统计

**从1801年到2001年200年间投资美国不同资产的回报率，得出股票扣除通胀因素的年化回报率达到6.6%，如果考虑2~3%的通胀率的话，那么股票长期投资的年化为8~10%。**

那么以第一年开销10W为基准，跑赢通胀7%，需要多少资金才能将投资养活自己这个事情维持下去呢(维持70年的生活)？
