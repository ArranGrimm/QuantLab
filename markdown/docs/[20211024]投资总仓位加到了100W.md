#  投资总仓位加到了100W

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

周五加仓了8W的股票，总仓位(股票+转债)达到了100W，然后留了一部分现金作为备用资金。

这是继今年2月《第一个100万完成》之后一个新的里程碑。

这个是主账户，已清仓的股票盈利和已卖出新股收益没有累加，所以这里显示收益为负数，实际上收益为正数。

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfJZGn2BPIt1Px5eTzJtAhoSy66m7LOTx3Tgqb5anibmKtico7wWW8QTks7uNsLTgozVMdMhWzKO0Tzg/640?wx_fmt=jpeg)

这个是转债账户，已经清仓的转债盈利没有累加进去，这里显示盈利3W多，实际盈利5W多(从最高盈利6W多回撤下来)

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfJZGn2BPIt1Px5eTzJtAhoSb5VMCf1mds0ygqf0GV81sYvviaPACStGia1DEzpkYUgYjBWnVXaWLz9w/640?wx_fmt=jpeg)

这两个是准备用来打新的账户，持仓一模一样，有清仓的股票亏损没有累计进去，实际收益为亏损几百块  

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfJZGn2BPIt1Px5eTzJtAhoSxSP1IHaZsEIG5Ribc9B9iao3V5GaUYGKxkE8ROYtBaXRQRBNzgUpoMnw/640?wx_fmt=jpeg)

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfJZGn2BPIt1Px5eTzJtAhoS0KpRsL6T14KyxXPOmLa6623ZGjic4QRyaFicrxm7RhSugekxLqkM89og/640?wx_fmt=jpeg)

加仓的股票是这些，选择这些股票带有强烈的个人偏见，不作为投资建议。  

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfJZGn2BPIt1Px5eTzJtAhoSvqJsxIuTUhrrA1my4VTJIicBM21vz8ibr8jD6g5ia7z0OyfRicrzDlnGGw/640?wx_fmt=jpeg)

加仓光大银行主要是为了投机，这个在《用小仓位来做一次基于强赎的投机》有写过。

威孚高科已经在自选里面呆了很久了，这次加仓主要是为了平衡打新门票的比例，需要多增加一些深市门票，考虑到威孚高科的业绩尚可估值偏低，计划总仓位4W，这次建仓先买入一半。

双箭股份也在自选里面呆了很久，加自选的时候是8块多，现在7块多，也一并建仓。它是橡胶传输带的龙头，这一段时候由于原材料上涨和运输价格上涨导致业绩变差，对于优质企业这样的情况可能是一个好机会。

其他票都是在现有持仓的基础上增加数量。

之前还有一个在自选里面，准备建仓的鲁阳节能，没想到现在成了这个样子，暂时放弃。

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfJZGn2BPIt1Px5eTzJtAhoSBufQ5tF2QtpKauiaup7nb5H767039Wp8AlYtTeKrmiaE5FrXGytXEUXA/640?wx_fmt=jpeg)

明年会根据现金流以及现金储备的情况，将总仓位增加到120W-150W的位置，那么股息和打新带来的被动现金流将进一步增加，虽然这次有个科创板新股破发，但是打新红利不至于那么快消失，并且即使打新红利消失，市场大概率会出现新的超额收益的机会，这个在历史上已经发生过多次，过去是分级基金和折价封基，现在是科创板新股和可转债，未来是什么现在还不知道，但是总会出现。

继续囤资产，  一切都在稳步推进。
