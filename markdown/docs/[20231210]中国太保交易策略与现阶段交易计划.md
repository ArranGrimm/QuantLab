#  中国太保交易策略与现阶段交易计划

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

交易策略是中长期的，交易计划是分阶段的

如果企业的基本面以及参与者的风格没有发生显著的变化，策略一般不做大的变动

而阶段性的业绩和市场环境变化，则需要阶段性的对交易计划做出调整

本周二(2023年12月5日)，按计划买入一小笔中国太保，价格22.98元(当天最低价22.89元)

今天从价值基本面、价格与波动以及预期收益率3个方面来说一下交易策略和交易计划的形成逻辑

  

####  价值基本面

基于价值或者说从价值出发，是普通散户的安身立命之本

虽然这种策略快速赚钱的概率很小，但是亏损的概率也很小，大概率可以长期赚钱。

先看看它的基本情况

企业属性，其他国有，大股东持股不太集中，勉强算国企

2018年到2022年年度利润分配，利润分红比例35.9%到50.87%，分红比较慷慨

目前PB 0.94 估值处于10年百分位的4.2%

距离10年历史最低位PB0.78(2022年10月28号，不复权价格18块)

按估值还有大约17%的下跌空间 (0.78/0.94 - 1)

按股价还有大约23%的下跌空间 (18/23.31 -1)

已经处于低估值区域

至于公司的保险业务，行业分析什么的，不用花太多精力研究，因为低估分散策略不太在乎业绩的增长，都已经破净了，还研究啥增长

只要企业这样稳住就行了，长期看只要增长能跟上GDP的节奏就行，也就是平均一年5%左右，大概率还是有的

经济环境稍微好一点或者市场稍微火热一点，又或者哪一年业绩稍微多增长一点，就能涨不少。

所以就不需要花太多精力研究公司了，交给公司的管理层就行了，散户就是搭顺风车赚钱而已。

  

####  价格与波动

再来看一下价格，由于没有进行送股转股，直接看最近10年的不复权价格变化

2013年12月到2023年12月，不复权最低价格14.94(PB 1.38，PE 15.18)，最高49.12(PB3.19，PE 30.55)

再看最近5年的价格变化

2018年12月到2023年12月，不复权最低价格17.92(PB 0.78， PE 7.1)，最高47.14(PB 2.03，PE 17.73)

可以看到价格最低在15块左右，最高在40多块

PB低位在1左右，高位在2到3

PE低位在10左右，高位在20左右到30，PE受当期利润和当年利润的影响较大，因此变化较大

可以看到波动还是挺大的，并且最高价出现在2017年11月21号，驱动因素是2017年10月28号发布的三季报，单季度利润同比增长64.38%，股价在市场的追捧下被推上了高点，高位减仓的爽了，高位接盘的惨了。

  

当前(2023年12月8号收盘)估值 PB 0.93，PE 12.96

PB处于低位，PE处于中位偏低，主要是2023年的利润相对2022年有所下降，因此使PE并没有到达低位

如果接下来的一两年利润稍微改善，而股价不变的话，PE将会降低

所以现在属于可以买入的区域，当然买入后还有可能继续下跌，但是下跌的空间已经不大了，而向上的空间则要大的多。

  

####  交易计划与预期收益率

基于目前的市场热度，可以将价格中枢定在27到30块之间，对应PB 1.09到 1.21

5档买入价定为 27，25，23，21，19

对应PB分别为1.09，1，0.93，0.84，0.76

对应股息回报分别是 3.77%，4.08%，4.43%，4.85%，5.36%

5档卖出价定为 30、32、35、38、41

对应PB分别为1.21，1.29，1.41，1.53，1.65

对应股息回报分别是 3.4%，3.18%，2.91%，2.68%，2.48%

  

如果在两年内买入前3档卖出前3档，则整体收益率为 (30+32+35)/(27+25+23)-1=29.33%，折算年化大概是13.7%

如果在5年内买入5档并卖出5档，则整体收益率为(30+32+35+38+41)/(27+25+23+21+19)-1=53%，年化大概是8.87%

注意，这还没有包含每年的股息，并且可能在这个过程中交易不止一轮。

  

当然可能存在更极端的情况，比如买入5档后几年没有机会卖出，那低位买入的优势就体现出来了，每年的股息收入可以极大的缓解压力，并且用不了几年也能回本

而如果几年都没有机会卖出，可能是企业业绩没有增长，那么一旦增长市场就会开始追捧

或者是企业的业绩有小幅增长，但是股价没涨，因此估值更低了，只要市场回暖，股价也会大涨。

当然，还有更差的情况就是企业越来越差了，为了预防这样的情况，最高仓位最好不超过5%

  

为什么定的5档买入价，比历史最低价高出不少呢？

因为如果企业正常经营发展，股价的底部通常是慢慢抬高的，另外历史最低价具有一定的偶然性，可能是市场的极端情绪或者极度缺乏流动性造成的，下一次的低点出现时可能情绪的极端程度或者流动性的缺乏程度比之前稍微好那么一点点，结果可能就是价格比之前高那么一点点，所以就不要纠结最低点了。

另外卖出价格也是一样，不要奢望卖在几年内的最高点，万一等不到就白坐一趟过山车了。

  

所以在目前的业绩和市场环境下，这个计划具备很大的安全性，同时预期回报率也是比较可观的。

虽然价格波动无法预测，但是我们可以等它再次出现。
