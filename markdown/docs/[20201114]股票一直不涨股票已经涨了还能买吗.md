#  股票一直不涨/股票已经涨了，还能买吗？

__ _ _ _ _

上周(2020年11月2号~2020年11月6号)，三傻(银行保险地产)涨了不少，这周又掉回去了一些，优质企业的股票就是这样在一波三折中慢慢增长，而这样会让投资者经常需要面对两个问题：

  * 股票一直不涨，还能买吗？ 

  * 股票已经涨了，还能买吗？ 

在讨论这个问题之前，让我们先把目光从股票上移开，来看看一个假设的模型。

假设有一个连锁火锅企业，有十家店，每家店报价100W，每家店每年的利润是10W，整个企业的年利润是100W。现在市场上交易这个企业的股份，1%的股份报价10W。

那么买下整个企业需要1000W，每年利润100W，相当于十年回本。回本后每年的利润是10%，看上去还算一个收益合理的投资，于是用10W买下了1%的股份。

第二年这个企业用100W利润开了一家新店，现在有11家店，按过往的历史，今年的利润将会达到110W，如果每家店价值100W的话，这个企业总价值是1100W。

这个时候会出现以下几种情况：

  * 1%的股份报价没有变，还是10W。 

  * 1%的股份报价跟着企业增长了，是11W。 

  * 1%的股份报价超过了企业的增长，是12W。 

第一种情况在万科A上就发生了。

万科A(不构成投资建议)的前复权股价2017年9月最低20.24元最高26.96元，2020年9月最低26.93元最高29.56元，如果按26块多的价格来说，可以认为股价3年没涨，并且2020年10月29号最低到了26.62。

再来看看利润

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKSt1GjVngp8rEic2O8AviaW1oZ2oFWuU7M7lpph1T4rx1mKkNcN5Rdeta931WIQv64WLGQOWiaicY4pg/640?wx_fmt=png)

2017年的利润是280亿，到2019年增长为337亿，对于2020年的利润，同花顺上30家机构给出的预测最低是428亿，平均值是447亿。如果再悲观一点，假设2020年净利润是400亿，相对2019年的增长率约为18%，那么股价还有可能继续不涨吗？这个可能性是存在的，这个时候同样价格的股票比一年前更加划算了，如果之前买入的理由是看好企业的业绩，那么现在这个买入的理由依然存在而且票价更划算了。那么如果买入以后还是不涨怎么办？来看一下分红

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKSt1GjVngp8rEic2O8AviaW1mnpgUBLu80B4zibwAKk1MnCXWXLp1nSyAUc9wBiaialWxjHGMxPX9M9ug/640?wx_fmt=png)

如果企业的利润一直在增长值，分红也是上涨趋势，如果打算长期投资，股价不涨，那可是难得的好事。

第二种情况和第三种情况，在一些股票上也出现过，这里不再一一列举。

所以现在的问题是：股票一直不涨还能买吗？股票已经涨了还能买吗？

对于一直不涨的，相对简单一点。看看企业的过往业绩。如果一直在增长并且现在的价格相对利润以及利润增长率以及净资产回报率是划算的，如果打算长期投资那就值得继续买入，当然为了防止未来可能出现的不利因素，需要分散。

对于同步涨的，和不涨的类似。

对于超额涨的，那需要慎重考虑，如果觉得未来增长强劲，也可以适当买入，当然也可以考虑买入其他的股票。

这些都是针对长期投资来说的，如果是想短时间获利，则是另外一个故事。
