#  买盈利VS买资产

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

巴菲特侧重买盈利，施洛斯侧重买资产

上周写了一篇《 [ 可复制的成功投资和不可复制的成功投资
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247485099&idx=1&sn=18d6308d72c08fe3d252e07ff5b06d27&chksm=cf17b721f8603e37f15f8bd735b2eb95771495778e53143e06b995ab5ff55ac1db30154348b2&scene=21#wechat_redirect)
》来说明价值投资的成功更容易复制，而且在价值投资流派中，施洛斯风格的投资方法门槛更低更简单易学，更适合普通散户。

这周刚好花了一点时间看完了《沃尔特•施洛斯资料集》，其中有一段

> OID：在看股票时，你们与格雷厄姆或巴菲特有什么不同？
>
> 沃尔特•施洛斯：从根本来说，我们喜欢买资产
>
> OID：为什么是资产而不是盈利？
>
> 沃尔特•施洛斯：资产没盈利波动的那么厉害

非常精准的说明了巴菲特和施洛斯的区别

巴菲特早年和施洛斯的投资风格是差不多的，并且两人在格雷厄姆的公司共事了两年。

后来两人都成立了自己的基金

在1962年，巴菲特将手里的一部分烟蒂股打包卖给了施洛斯

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIpVug3xZtrpkuRL2JXmicTKEO85z2lxxibiaAw9ITSFOZtIUDXxzkMYOCxoql73wzoBOXiakxY4pgEQQ/640?wx_fmt=png)

因为巴菲特的资金太大了，继续搞这些烟蒂股已经容纳不下了，并且巴菲特的商业眼光和能力也是非凡的

普通人想要学，先掂量掂量一下自己的能力

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIpVug3xZtrpkuRL2JXmicTKRhCEuawd0s0jibYJvlSczY7EiaSoTjEZ2QKibVhsk5s4TI5jtpBBLI8Bw/640?wx_fmt=png)

巴菲特这个商业能力，在唐朝的《巴芒演义》里面也有写到

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIpVug3xZtrpkuRL2JXmicTKA9AphZn4Ha6icdwGicqhhwibzAP9P9nTDgCF84m2nR2NwV48kNWzcyVpg/640?wx_fmt=png)

（注：凯瑟琳是华盛顿邮报的总裁，凯瑟琳的老公恰好也姓格雷厄姆）

所以对于买盈利还是买资产这个问题，我选择的是买资产，因为我肯定没有这个商业能力(有这个商业能力我就去创业了，比投资赚钱快的多)

而市场上选择买盈利的人居多，因为确实很酷，手里的股票都是关注度高的股票，参与感非常强烈，而且如果买对了，成就感也是非常强烈的(感觉自己比肩巴菲特了，能不带感吗)

因此市场对盈利的关注也是非常强烈的，一个季度利润涨了，股价的涨幅可能比利润的涨幅多的多，一个季度的利润跌了，股价的跌幅也可能比利润的跌幅多的多

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIpVug3xZtrpkuRL2JXmicTKnqhn7YnD5TZkwFvdrhzgXicnYErOOuwicLcqRHYicAat0fpoddSY07xUg/640?wx_fmt=png)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIpVug3xZtrpkuRL2JXmicTKnoHZCBWpzukBQGhc3lnldyaZHPYOExqlrFr0XvZHwexp039zRNeAPQ/640?wx_fmt=png)

感觉用这样的方式投资难度还是太大。

而用买资产的方式，则简单的多

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIpVug3xZtrpkuRL2JXmicTKvibgibGHibyaW6VSTqjKyp0UlkhfsAHrEA1C4ia2Fe4HVOFoDeABe6ibbuw/640?wx_fmt=png)

比如我在2021年2月以8块多的价格(0.74PB)买入兖矿能源，在2021年7月8月9月逐步卖出最后在PB 3.x的位置出清，当时还写了一篇《  [
胜率、赔率与价值投资
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484372&idx=1&sn=cfce52fd7ed957c435a3891ccaa30202&chksm=cf17b25ef8603b4867eb8586056711689d93f0899500c399be19aa534141acbc3d832543765c&scene=21#wechat_redirect)
》来回顾，现在想想，这不就是施洛斯的投资策略吗？

比如建发股份

建仓的时候是8块左右(PB0.6x)，卖出的时候是13到15块(PB1左右)，同样写了一篇《 [ 建发股份投资获利回顾
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247485004&idx=1&sn=ca48155da85e1f7a0afbbcaeb61fbb4b&chksm=cf17b7c6f8603ed03d528e89163df71ccad87db8c9b026f1cb82e56297ddbff131dd82b61e48&scene=21#wechat_redirect)
》来回顾，这不也是施洛斯式的投资策略吗？

虽然每一次都赚的不多，但是积少成多，总体下来收益率还是可以的

施洛斯的基金每年将卖出后获得的利润用分红的方式分配给客户(客户也可以选择复投)

所以对于个人投资者，每年都会有一些股票可以获利了结，既可以作为生活费，也可以复投(特别是某一年如果整体浮亏的话，说明市场上的便宜货很多)

这不就是稳稳的幸福吗
