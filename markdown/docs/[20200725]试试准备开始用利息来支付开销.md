#  试试准备开始用利息来支付开销

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

注：本文写于2019年11月，文章举例的品种已经发生了变化。目前我已经将活期部分从余额宝更换为民生银行的活期理财，收益大概是3%左右，其他比如兴业光大招商都有类似的品种，可以自行选择，定期部分已经逐步替换成支付宝中的定期存款产品，派息周期从一个月到一年不等，利息从4.0%~4.88%不等。品种虽然有所变化，但是思路和方法是不变的。

\---------------------------------

如果每年能有半个月薪水的利息用来犒劳自己，是不是一件很爽的事情？来看看如何做到。

首先需要明确一个概念：任何金融工具都不可能同时满足收益率、安全性、流动性，也就是说所有的金融产品都在这3者之间做取舍，例如定期相对活期收益高，但是定期的流动性就不如活期，而通常对于现金管理都强调需要留3到6个月的生活费备用金以备不时之需，因此需要将不同的产品组合搭配，达到兼顾安全性和流动性且具备合理预期收益的目的。

以月薪6000元为例，如果实际工资高于这个标准可以自行按比例计算，比如12000或者18000或者更高，等等，具体数值不对方案产生影响。

任何理财投资的前提是需要有结余的钱，如果长期月光，那么所有的方案都是无源之水无本之木，因此先假设储蓄率为1/3，即6000月薪，每个月开销4000结余2000，12000的月薪每个月开销8000结余4000，以此类推。

特别注意，本文说明的是一种思路，涉及到的品种只是用来举例，实际上适合的品种很多，例如民生银行和光大银行提供了利息为4%左右的活期理财(只能在工作日申购和赎回)。

第一阶段，积累备用金。

将每个月2000元的结余存入余额宝，连续3个月可以累积6000元。第4个月开始，将结余存入支付宝中定期30天的理财产品，目前支付宝中非净值型货币基金产品30天期最高收益的品种3.71%(2019年11月26号)。存入以后选择自动续期，第5个月存入的时候将时间向后推5天，即假设4月是10号存入，则5月15号存入。每一笔都在之前最后一笔续期后5天存入，这样在存入6笔后将形成一个滚动资金，每5天有一笔理财到期可以取出使用，如果不需要使用则自动续期。经过9个月存入9笔后，结余资金为余额宝6000元，30天理财产品12000元且每5天有一笔可用。此时结余为4.5个月的生活费，完成第一阶段目标。

第一阶段完成后每个月利息收益大约为50块，一年大概600块，虽然不起眼，但是毕竟啥都不干一年有600块，相当于十分之一的月薪，可以买一张机票了，如果月薪12000，则一年利息大约1200，可以买来回的机票了。

第二阶段，提高收益率。

在第一阶段完成了4.5个月生活费备用金积累以后，继续存钱，此时将选择周期更长，收益更高的品种，目前看到有期限为360天收益率为4.46%的品种(还有收益更高的品种，净值型，也就是说存在本金的波动)，将每个月的结余存入，连续存12个月，则一年后每个月将有一笔理财到期可用，如果不需要用则可以继续存入(没有自动续期功能)，此时完成第二阶段。

第二阶段完成后，储蓄情况为：余额宝6000元，30天理财12000元，一年期理财24000元。每个月利息收益为余额宝13元，30天理财37元，一年期理财约89元，合计约139元，一年约1700元，略大于1/4个月的薪水。

第三阶段，再存12个月的一年期理财。

由于在第一阶段已经积累了4.5个月的生活费备用金，并且在第二阶段积累的一年期理财每个月有一笔到期可用，因此在一个月内可用动用的资金为4.5个月生活费加一笔0.5个月生活费的一年期理财，也就是5个月，而每个月都有一笔一年期理财到期，也就相当于3个月内能动用的资金为4.5个月生活费+1.5个月生活费的一年期理财，相当于6个月的生活费，对付短期的问题例如急需用钱，换工作等已经非常充裕了，因此这一阶段的结余将继续存入一年期理财，再次存满12个月则第三阶段结束。

第三阶段结束后，储蓄情况为：余额宝6000元，30天理财12000元，一年期理财48000元。每个月利息收益为余额宝大约13元，30天理财37元，一年期理财178元，合计228元，一年约2800元，约等于半个月的薪水。

以上方案完成后，每年利息约等于半个月薪水，月薪6000的话利息可以用来支付小长假的自由行，月薪12000的话利息可以买iPhone11。
