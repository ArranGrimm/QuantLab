#  用数学计算来支撑投资计划

今年(2021年)春节前我的可投资资金超过了100W，然后我逐步开始增加股票的份额，以优质低估高息的企业为主，分散增加，同时也增加了一个双低可转债组合。4月开始在家人的账户里面分批增加股票，准备构建打新账户让家人来打新股。

目前我自己的账户是这样的

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfI77NX8V1bPoHxTpDXde2s1JD2ILLbNooRznWNBiaqeITUTG1JE6k52HtW0lxBDtDDVpStHfMbtrWQ/640?wx_fmt=png)

在占比高达11%的平安从年初的86.98元跌到目前的70.1元，跌幅约20%的情况下，还能取得微弱的正收益，已经在一定程度上说明了我的组合非常稳健。

可转债账户是这样的

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfI77NX8V1bPoHxTpDXde2s1VCQwHfm52xNWRGL2uIibG9OI5xtuCjFibafCK5XdI8yLxL7LlAhn7GGA/640?wx_fmt=png)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfI77NX8V1bPoHxTpDXde2s18ia0Ra0otnse41oGIeoUAQHlxYMmXYNaXY4SyDDiapbWLUiaM83bdlYUA/640?wx_fmt=png)

已清仓的转债不会计算在持仓浮盈里面，因此实际浮盈是10045+2518+555，大约为7.7%，另外家里人用于打新的两个账户目前市值较小，浮亏约1%。

整体市值目前接近80W，整体浮盈接近2%，考虑到股票部分的加权股息率是4.93%，如果派息后填权同时还有可转债上岸的话，今年的整体收益有比较大的希望能达到8%到10%。

去年很多人的收益率很高，20%或者30%甚至40%的并不少见，然而去年高收益的持仓风格，今年遭遇了不小的回撤，假设去年高收益组合的收益是30%而今年最终的收益是-5%，那么累计两年的收益率则是(1+30%)*(1-5%)
-1 =23.5%，而我去年的收益率是15.05%，如果今年取得8%的收益，则累计两年的收益率为(1+15.05%)*(1+8%) -1 =24.25% 。

看上去平平无奇，最终效果却更好。  由于今年大幅增加了股票的份额  ，如果有中新股，收益将进一步提高。

即使今年不能达到这个收益率也没有什么关系。

回顾过往，截止到目前我的组合最大回撤不到10%，过往年度最大浮亏大约为5%，今年的最大浮亏不知道会是多少，但是我预计很难超过10%，因为组合的估值偏低，加权PE为9.05，加权PB为1.34，加权ROE为14.86%。

> 从长期来看,一只股票的回报率与企业发展息息相关,股票的年化收益率也无限接近于企业的ROE。
>
> 查理芒格

在这个低估值的情况下，加权ROE为14.86%意味着，组合的净资产理论上将得到14.86%的增长，如果估值不变并且不分红的情况下，市值将增长14.86%，而由于存在4.93%的股息，这一部分现金分红相当于将PB为1.34的资产以1PB的价格清算为现金发到账户，因此最终的结果为
14.86%*(1-4.93%)+14.86%*(4.93%)/1.34=14.67% 。  

如果今年一直不涨或者继续阴跌，对我来说可能是一件好事，我可以按比例直接增加现有持仓到100W市值，这个时候我的组合的估值将进一步小幅降低，而净资产和分红却会增加，只要估值回到原来的水平，则会获得一部分超额收益，如果估值能进一步提升，那将获得较大的收益。

而如果今年后面的时间，持仓的股票开始回升，那么就可能需要再挑选几个低估的新成员用来增加份额了，这个完全不是问题，目前低估的标的还有很多，不怕找不到。

因此对我来说，今年市场是上升也好，下降也好，对我来说都没有什么问题，我只需要按计划推进我的投资就行了。

稳健的组合每年都是平平无奇，直到多年以后，变成一个巨大的数字。
