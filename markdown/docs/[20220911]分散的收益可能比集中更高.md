#  分散的收益可能比集中更高

从表中可以看到我的股票组合中占比大于1%的股票合计有28个，而实际上我的股票组合目前有40支股票

于是经常碰到有人说持仓太分散了，会降低收益率，集中持仓收益率更高

然而，事情真的是这样吗？

用几个例子计算一下就能发现其中的奥妙

假设我们通过各种因子筛选了一批股票，预计在未来的几年里，每年都有50%的概率翻倍同时也有50%的概率腰斩，那么对应的数学期望就是
0.5*0.5+0.5*2=1.25，也就是年化25%

如果用全部的资金买入其中的一个，那么经过10年(中间可以换入其他同样条件筛选出的涨跌概率和幅度一样的股票)

比较大的可能是经历了5次腰斩和5次翻倍，最终的市值和最开始是一样的，因为0.5*2=1，计算5次还是1

而如果运气好一点，经历了4次腰斩6次翻倍，最终相当于2次翻倍，最终市值是2*2=4，10年获得了300%的回报，折算年化为14.87%

那么如果运气差一点，经历了6次腰斩4次翻倍，最终相当于2次腰斩，最终市值是0.5*0.5=0.25，10年的回报是-75%，折算年化 -12.94%

现在换一种方式，每年分散买入10个这样的，每年再平衡(轮换)一次

那么比较大的可能是，每年有5个股票腰斩，5个股票翻倍，结果就是0.1*0.5*5+0.1*2*5=1.25，也就是年化25%

如果运气好一点，4个腰斩，6个翻倍，结果就是0.1*0.5*4+0.1*2*6=1.4，也就是年化40%

如果运气差一点，6个腰斩，4个翻倍，结果就是0.1*0.5*6+0.1*2*4=1.1，也就是年化10%

这个结果，是不是很出人意料？

在数学期望为25%的情况下，集中一个股票的收益在很大概率下低于25%，而分散10个股票的收益情况却好的多。

上面的这个例子中设定的概率和收益率比较极端，现在换一个稍微不那么极端的

假设筛选出的股票当年有30%的可能跌30%同时有70%的可能涨50%，那么对应的数学期望就是0.3*0.7+0.7*1.5=1.26，也就是年化26%

如果全资买入一个股票(每年轮换或者不轮换都一样)，10年后，最终的结果是 (0.7的3次方)*(1.5的7次方)=5.86，折算年化19.34%

如果分散买入10个每年轮换，则比较大的概率获得的结果就是0.1*0.7*3+0.1*1.5*7=1.26，也就是年化26%

而且可以发现，波动越大，长期看，分散的收益比集中要高

当然，集中持股在某一个大涨的年份，肯定是收益率更高的

但是，如果投资只入市一年，那似乎并不能算投资，更像是赌博押大小。

而如果每年都能找到当年大涨的股票并集中买入，那用不了多久就会成为投资界的传奇

如果不是天赋异禀或者欧皇附体，最好还是不要做这样的梦。
