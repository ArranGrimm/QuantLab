#  持仓组合的加权PB/PE/ROE计算方法

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

上周写了一篇《增援未来
持仓汇总表2021-No.3(基于三季报)》，然后发现之前在Excle里面按加权平均计算的PB/PE/ROE有点问题，后来改过来了，今天就来说一下这个问题。

最开始发现问题是因为温氏股份的PE是负数(因为亏损)，而在上一个汇总表里面是正数，这个问题就很明显了，某个成分股的PE为从正数变为负数，则其他成分数值不变的情况下，说明组合的每股收益降低了，那么PE应该升高，但是由于是负数，加权平均计算的话，会让PE变小，这个就不对了。

比如，组合  中
只有两个股票AB，各占50%，原本PE分别是15，20，则加权平均PE是17.5，而如果A变为亏损每股收益为负数，则通常显示为亏损，但是我的汇总表里面会计算为负数(股价/
每股收益  )，如果A的PE计算出来是-5，则组合的加权平均PE就变成了-5*0.5+20*0.5=7.5，这显然不对。

而且即使不是负数，也会有问题。

比如股票AB的股价都是10元，每股收益分别是1元，0.1元，则股票A的PE为10/1=10，股票B的PE为10/0.1=100，则加权平均为10*0.5+100*0.5=55，这显然也不对，因为对于A股票静态的看20年累计的每股收益是20元，AB股票合计20元，也就是说即使B股票每股收益为0，仅仅靠A股票，都可以做到20年累计每股收益等于20元，也就是说这个组合的PE应该接近于20。

所以正确的计算方式应该是将组合中每只股票的 每股收益X股数 累计作为分子，然后将股票的总市值作为分母。

比如组合中有ABC 3只股票，数量，股价，每股收益分别是

A 1000股 10元 1元，PE为10

B 2000股 20元 1.5元，PE为13.33

C 3000股 25元 2元，PE为12.5

则组合的PE=(10*1000+20*2000+25*3000)/(1*1000+1.5*2000+2*3000)=12.5

而如果组合中出现一个亏损的，比如这样

A 1000股 10元 -1元，PE为亏损

B 2000股 20元 1.5元，PE为13.33

C 3000股 25元 2元，PE为12.5

则组合的PE=(10*1000+20*2000+25*3000)/(-1*1000+1.5*2000+2*3000)=15.625

这样就没有问题了，组合整体的PB、ROE和股息率的计算也是类似。
