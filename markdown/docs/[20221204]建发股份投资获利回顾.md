回顾一下过程

2020年6月29号每股派息5毛(2019年利润分配方案)

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLqwCkd0a1gNl1p6vT1FmcmiartlZ8rNImAWy0vHR6qw9uOwEwFwgbOWb3LFmGO6B6SauQHFf4y1iaA/640?wx_fmt=jpeg)

2020买入的时候价格在8块出头，对应的股息率大约为6%，随后股价慢慢跌到7块多

2020年的年报显示利润相比2019年还降低了5.39%，股价就在8块上下做俯卧撑

本来想着就这样也行，因为这个利润水平每股5毛的股息大概率还是有保障了，所以继续持有。

时间来到2021年中报发布(2021年8月27号)，显示利润同比增长12.47%，股价也开始从8块一路涨到10多块

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLqwCkd0a1gNl1p6vT1Fmcm4ru1QnTVfmLSF4Yz3PWUuAmbXicn1ThjibibMYB0QBTxBc6Z09cNRen3Q/640?wx_fmt=jpeg)

2021年年报发布(2022年4月18号)，显示利润同比增长35.4%，股价继续涨，最高涨到了接近15块，并且年报利润分配方案为每股分红6毛

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLqwCkd0a1gNl1p6vT1FmcmfQyWibJqoQYB8s4NwCWFUEII00ibZtxVCmg27icskNPNrk8nmQfKibUBHA/640?wx_fmt=jpeg)

考虑到性价比已经降低不少，于是在这一段时间14块上下出一小部分，

今年7月全市场开始普跌，它的股价也随之跌了下来

在三季报发布前涨到了15块以上，于是在这一段时间14块上下又出了一小部分。

剩下的也不多了，一方面想看看三季报的情况，一方面也在等更加好的价格。

最终三季报的增长一般，11月29号的地产板块大涨它也随着涨停

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLqwCkd0a1gNl1p6vT1FmcmohSGGzibzVmpabXRld24vugIsHQZR5YBNRcGRjbaGMPS42Ch2998Ufw/640?wx_fmt=jpeg)

板块普涨之下，考虑到它的性价比已经很低了，未来高增长的可能性也不大，于是选择了获利了结，留了2手观察仓。

虽然建发在福建属于知名房地产商，但是其实建发股份的主营业务是物流供应链

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfLqwCkd0a1gNl1p6vT1Fmcm9UDiahicSufiarZvfO9DGRaPVrJUbnXdNuylh4oBicw7Peibe1F6LLWxGOg/640?wx_fmt=png)

总体来说，这一笔投资算是成功的，几个要点(当然，也可能纯粹就是巧合)

1 买的便宜，在最近3年的低点附近买入

2 股息保底，买入时股息回报大约6%，即使企业利润不怎么增长，估值不提升，股息回报也能接受

3 碰到了利润增长和估值提升的戴维斯双击，买入时是PB 0.6x，卖出时是PB 大约为1，同时利润也是增长的

第3点有运气的成分，买入时我肯定是不知道利润会提升的

不过我一贯的做法是在买入时假设利润不增长或者只有小幅的增长，市场给的估值也不提升，在这个情况下如果还认为值得买入，那就买入。

买入后要么股价不涨，那就拿着收股息

要么股价涨了性价比降低后分批卖出后按同样的逻辑换入新的股票

接下来的时间建发股份还会不会继续新高，无法预测，但是目前的价格性价比不具有吸引力，如果后面继续涨，那就该有格局的网友赚钱了。

而如果后面跌到值得买入的位置，我也会再次买入。

这个过程中我们可以看到

利润的增长(对应每股收益和每股股息的增加)，对股价增长的推动作用还是有的

不能说一定会推动，也不能说立刻能推动，但是在市场上还是普遍存在的

我之前的持仓中还有几个类似的股票也经历过利润增长推动估值提升带来的戴维斯双击。

目前我的持仓中还有几个类似的票，希望在接下来的时间里能有机会享受戴维斯双击。
