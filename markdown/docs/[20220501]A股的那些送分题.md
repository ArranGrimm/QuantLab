#  A股的那些送分题

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

前一段时间市场下跌，市场上的谩骂之声多了起来，各种吐槽，各种段子也是层出不穷。

反正就是A股垃圾，上市公司就知道圈钱，机构就知道割韭菜，等等。

说来说去就一个意思：A股没让我赚到钱，所以A股垃圾。

放着送分题不做，非要去做拔高题，做送命题，能不亏钱吗？

拔高题，门槛高，难度大，成功率低，然而赔率高导致成功案例非常耀眼，对散户的吸引力非常大，但问题是别人在合适的时候交卷了，而绝大多数散户不仅答案写错，还在想交卷的时候发现卷纸已经没人要了，交了钱考了一个寂寞。

而送命题不仅难度高，而且还有人作弊甚至考试就是那些人组织的，作为散户，在送命题上没有优势，进去基本上送人头，当然也不是说一定不能赚钱，谁还没有几天运气好的时候？但是最终大概率会还回去，并且还要倒贴，毕竟组织考试的人不是活雷锋。

而以我对自己现阶段投资水平的定位，我觉得现阶段还是做送分题更靠谱，哪天要是水平见长了，也可能会去尝试做一些拔高题，至于送命题，就没有什么必要了。

那么问题来了，哪些是送分题？

以去年为例，最大的送分题就是可转债，尤其是双低或者低价风格的转债，这个我之前写过《 [ 双低可转债摊大饼策略简要分析
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483949&idx=1&sn=919077321db56f33fa01ac35b6581927&chksm=cf17b3a7f8603ab1ef1c5d29cbd3acc5e7503ba5a7f99531da17d1d739d61cfcaad85ce2f3f0&scene=21#wechat_redirect)
》

不过可转债今年不再是送分题了，已经开始向拔高题靠近。

而今年的送分题，大概就是破净的高息股了。

一方面价格下跌让股息上升，另一方面还有潜在的估值修复的可能，这些公司从来都不是明星，大部分时候都是平平无奇的路人角色，比起那些动辄几个月50%甚至翻倍的拔高题或者一年几倍十几倍的送命题，这些送分题的分值都不高，一年也就能给个10%的样子，但是他们都是实实在在的送分题，只要不瞎搞就不会亏了学费。

比如，大秦铁路(利益相关，我少量持有，不构成投资建议)

公司章程是这样写的

> 4、利润分配的条件和比例
>
> 在公司实现盈利且无重大投资计划或现金支出计划等事项年度，公司每年以 现金形式分配的利润不少于当年实现的可供分配利润的 30%。

最近4年利润虽然略有波动，但是分红都是每股0.48元，而现在的价格不足7块，股息回报在7%左右，而未来几年继续分红0.48元的概率是极大的，至于原因嘛，对国企有那么一点了解就知道了，这个我之前也写过《
[ 国有企业的投资价值
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484232&idx=1&sn=46251e7d8ef5a89b7c4b092f85de6a93&chksm=cf17b2c2f8603bd476cf3be70a0ca8ecb069da72ef9718993be9a444e402fd421ae9a9c93ad8&scene=21#wechat_redirect)
》。

当然有人会说，煤炭是夕阳产业了，不行啦，未来还是新能源的。

这些人，去年拿着券商预估的宁德时代2060年的业绩，走进了送命题的考场，现在就呵呵了。

煤炭肯定是夕阳产业，新能源也极大可能是未来的主流，但是如果持股周期很少超过一年的散户，说着几十年的未来，是不是显得很滑稽？

类似大秦铁路这样的送分题在目前的市场上还是不少的，每年拿着6%到7%的股息，再加上单户10W市值主板打新(抛开科创板和创业板)每年4%到5%的新股收益，每年能有大约10%到12%的回报，另外净资产增加最终会推动股价上涨，而且未来还存在估值修复的空间，这些都能进一步提升收益率。

对于普通的散户，这个已经足够了，如果想要提高到15%以上，那就得做一些拔高题了。

要是既不会做拔高题，又找不到送分题，但是就想炒股，怎么办？
