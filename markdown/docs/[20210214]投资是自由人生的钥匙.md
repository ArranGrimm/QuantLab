#  投资是自由人生的钥匙

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

春节是一年中难得的长假，尤其是今年很多人都就地过年，省去了不少奔波也省去了不少应酬，对比往年应该会更加的轻松愉快。

假期总是让人开心的，远离了工作，可以得到充分的休息和娱乐，也可以把时间分配给自己的兴趣爱好，最重要的是，收入不会受到影响。如果条件允许，很多人大概会希望这样的日子能一直持续下去。

这样的日子，我们都曾经拥有过。

让我们把思绪拉回到小学的时光，仔细回忆一下小学的我们是不是就是过着这样的日子。每天去学校上几个小时的课，回到家做完作业就可以玩了，不用操心家庭的琐事，不用操心工作的难题和压力，也不用担心收入，这些都是我们的父母替我们在承担。父母维持着我们的生活，而我们则过得轻松愉快。

后来事情就发生了变化。

毕业以后开始工作，需要自己养活自己。不同的人能力有强有弱收入有高有低，相同的是工作才有收入不工作就没有收入。经过一些年的上升期后，如果没有特别的机遇，工作上的提升开始变的缓慢，收入的增长开始停滞不前，而开销却依然在增长。对未来，也由刚开始几年的满怀希望，变得开始怀疑。

如果想要回到像小时候那样开心的日子，除了心态，更重要的是有稳定的供养，这个供养只能是我们自己积累的资产。这个资产可以是用于收租的房产、能带来利润的店铺或者公司、也可以是能带来分红的股票和基金。

而这些之中，股票和基金大概是门槛最低的，只不过要达到能供养自己的程度，需要一个较长的积累过程。这个过程既包括本金的积累，也包括知识的积累，还包括经验的积累。

直到有一天，投资带来的现金流代替了工作的收入，我们就拿到了自由人生的钥匙。

以后的日子，我们就可以选择我们想做的事情，拒绝我们不想做的事情。
