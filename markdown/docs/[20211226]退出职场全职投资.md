#  退出职场全职投资

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

没想到我这么快就成为了一个职业投资者。若是放在几年前，恐怕连我自己都难以相信。

好几年前开始考虑投资股票，密集的阅读了一些经典的投资书籍，慢慢将钱从理财转到股票，慢慢的增加仓位。

虽然《富爸爸》和《财务自由之路-7年内赚到你的第一个1000万》都为长期投资描绘了一个玫瑰般美好的未来，但是我总是觉得有些缥缈，鸡汤味太浓。

当然，这两本书也确确实实给我的思想带来了一些变化，也算是有不小的价值。

但是显然，想要像富爸爸那样富对我来说过于遥远，而7年赚到1000W的难度实在是太大，所以即便是我开始了投资，也并没有做这些不切实际的美梦。

时间过了2-3年，投资略微赚了一点点钱，真的是一点点，因为投入的资金很少，但是在这个过程中，我能明显感觉到我的思路是正确的，自己的投资体系也在慢慢的形成。

基本上我开始相信，未来我会通过投资赚到比较多的钱，可以不用再去上班，但是这个未来很遥远。

为了回顾总结并改进自己的投资系统，我开始每周写一篇关于投资的想法和思考，也就在差不多同样的时候，我看到了一篇文章《 **那些用利息生活的年轻人** 》

原来还有人用每年几千块的利息生活着，虽然这样的生活我是不太能接受的，但是如果一年的利息是几万或者十几万，不就可以用利息生活了？而我的投资组合以高息股为主，每年能提供5%以上的股息。

又经过了一段时间，我根据家庭的收支情况进行了一些估算，并且写了一篇《假如有300万人民币的现金，可以全职投资吗？》，用利息生活的目标，似乎离我不再那么遥远。

这篇文章回复的人很多，有赞同的，也有反对的，总体来说反对的居多，基本上都觉得不够。

考虑到不同的城市不同的家庭开销情况不一样，于是我又写了一篇 《通过储蓄率来计算一下财务自由到底需要多少钱》

当然严格的说，按我估算的结果，叫财务独立可能更合适一点，也就是被动收入覆盖生活开销。

虽然300W依然是一个需要比较长的时间才能完成的目标，但是至少不再那么遥远。

可能是念念不忘必有回响，今年在和一个朋友的交流中，我有了新的发现。

**那就是我严重低估了打新的收益**

在《通过储蓄率来计算一下财务自由到底需要多少钱》中，我对打新收益估算的是2.5%，但是实际上如果家里人的账户都开了科创板，市值均分到各个账户，全家一起打新股的话，打新收益率大约在15%，这个时候局面就完全不一样了。

在之前我估算想要获得每年20W出头的现金流需要300W的市值，而按照这个模式，则只需要100W出头，也就是我前面两篇写的《低波动稳健20%现金流回报的策略(上)
》和《低波动稳健20%现金流回报的策略(下) 》

而在今年年初，我手里的可投资资金已经超过了100W，同时为了可转债打新，去年我已经动员家里人都开通了证券账户，明年就都可以开通科创板了，同时我的投资系统和心态也日趋成熟。

这一下子又把计划从好几年后才能启动，直接快进到明年就可以启动。

然而事情的发展再一次超出了我的预计

由于年初我在可转债砸出黄金坑的时候配置了一部分可转债，这部分可转债到今年下半年的时候收益已经超过了40%，其中一半的收益已经落袋，加上还中了几个新股，直接相当于赚了好几个月的工资。

考虑到后续这些账户的股票配置仍然需要在现有基础上做一些调整，需要花费一些时间筛选，而另外还有一些杂七杂八的事情，加上今年的投资已经赚到了好几个月的工资

最终，我决定提前启动。

目前的策略在未来的2-3年内大概率是没有什么问题的，
新股收益很可能慢慢降低，不过我每年还有一些结余可以继续增厚本金，因此股息回报会慢慢增加，此消彼长，总体看被动现金流大概率能有保障
，同时我也要慢慢的优化选股并且在组合内轮动同时在组合间轮动以及识别并把握系统性机会，这样还能小幅的提高收益。

后续我会继续更新，继续记录自己关于投资的想法和思考，以及记录我自己投资的进展情况。

希望能给想要全职投资的朋友们提供一点参考。
