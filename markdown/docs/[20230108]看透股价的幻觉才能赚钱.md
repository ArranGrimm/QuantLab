#  看透股价的幻觉才能赚钱

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

股价是一种幻觉，但是能看透的人却很少。

这个论调看上去很奇怪

比如，我手里有1000股某股票，昨天买入的价格是10元，今天收盘涨了2毛

如果我卖出的话就赚了200块，这股价怎么能是幻觉呢？

那么再来看另外一种情况

早上收盘的时候涨了2毛，账面浮盈200元，中午去吃了一个饭午休了一下

下午收盘的时候价格又跌了2毛回到了10块，账面持平

那么请问，中午赚的200元，哪里去了？

而我们经常会看到财经新闻写到：某某股票几天蒸发了xx亿

总有人会疑惑，这些钱去了哪里

实际上这些钱可以认为根本就没有存在过，只不过是一种幻觉。

因为现在股票的价格以及公司的市值，总是在交易过程中不断地被刷新

最新的价格高1分钱，公司的市值就增加一些，最新的价格低1分钱，公司的市值就减少一些

在这个短暂过程中，除了股价和市值在不断地变化，其他什么都没有变。

但是很多人在这个过程中，进行了交易，只要买卖的价格不一样，账面就会产生实际盈利或者亏损

这也是很多人认为股价是真实的财富的原因

再来看一个关于分红的

A股分红是需要除权的，因此很多人会认为是自己的钱分给自己

但是实际上分红用的是公司的利润，是真金白银，价格进行除权才是幻觉

比如一个公司去年的股价是每股10元，一年下来赚了10亿，公司决定将这10亿利润全部用于分红，每股分红5毛

由于市场低迷等等原因，一年下来公司的股价没涨还是10元，分红除权后第二天开盘9.5元。

这个公司因为将这一年的利润全部分给了股东，相当于分红后企业回到了去年年初的状态

而去年年初股价是10元，现在股价是9.5元，公司有什么变化吗？

没有，只不过股价跌了5毛而已，接下来的时间股价可能继续跌，也可能涨

看上去分红降低了市值，而实际上分红用的是企业的利润，市值是无法用来分红

市值只不过是用最后一次交易的价格给整个企业计价而已

而如果一个散户一直盯着这个股价的话，估计能把自己累死

对于上面的这个企业，显然9.5元的投资价值比10元更高。

因此我们需要盯着一些别的东西，而不是股价

比如有人盯着ROE，也有人盯着PEG

而我盯着的是每股净资产、每股收益以及每股股息

比如在我的持仓汇总表里面，我会每个季度根据企业的财报，将每股净资产和每股收益更新为最新值，每年公布分红方案的时候会将每股股息更新为最新的值

对我来说这些才是真正的价值所在

比如某个企业一年下来每股净资产、每股收益、每股股息都增加了10%，则我认为这个股票的价值增加了10%

至于这个股票的价格是增加了10%还是下跌了10%，我并不在意

如果股价跌的比较多，我甚至会再买入一些，看上去我的账面市值减少了，但是我的持仓的实际价值增加了

因为我并不急于卖出股票换取现金，因此这个下跌对我来说就是一个假象，而每股股息的增加却可以让我拿到比去年更多的真金白银，这些才是真实的财富。

而如果哪一天股价涨的太多，我会卖出股票，将股价的幻觉变成真实的差价利润收入囊中。

识破幻觉，利用幻觉，获取真正的价值才能长期赚到真金白银。
