#  散户如何才能不为国际局势买单

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

每当国际上发生什么大事，如果看上去对中国不利，在接下来的交易日就会有比较大的可能出现普跌

而散户通常戏称为：大A为此事买单

虽然是戏言，虽然账面的浮亏只是浮亏，但是也有人会选择减仓或者清仓，等待局势明朗

而减仓或者清仓，会将浮亏变成实亏，那就是实实在在的买单了

而等到局势真的明朗了，接回来的时候又通常会比当时减仓的价格略高

平白无故的让对手盘赚了自己的差价，属实不值得。

端午期间，毛子国出现了兵变，估计很多关注国际局势的散户看到新闻，马上就想到假期结束后A股又要买单，手里的粽子瞬间就不香了。

结果才过了不到24小时，事情结束了，庆幸得亏25号虽然上班，但是周天不开盘，手里的粽子顿时又香了

不过情况并没有那么乐观，26号周一开盘不久后，市场还是普跌，收盘时全市场4041家下跌。

如果每次国际上出什么事情，作为一个散户，都提心吊胆，先不说赚没赚到钱，就这影响睡眠、影响健康，貌似并不值得。

因此也有不少人选择节前空仓，碰到什么节日，放假之前空仓，给自己放个假。

不能说完全没有道理，但是这样投资(炒股)属实太闹心。

实际上，这些事情，也只有短期的影响，长期看影响基本上可以忽略。

还记得疫情开始时第一个春节结束后开盘第一天，三千股跌停，现在回头看，好像也没那么严重

再往前，贸易战刚开始，市场也会买单(大跌)，现在回头看，好像也没那么严重

再往前，还有几个人记得非典，当然现在场内的散户可能很多人在非典时还没入市。

入市时间短，碰到这样或者那样的事情，就会觉得是什么不得了的大事，但是把时间拉长来看，影响并没有想象的那么大，甚至可能基本没啥影响。

我之前看过一篇文章《 [ 你在期待一个怎样的环境
](http://mp.weixin.qq.com/s?__biz=MzI4NjU4ODk1Mw==&mid=2247487606&idx=1&sn=13dab6e675e4a12be55322f7aa9dd56c&chksm=ebdbf900dcac7016a6c8a7ca89a65846e23625665718e472a10ab65d984cb028f514a78bea95&scene=21#wechat_redirect)
》，  推荐大家阅读一下，  里面有个图我贴出来给大家看一下

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLgekJeREZg1KY5c18ZvgeQQdwIXhwC9ON7iaOGUHC0lQKXSS6dKb6pGlTHiblEicYlAYtKxqnLVuJpA/640?wx_fmt=jpeg)
里面有一句话是这样写的：” **一个理性的投资者，不应该期盼一个完全确定的、完全可预期的外部环境。人类世界不存在那种环境** “

如果只想进来捞一把就走，那没啥好说的

但是如果打算长期投资，比如像我这样，打算终身投资，那么在整个投资生涯中，一定会碰到这样或者那样的大事。

从我投资股票以来，碰到过贸易战、疫情、美国对我国的制裁和封锁，以及最近一年多的俄乌冲突，每次都给我的账户带来了一定程度的回撤，但是最后都恢复到原来的水平甚至超过，最终的结果就是我的持仓每年都取得了正收益。

所以这些事情，对市场和对账号有影响吗？

肯定是有的

但是这些事情，影响最终的投资结果吗？

至少没有影响我的投资结果。

再过3个月，国庆假期又到了，说不定到时候又有什么幺蛾子。

早点做好心理建设，应对计划

省的到时候正在外面自驾游呢，一看新闻，A股又要准备买单了，顿时玩的心情也没有了

这不值得
