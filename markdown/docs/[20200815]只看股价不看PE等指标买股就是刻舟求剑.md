#  只看股价不看PE等指标买股就是刻舟求剑

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

刚开始投资股票的时候，大家最先看到的就是股价，股价涨了股价跌了，持仓浮盈了持仓浮亏了，止损了止盈了。由于关注的是股价，心情也是随着股价起伏不定，时而高兴时而沮丧，时而庆幸时而懊恼。这里是不是有什么不对劲？实际上只盯着股价是有问题的，只看股价不看PE等指标，投资股票无异于刻舟求剑。

##  看PE

单纯的看股价并不能正确的反应企业是否值得买入，PE则是常见的评估指标，适用于大部分企业。

假设一个企业当前的股价是8块，总市值是800亿，去年净利润是100亿，则PE等于800亿/100亿，8倍PE。意味着假设我当下将整个企业买下来，则按目前的利润情况，8年可以收回成本，第九年开始每年有12.5%的回报。当然我显然是无法买下整个企业的，我只能买入一部分股票，但是这个并不影响每年12.5%回报的结论。

那么这里问题来了，这12.5%的回报如何体现？

通常有两种途径：

  * 分红 

将利润分给股东，也就是派息。将所有利润全部分给股东，这样的企业虽然很少但是是确实存在的，例如双汇前几年就是这个样的，还有其他几家就不一一列举了。按上面的数据假设的话，则股东每年能获得12.5%的股息现金回报。当然有人可能会说除权了股价降低了，分红有啥意义。这个可以参考《
[ 如何正确的认识股息
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483661&idx=1&sn=4aea859a3fbdbcbad45a9d5ad10bb4df&chksm=cf17b087f86039918ab00a9f2d1c1967c21f6a58992377feb50f48eade8011beea4a28240283&scene=21#wechat_redirect)
》。

  * 增厚净资产 

有的企业将利润全部留存在企业里面，扩大规模，增加下一年的利润使利润总额增加，这样企业的净资产越来越多，同时市值也是随着增加，反应到股票上就是股价涨了。最具代表性的企业就是巴菲特的伯克希尔。

  * 两者混合 

也有部分企业采用的是两者混合的方式，即将一部分利润拿出来回报股东，剩下的部分留存在企业。

那么上面假设的这家企业如果采用两者混合的方式，若干年后上一年的  净
利润是150亿，股价是10块，对应的总市值是1000亿，股价上涨了25%，那么问题来了，这个时候应该买进还是卖出？
如果看股价有25%的浮盈，卖出能获利好像没什么毛病，但是计算一下却能发现新的东西。

当年买入股票的时候，这个企业8倍PE，静态的看相当于股票的回报率是12.5%，现在这个企业是6.66倍PE，对应股票的回报率是15%。显然这家企业当下比之前更具备投资价值，回报更高。因此现在10元买入，比之前8元买入的时候更划算，而此刻如果只是看股票觉得比之前高而觉得不应该买入，那无疑就是刻舟求剑。

##  看股息

股息是另外一个角度，对应的企业是国有银行类企业和国资委控股的公共事业企业例如水电交通等。

这一类企业的特点是实际控制人和大股东是国资委财政部等等。没有游资炒作，股价没有大涨大跌，业绩也缺乏想象力，增长很小，派息稳定，长期看每年略有上涨。同时企业利润每年也是略微增长。

股息率取决于市场情绪对股价的影响。例如农行因为股价的波动，最近几个月股息率就在4.x%~5.x%之间波动，注意这里股息没有变，变化的是股价，股息率=股息/股价，因此股价的波动带来了股息率的波动，如果抛开股价而看股份的话，不论股价如何波动，每股分红是基本稳定的。

那么就有一个值得思考的地方，假设在上一次派息后，用3.4的价格买入农行，那么经过大概一年在本次派息前，多少价格买入和之前是等价的？

粗略的计算一下：

  * 2019年6月派息后3.4买入的农行，持有一年后2020年7月派息得到0.18的股息。经过大约一年的持有手里的股份没变，3.4元一年的收益是0.18元。 

  * 2020年7月之前3.58买入，7月派息后得到了每股0.18的现金，持有时间几天忽略，持有成本3.4。持有一年后2021年再次派，假设还是0.18，则3.4元一年的收益是0.18元。 

由此可见，这两种情况基本上是等价的。当然最近(2020年8月)农行的票价跌了不少，单从股息的角度看，此刻农行的股票比之前更具投资价值了，相当于获得同样股息的成本更低了。当然这个需要充分的理解股息的本质。

##  看前景

前景是市场上最热的话题了，什么十倍股、什么明星企业、什么优质赛道、什么市场空间，等等。每年都有新的名词被发明出来，而且似乎都有道理。

看PE和看股息计算回报的确定性相对比较高，而前景回报的确定性比较低，但是空间却可能很大。因此这部分企业也会受到市场的追捧。

从前景的角度判定一个企业当前是否值得投资的难度会更高，这里用科技企业常见的风险投资来类别一下。

假设有一个刚起步的科技企业(可以想象为谷歌腾讯之类的)，由于是刚起步，业务具备创新性，前景不确定，成长空间也不确定。

在刚起步阶段，天使轮投资开始了，市场给予估值1000W。假设成功率是1%，潜在回报是200倍。这个时候风投公司投资100W，占有10%的股份，那么静态的看这笔投资回报的数学期望是
1%*200=200%。相当于风投公司平均投资100家这样的公司，耗资一亿，最后成功一家，能得到两亿。

过了几年，风投公司投资的公司死了一批，还有一部分继续发展，有几家发展的还不错，开始进行A轮投资。这个时候某个公司前景比起步阶段明朗了不少，成功率被市场预估为10%，潜在回报20倍，市场给予估值一亿。这个时候另外一家风投公司投资1000w，占有10%的股份，那么静态的看这笔投资回报的数学期望是
10%*20=200%。相当于风投公司平均投资10家这样的公司，耗资一亿，最后成功一家，能得到两亿。

经过不断的发展，不断有公司淘汰，最后有几家获得了成功，投资人获得了回报。从投资成功某一家企业看，投资人获得了高倍的回报，然而这个是建立在大量打水漂的投资的基础上的，因此从整体上看，投资回报率并没有那么高。

在这个例子中，风投公司在天使轮投资的100W，和在A轮投资的1000W，都是占有了10%的股份，价格相差10倍，但是从理论上计算这两份10%的股份的投资价值是一样的。

在股票市场上投资那些被认为有良好前景而当前却没有怎么赚钱的企业，在行为和结果上，与风投公司的投资是类似的。

##  最后

这里只是粗略的写了一点最基础的内容，没有涉及到PB和ROE，上面举的常规非科技类企业的例子都是假设PB不变的情况。ROE和PB是有关联的，考虑到篇幅问题，暂且略过。

文中也没有涉及到周期性企业，这类企业的利润是周期性变化的，往往利润最低的时候股价相对低而PE却相对高，这个时候却是较好的投资时机，不过把握这个时机的难度很大。

由于企业处于不断的变化之中，可能在发展中壮大，也可能在发展中衰落，因此单看股票的价格来决定是否值得投资，那就和刻舟求剑没有什么区别。

最后祝大家投资顺利！
