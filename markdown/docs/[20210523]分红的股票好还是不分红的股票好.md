市面上的股票，有一些是常年分红的，有一些是常年不分红的，分红相当于把净资产按PB=1清算一部分用现金发给股东，那么是分红的股票好还是不分红的股票好？

假设某股票，ROE=10%，每股净资产10元，估值不变，计算过程做了适当的四舍五入。

##  假设PB=1

初始股价为10元(净资产*PB)，不分红。

  * 初始买入1W股，账户市值10W。 

  * 1年后，每股利润1元，每股净资产11元，股价11元，账户市值股票11W。 

  * 2年后，每股利润1.1元，每股净资产12.1元，股价12.1元，账户市值股票12.1W。 

  * 3年后，每股利润1.21元，每股净资产13.31元，股价13.31元，账户市值股票13.31W。 

初始股价为10元(净资产*PB)，利润的50%用来分红。

  * 初始买入1W股，市值10W。 

  * 1年后，每股利润1元，分红前每股净资产11元，股价11元，分红后每股净资产10.5元，股价10.5元，得到现金0.5元，账户市值股票10.5W+现金0.5W=11W。现金买入股票476.2股，合计股数10476.2，账户市值11W。 

  * 2年后，每股利润1.05元，分红前每股净资产11.55元，股价11.55元，分红后每股净资产11.025元，股价11.025元，得到现金0.525元，账户市值股票11.55W+现金0.55W=12.1W。现金买入股票498.9股，合计股数10975.1，账户市值12.1W。 

  * 3年后，每股利润1.1025元，分红前每股净资产12.1275元，股价12.1275元，分红后每股净资产11.5763元，股价11.5763元，得到现金0.55125元，账户市值股票12.705W+现金0.605W=13.31W。 

3年下来

不分红，账户市值分别是

11W，12.1W，13.31W

有分红，账户市值分别是

11W，12.1W，13.31W

不分红与分红后复投，结果一样。

##  假设PB=1.5

初始股价为15元(净资产*PB)，不分红。

  * 初始买入1W股，账户市值15W。 

  * 1年后，每股利润1元，每股净资产11元，股价16.5元，账户市值股票16.5W。 

  * 2年后，每股利润1.1元，每股净资产12.1元，股价12.1元，账户市值股票18.15W。 

  * 3年后，每股利润1.21元，每股净资产13.31元，股价13.31元，账户市值股票19.965W。 

初始股价为15元(净资产*PB)，利润的50%用来分红。

  * 初始买入1W股，市值15W。 

  * 1年后，每股利润1元，分红前每股净资产11元，股价16.5元，分红后每股净资产10.5元，股价16元(除权减0.5元)，得到现金0.5元，账户市值股票16W+现金0.5W=16.5W。现金买入股票312.5股，合计股数10312.5，账户市值16.5W。 

  * 2年后，每股利润1.05元，分红前每股净资产11.55元，股价17.325元，分红后每股净资产11.025元，股价16.8元(除权减0.525元)，得到现金0.525元，账户市值股票17.325W+现金0.5414W=17.87W。现金买入股票322.26股，合计股数10634.76，账户市值17.87W。 

  * 3年后，每股利润1.1025元，分红前每股净资产12.1275元，股价18.1913元，分红后每股净资产11.5763元，股价17.64元(除权减0.55125元)，得到现金0.55125元，账户市值股票18.76W+现金0.58W=19.34W。 

3年下来

不分红，账户市值分别是

16.5W，18.15W，19.965W

有分红，账户市值分别是

16.5W，17.87W，19.34W

不分红优于分红后复投。

##  假设PB=0.5

初始股价为5元(净资产*PB)，不分红。

  * 初始买入1W股，账户市值5W。 

  * 1年后，每股利润1元，每股净资产11元，股价5.5元，账户市值股票5.5W。 

  * 2年后，每股利润1.1元，每股净资产12.1元，股价12.1元，账户市值股票6.05W。 

  * 3年后，每股利润1.21元，每股净资产13.31元，股价13.31元，账户市值股票6.655W。 

初始股价为5元(净资产*PB)，利润的50%用来分红。

  * 初始买入1W股，市值5W。 

  * 1年后，每股利润1元，分红前每股净资产11元，股价5.5元，分红后每股净资产10.5元，股价5元(除权减0.5元)，得到现金0.5元，账户市值股票5W+现金0.5W=5.5W。现金买入股票1000股，合计股数11000，账户市值5.5W。 

  * 2年后，每股利润1.05元，分红前每股净资产11.55元，股价5.775元，分红后每股净资产11.025元，股价5.25元(除权减0.525元)，得到现金0.525元，账户市值股票5.775W+现金0.5775W=6.35W。现金买入股票1100股，合计股数12100，账户市值6.35W。 

  * 3年后，每股利润1.1025元，分红前每股净资产12.1275元，股价6.06375元，分红后每股净资产11.5763元，股价5.5125元(除权减0.55125元)，得到现金0.55125元，账户市值股票6.67W+现金0.667W=7.337W。 

3年下来

不分红，账户市值分别是

5.5W，6.05W，6.655W

有分红，账户市值分别是

5.5W，6.35W，7.337W

分红后复投优于不分红。

##  为什么

实际上这个不难理解

当PB=1的时候，如果ROE/PB/PE都不变的情况下

1元钱的利润留存在企业里就能增厚1元的净资产，

不分红的情况下企业每增加1元的利润，投资者持有的股票的净资产就增加1元，市值提升1元。

分红50%的情况下，企业每增加1元的利润，投资者持有的股票的净资产就增加0.5元，然后用另外的0.5元买入0.5元额度的股票(净资产0.5元)，
投资者的账户总体市值提升还是1元  。

两者一样

  

当PB=1.5的时候，如果ROE/PB/PE都不变的情况下

1元钱的利润留存在企业里就能增厚1元的净资产，

不分红的情况下企业每增加1元的利润，投资者持有的股票的净资产就增加1元，市值提升1.5元。

分红50%的情况下，企业每增加1元的利润，投资者持有的股票的净资产就增加0.5元，市值提升0.75元，然后用另外的0.5元买入0.5元额度的股票(净资产0.333元)，市值总体增长是0.75+0.5=1.25元。

拿到的0.5元分红，只能买入0.5元额度的股票，这部分没有享受到1.5倍PB带来的市值增长。

  

当PB=0.5的时候，如果ROE/PB/PE都不变的情况下

1元钱的利润留存在企业里就能增厚1元的净资产，

不分红的情况下企业每增加1元的利润，投资者持有的股票的净资产就增加1元，市值提升0.5元。

分红50%的情况下，企业每增加1元的利润，投资者持有的股票的净资产就增加0.5元，市值提升0.25元，然后用另外的0.5元买入0.5元额度的股票(净资产1元)，市值总体增长是0.25+0.5=0.75元。

拿到的0.5元分红，买入了0.5元额度的股票，如果这部分不分红则只能带来市值0.25元的增长。

当然可能有人会觉得分红每年可以拿到现金，而不分红只能看账户上的数字。

其实分红复投后账户也没有现金剩下来，而不分红的话，也可以自己卖掉一定比例的股票收回现金。
