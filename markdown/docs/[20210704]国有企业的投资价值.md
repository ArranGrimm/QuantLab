#  国有企业的投资价值

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

很多上市的国有企业都具有可观的投资价值，当然，想要暴富那就不要指望了。

国有企业给人的印象通常是臃肿、低效、落后，但是这并不妨碍它们具备可观的投资价值，在讨论这些企业的投资价值之前，让我们切换一下视角，先来看看国有企业和体制内的岗位。

因为毕业有一些年头了，对于现在毕业生的就业意愿不是太了解，从公开的媒体报道中大概可以看出国有企业和体制内的工作还是有一定吸引力的，这个和我毕业那会的情况大相径庭。

我毕业的那几年，周围的同学只有个别人考公务员，进国有企业的也很少，一来好的央企国企不容易进，二来一般般的国企大家又看不上。

周围听到很多故事，比如某某师兄进了某某民企，工作几年后升职加薪，当了经理总监啥的，妥妥的上了一个台阶。

这样的故事吸引着我们这些初出茅庐的愣头青，总觉得这样的职业生涯才是值得投入的，而国企和体制内那种一眼就能望到头的生活简直是对生命的浪费。

嗯，是不是和很多人买股票研究企业后选择了眼中的明日之星希望获得丰厚的回报非常像。

时光飞逝，当年的愣头青，如今都年过30，有家有娃，其中有一些确实在职业上取得了巨大的成就晋升为经理总监甚至副总，成为同学中的传奇，也有一些在体制内或者过的很好或者过得平庸。

更大的一批人则是在民营企业担心着自己的未来，再过几年岗位还能不能保住，会不会被裁员等等。这个时候开始后悔，为什么当初没有想办法进入体制内。

为什么会这样？当初豪情万丈的愣头青，如今却开始羡慕起体制内的平庸生活。这就是社会的毒打。

让我们把视角再切回到股市。

股市中最吸引人的，都是那种一年几倍或者几年十几倍的股票，这样的股票不论是因为业绩爆炸确实值这个价，还是由于炒作导致的，并不重要，重要的是有一小部分人在这个过程中获得了丰厚的回报，但是注意，只有一小部分人，是不是和某个曾经普通的同学进去一个企业后取得巨大的提升很像？

那么在股市上去追求这样的回报行不行？

当然可以，但是不要忘记，这样的方法，虽然赔率很高，但是胜率很低，低到和中彩票相差无几。很多人可能不服气，说谁谁谁或者自己曾经押中某股票一年涨了多少多少。是的，这个是存在的，但是可以回想一下，这样的事情，发生过几次，周期是多少年？

我想，通常是只抓到一次，而且还是投资股票那么多年来唯一的一次，能抓住两次的微乎其微。这样带来的结果就是，这一次赚了一些钱，也只是这一次赚了一些钱，可能后面又亏回去了。当然也有一些民营企业，虽然没有暴涨，但是也在比较长的时间内让投资者获得了良好的回报。

那么投资国有企业，我们能得到什么？

通常能得到稳定的分红，较小的波动(不会跌太多也不会涨太多)，是不是和体制内的工作一样，乏味、没有惊喜，但是有保障。

对于大部分投资者来说，这样的回报是没有吸引力的，  就像刚毕业的愣头青希望几年就能走上人生巅峰  。

但是这样的乏味的企业，把时间拉长来看，很多都能取得平均年化10%左右的回报，加上股价波动小适合作为打新门票，还能带来打新的额外收益。

更进一步，如果能在熊市多买入，牛市适当卖出，还能再获得一部分超额收益，当然这样的机会很少，可能5年8年才能有一次。当然如果反过来做，那就要吃大亏了，比如高估值买入中石油。

如果觉得自己能超越大多数人，当然可以选择潜在回报更高的企业，只不过需要承受看错的代价。如果能接受小概率赚大钱，大概率输了时间又没有赚到钱的结局，这样的选择并没有什么问题。

而对于我来说，我只希望能度过平庸而富足的一生，平均年化10%再加上打新收益和牛熊交易的超额收益，可能在2、3年内看起来并没有什么卵用，但是别忘了复利要靠时间来发挥威力。

年轻的时候发不了财，但是可以让自己从中年开始慢慢过上舒适而富足的生活，就像体制内的工作一样。

这是不是具有巨大的价值?
