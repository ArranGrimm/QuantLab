#  用网格交易在组合内轮动的策略

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

网格交易简单的说就是按阶梯价格分批买，按阶梯价格分批卖

比如对于某个股票

如果觉得10块可以开始买入了，则在10块的时候买入一部分，然后9块8块7块分批买入

如果觉得15块可以开始卖出了，则在15块的时候卖出一部分，然后16块18块21块分批卖出

这样的好处是在下跌的过程中，可以逐步的低价买入，平均买入成本会比一次性买入低

而在上涨的过程中，可以逐步的卖出，平均卖出价格会比一次性卖出要高

这个是基于我们可以通过估值判断股票处于低估区域，但是无法判断底部的价格，而采取的一种容错方式

避免买入价格过高，卖出价格过低

既然有优点，那么就一定有代价

代价就是

买入的时候可能只买了一部分就不跌了，后面的部分没有买到，最终没有买够计划的份额

卖出的时候可能只卖出了一部分就不涨了，还有一部分没有卖出，仍然留着手里

当然这个代价是无法避免的，因为很难精确的抄底和逃顶。

另外还有两个问题

一个就是为了按计划买入，必须得保留一定比例的现金，造成这部分资金的收益降低

另外一个就是，容易在牛市的起步阶段卖完而踏空后面的涨幅。

而如果在股票组合内用网格交易来轮动则会比单一股票的网格交易要好的多，可以在一定程度上降低上面所说的代价

假设一个股票组合里面有20只股票

按上面的例子，一个股票设定了4档买入价格，假设最后两档价格出现的概率是50%

假设计划的总资金是100W，那么用于买入后面两档的资金，有一半的概率可以买到，一半的概率买不到

长期下来看数学期望就是只能买入 50W+25W，平均有25W的资金没有机会买入

但是如果组合里面有20只，在买入前面两档后(耗资50W)，给每个股票的最后两档设定5W的买入计划，哪个跌到设定的价格就买入哪个，由于最后两档的价格出现的概率是50%，因此有10只股票不会出现最后两档价格，有10只股票会出现最后两档的价格

那么最终用于买入最后两档的50W资金，将会按计划买入其中10只股票。

这样相对单一股票，可以获得更好的买入价格。

而在卖出时，如果组合中的股票有多个(比如20个)，那么卖出效果也会更均匀

比如单一股票的卖出计划是4档

则有可能只卖出了2档，也有可能卖出4档

那么如果是在组合中，会有一部分股票卖出2档，一部分股票卖出3档，一部分股票卖出4档

然后卖出的资金又可以作为买入计划的资金，而这一部分资金在组合中的利用效率是高于单一股票的

那么在横盘的阶段，就可以用网格交易，在组合中不断的轮动。

当然这里仍然没有解决踏空牛市的问题

如果要尽可能的避免踏空牛市，那么对于网格的卖出计划就需要做一些调整

比如，4档卖出价格，对应的卖出数量设定一个上限，比如20%

如果接下来的是结构性牛市，也就是不同的板块轮番涨跌，那么持仓中的20%的仓位将通过这个策略在这种行情下不断的轮动持仓而获得超额收益。

如果20%卖出后，市场行情继续上涨，那么可以再规划20%的仓位来做第二阶段的4档卖出。

如果卖出后，价格回到第一阶段的4档价格范围，可以分批再接回，如果接不回，就得再次规划20%的仓位卖出

这样如果市场是阶梯式走向牛市，我们就可以分几波卖出，获得较大的收益

不要奢望从头吃到尾，这样容易在转熊的时候锚定高点舍不得卖出，死扛而坐过山车，而市场终究是会均值回归的。

如果接下来的是全面熊市，那么这个没有什么好办法可以解决，只能等待。

总体来说，用网格交易的方式，在组合间不断的轮动，可以获得更平滑的收益曲线，改善持股体验，并且在横盘阶段和结构性牛市阶段能获得一部分超额收益。
