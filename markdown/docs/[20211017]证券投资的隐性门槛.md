#  证券投资的隐性门槛

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

参与证券投资几乎没有门槛，长期持续获利却有极高的门槛。

基本上只要完成了义务教育，年龄够了，就可以开户来参与证券投资，买入常规股票一手少的话小几百就可以了，ST股的话则需要的资金更少。

但是想要长期持续的获利，实际上是有很高的隐性门槛的，之所以说是隐性门槛，是因为很多人可能并没有意识到这些门槛的存在。

下面一个一个来说说

##  区分偶然和必然

抛硬币游戏，每次下注一元，如果是正面则赢一元，连同本金能拿回两元，如果是背面则输掉本金。

那么是不是可以这样，我去下注，只要正面比背面多一次，我就可以赢一元，然后离场，这样就可以赚一元。

看上去好像很容易对不对？事实上很多人参与股票投资也是这样的。

技术分析或者趋势投资，确实有少数有天赋的人可以赚钱，这个就像职业德州扑克选手，可以通过赢对手来赚钱。另外做局的人也可以赚钱。

那么对于大部分的普通散户来说，这更像是一个猜大小的游戏，和抛硬币没有本质区别。

但是即使是这样，仍然会有一些人在短期内获得几倍的回报，就像连续抛硬币抛出两次正面，一元能变成4元，而且比例能高达25%左右。

如果赢钱的人就此离场，当然是可以带着赢到的钱。但是有多少人在通过某种技术指标或者某种策略赚到钱以后会离场从此不再入场？又或者在牛市的氛围中买入后看着账户的钱在增长能迅速离场？

这样的人是极少的，更多的是再次操作，并且通常会增加本金，而这个时候一旦亏钱，则会亏的更多。

所以参与抛硬币的人，在整体上是零和的，其中有人赚钱，有人亏钱，有人不赚不亏。

如果散户没有过人的优势(不论是交易天赋，还是技术分析天赋，或者资金优势，抑或信息优势)，在这个过程中，赚钱是偶然的，长期亏钱则是必然的。

而如果沉迷在几次赚钱的偶然中无法自拔，那就会像抛硬币一样，赚了亏，亏了赚，始终不能积累盈利。

##  耐心

有耐心的人本来就少，在投资市场中就更少了，毕竟大家进场是来赚钱的，能快一点赚钱谁愿意慢慢变富？

但快速赚钱，不是想不想的问题，而是能不能的问题。

除了少数确实有显著的优势并且抓住了机会踩准了点的人，可以快速赚到比较多的钱，大部分短期赚到钱的情况都具有偶然性，这就又回到了上一个问题，区分偶然和必然。

有一些人经过多次被割，最终意识到很难快速赚钱，或者有些人一开始就已经明白了很难快速赚钱，而选择了稳健的价值投资风格的策略，但是依然会面临其他的阻碍。

##  资金量

如果说技术分析、趋势投资等偏重交易的投资方式难度较大，对个人能力的要求很高，同时由于散户在这场竞技中处于劣势而导致亏钱，那么传统的价值投资则对个人能力的要求低得多，尤其是格雷厄姆风格。

但是这样的投资方式依然存在很高的门槛，比如资金量。

根据统计，A股的投资者，72%的资金量在10W  以下  。

这意味着什么？

如果是按照传统的价值投资方式，由于散户能投入查看公司资料及数据的时间非常有限(毕竟还要上班)，按照最简单的策略通常能做到在长周期(比如5年到10年)大概年化10%的水平。

那么这个水平就意味着，在不增加资金的情况下，平均经过7年，10W本金可以增长为大约19.5W，平均一年大约1.35W。

如果一个人拿着积蓄，花了7年时间最终平均每年的回报大约只有1.35W元，有多少人会满意？

因此也有很多人说，资金量大(大几十万上百万)，年化10%的回报就满足了，那么实际上意思就是钱少的时候10%的回报肯定是无法满足的。

##  回撤

如果说第一年就有1W多的回报，应该会有很多人能接受，但是投资的过程通常不是线性的，第一年未必能赚钱，可能连续两年都没有赚钱甚至浮亏，这个时候还有多少人能接受？很多人可能就割肉，然后就开始寻找快速赚钱(回本)的方法，这就又会面临最开始的问题，区分偶然和必然。

##  现金流

在投资遭遇大幅回撤的时候，通常也是市场处于底部区域的时候，这个时候最应该做的不是割肉，而是用打折的价格买入优质的资产。

而这个时候，如果投资者的现金流不充裕，不要说趁便宜买入优质资产，想要熬过这段时间等到回报的到来都极其困难，更何况还有可能遇到需要急用钱的情况，就只能被迫割肉了。

而这个时候，如果投资者的现金流充裕，不仅可以趁便宜买入优质的资产，还能在别人割肉或者爆仓导致市场出现极端低价的时候顺势买入，然后用充裕的现金流度过漫漫熊市，积攒大量的便宜筹码，等待牛市市场火爆的时候将筹码还给市场。

##  知识

知识在投资中的价值非常大，但是这个知识的范围太大了，这里只说几个最基本的并且只需要花一点时间就能掌握的。

比如股票的PE/PB/ROE，不仅很多新蔬菜不知道这几个财务指标，不少老蔬菜也不知道或者不看。即使一个企业再优秀，如果在市场火热的时候高价买入，能赚钱的空间也非常小。

至于财报中体现出来的其他可以用来排除潜在隐患的指标(当然也可能误杀)，也并不复杂，但是能认真去看的人也是非常少的。

转债的转股价值、溢价率、强赎规则、回售规则等，也没有任何难度，只需要花一点时间去了解都能正确掌握，但是又有多少人能静下心来了解，如果愿意花一些时间去了解并且不贪心，按照有利于散户的规则去做稳健的投资，至少能避免这样的悲剧

还有其他有价值的知识，就不一一列举。

  

那么作为一个普通散户，可以问问自己，这些隐性门槛，自己是否已经跨越。
