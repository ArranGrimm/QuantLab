#  企业(股票)的生命周期与投资人在这个周期中的获利来源

在小说阅读器中沉浸阅读

企业最终的归宿是消亡，但是这并不妨碍投资人在企业的生命周期中获得应有的回报。

这篇文章对大部分人来说应该是没有什么用的，因为大部分人是不会真正理解股票与投资的，可能是没有真的想去理解也可能是理解了但是不愿意接受。

但是，总是有少数人能理解并最终通过投资获得长期的回报，如果你希望成为这样的人，那么这篇文章应该对你有用。

好几年以前，我看过一些关于投资的书，但是还没有开始投资股票，有一天看到一篇文章，文章标题已经不记得了，大概意思是说，年轻人想要获得大额财富，只能靠股票(股权)。

文章里面写了很多在硅谷工作的人通过期权获得财富自由的例子。

实际上这样的例子在国内也有不少，尤其是IT行业，当然传统行业也有，只是稍微少一点。

当时我对文章的内容没有什么异议，毕竟这些情况是确实存在的。

但是我脑海里面有一个问题：

企业最终的命运是消亡，如果企业消亡了，股票归零了，那岂不是一场空？

这个问题如果没搞明白，则说明我根本就没有明白股票到底是什么，以及投资股票到底赚的是什么钱。

我始终在思考这个问题。

**因为我打算通过投资度过富足的一生。**

在随后的几年中，我终于找到了答案，于是我开始下场进行股票(以及可转债)的投资。

要回答上面的这个问题，首先需要知道一个企业的生命周期。

抛开那么骗子公司或者昙花一现的公司不说，常规企业的生命周期通常应该是

创始阶段->发展/高速发展阶段->发展/稳定发展阶段->成熟/衰退阶段->消亡。

区别在于，有些公司的发展阶段特别长，而且可能经历多个发展阶段。

在发展阶段，企业的利润一直在增长，这个阶段股票获利主要来源于企业的分红或者企业净资产的增厚带来的市值增长。

比如企业的年利润从10亿增长到50亿，规模(净资产)扩大了5倍，利润变大了5倍，现在这个企业相当于1变5，如果股票没有增发并且估值也没有变的话，市值也应该是扩大5倍，那么投资者就获得了400%的回报，这个是企业发展带来的回报。

那么，最终这个企业会走到 成熟/衰退阶段，这个时候企业的利润不再增长(甚至开始降低)，这个时候对企业应该如何估值，持有股票如何获利？

股票属于权益类资产，通常认为股票的回报率要达到无风险收益的2倍以上才具备吸引力。

假设当前无风险收益率是3%的话，那么股票的回报率要在6%以上，才具备吸引力。

那么假设我们对股票的期望是8%的回报率，那么对于利润不增长的企业，我们能接受什么样的估值？

##  假设利润全部分红

假设一个企业的总市值是100亿，总股本10亿，则股价是10元。

年利润是8亿，全部分红。

则PE=总市值/利润=12.5

如果我们买入这个企业的股票，每股10元，每年能分红0.8元，则年回报是8%。

那么这个企业的PB是多少？

实际上PB是多少已经不重要了，因为在不能进行破产清算的情况下，不论这个企业的PB是0.5还是2，只要每股每年分红是0.8元，并且市场接受8%的回报的情况下，股价就会是10元。

如果不能进行破产清算，在这个企业上投资者短期可能赚到波动(博弈)的钱，但是长期看只能获得这个企业的利润分成。

因此对于这个企业，估值就是PE=12.5，PB高低都可以。

但是实际上这个情况是不可能发生的，因为如果这个企业能够确定每股分红0.8元，这0.8元就接近于无风险收益，那么机构可能会将其作为基金或者理财产品的底层资产，那么可能收益率只需要有4%到6%即可。

那么这个时候对应的股价就是13.3元到20元，而对应的PE则是16.625到25。

现实中肯定不存在每年利润和分红都维持在一个水平的企业，但是接近这个情况的企业我倒是知道几个。

比如重庆水务，过去十年分红基本稳定，过去5年的净利润在10亿到20亿之间，目前的估值PE是11.97，PB是1.55。

如果能破产清算，则是另外一番情景，这个在不分红的情况中讨论。

##  假设利润不分红

假设一个企业的总市值是100亿，总股本10亿，则股价是10元。

年利润是8亿则PE=100/8=12.5，不分红。

假设PB=2，则净资产为50亿，ROE=8/50=16%。

把时间推移到30年后，净资产为 50+8*30=290亿，年利润还是8亿。

如果股价还是10元，则PE=100/8=12.5，PB=100/290=0.3448，ROE=8/290=2.76%

投资者面对的是一个企业的净资产回报率从16%下降到2.76%，如果从净资产回报率的角度看，这个企业的盈利能力已经非常弱了，净资产回报率比无风险回报都低，而投资者在持有股票30年的过程中，可能一分钱都赚不到。而市场可能连PE12.5和PB0.3448都不能接受而给出更低的估值。

除非，企业进行清算。

清算不一定就是关门变卖，还有很多种形式。

比如，大比例分红，今年很多企业就这样干了。

比如，被别的公司收购，通常会溢价，这样投资者倒是能获得一些回报。

如果不能清算(比如国有企业)，则出现这样的估值，并不稀奇，比如上海实业控股，目前PE是5.95，PB是0.3，当然它的盈利能力稍微好一点而且有一定的分红，2020年的ROE为5.22%，之前4年在8%，如果ROE进一步降低则PE会上升。

所以一个企业如果利润不再增长，长期看(几十年周期)是不值得投资的。短期看(几年周期)如果估值较低ROE尚可倒是可以考虑。

  

上面这两种方式，是比较稳妥的获利方式，也就是常说的价值投资。当然市场上还有很多人是通过股票博弈，不论是看资金流向、K线、消息政策等，都是博弈，如果想通过这样的方式获利，得想想清楚，自己有什么优势，凭什么能获利。
