#  散户不要轻易相信小作文

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

前几天高盛发布了一篇关于国内银行的看空报告。

咋说呢，文章内容列举的数据，肯定都是真的，但是结论并不可靠。

比如它的核心逻辑是这样的

** 因为国内银行持有较多地方政府的城投债，比如某某银行持有多少，而某某银行又持有多少，所以给于某某银行什么评级。  **

看上去好像好有道理的样子，实际上经不起推敲。

随后几天，几家券商的研究员就发表了针对性的文章来说明高盛的报告中的问题。

当然，这里就有一个相信谁的问题，尤其是对银行股并没有深入了解的散户。

相信老外还是相信自己人？

这里顺便提一嘴，之前美国的硅谷银行和第一共和银行破产之前，高盛貌似就没有唱空，至少没有看到有报道说高盛唱空。

这个很有意思了，美国的这几个银行都特么要破产了，没有唱空。

中国的这些银行，每年稳定分红，估值还这么低，却唱空。

确定是认真的吗？

这里我可以现场编一个笑话

**高盛：你国的银行，持有很多城投债，这些债后面可能违约，所以我看空你国的这些银行。**

**我：那么请问阁下，既然我国的这些银行资产质量有瑕疵，那么应该给到什么样的估值呢？**

**高盛：起码应该低于净资产，打个7折也不过分。**

**我：阁下莫不是在说笑吧，贵国的硅谷银行在摆明了要破产，停牌的时候PE是3.76，PB是0.61，1992年之前每股派息几美分，1992年之后再也没有派息，而您给于卖出评级的兴业银行，目前(2023年7月7日收盘)PE是3.63，PB0.47，股息回报7.66%，比硅谷银行的破产价还低，还需要看空吗？**

**高盛：。。。**

  

说完银行，再说一下前一阵的另外一个小作文

**阿斯巴甜将在7月被世界卫生组织（WHO）旗下的国际癌症研究机构（IARC）正式列为“对人体可能致癌”（possibly carcinogenic to
humans）的物质。**

吓得一众散户赶紧看看自己手里的持仓会不会中枪。

然而事实是什么？

**阿斯巴甜列为“2B类”致癌物**

当天好几个了解致癌物标准的博主表示，看到这个新闻，赶紧喝了一瓶无糖可乐压压惊，为什么呢？

IARC一般致癌性分为5种：1类（有确认致癌性）、2A类（很可能有致癌性）、2B类（有可能致癌）、3类（尚不能确定是否致癌）、4类（基本无致癌作用）

2015年10月26日，IARC曾发布报告，将香肠、火腿、培根等大众熟悉的加工肉制品列为“致癌物”，新鲜牛羊肉等红肉列为“较可能致癌物”。

这个意思就是，牛羊肉比阿巴斯甜更可能致癌，火腿肠比阿巴斯甜可怕多了，而白酒则是一类致癌物，茅台和五粮液的股东表示情绪稳定。

这里我还可以现场编一个笑话

**散户：阿巴斯甜可能致癌啊？咋整**

**我：你平时晚上熬夜玩手机吗？**

**散户：我经常这样啊**

**我：那你晚上熬夜玩手机的时候喝无糖可乐吗**

**散户：不怎么喝**

**我：那你今天晚上喝一瓶无糖可乐，然后早点睡**

**散户：为啥呢？**

**我：因为熬夜可能致癌，而且可能性比阿巴斯甜高。**

**散户：。。。**

  

而且市面上的小作文有一个特点，经常是大家普通看好的时候出现利好小作文，在大家普遍不看好的时候出现利空小作文。

这有两个原因，一方面是看好的时候利好的小作文会被更多人转发而传播的更广，同理利空也是。另一方面，可能是有人在其中故意运作，达到自己的目的。

散户如果不想亏钱，还是不要轻易相信小作文
