#  散户可以在市场中赚哪些钱

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

上周写了一篇《 [ 散户的分析时常令人发笑
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484850&idx=1&sn=e816348f72d152c3dc7d6552b1e110f9&chksm=cf17b438f8603d2e978cc9d3d3ec8710b82c909e00633e0094b5e319d3e114496425b3bead12&scene=21#wechat_redirect)
》

虽然说绝大部分散户其实没有分析能力，但是并不代表不能  在  市场中赚钱

因为市场中的很多钱，要赚并不难，并且也不需要高深的能力

比如买入指数基金，赚市场平均收益

这个平均收益在10%左右，这个已经是被证实的最简单的投资方式，对能力没有要求

唯一的要求就是要有定力

比如假设采用定投沪深300的方式，不能说市场跌了(比如今年)就觉得市场不好不投，哪一年市场涨了，就忍不住多投，这样的结果就是便宜的时候买的少，贵的时候买的多，结果会变差甚至亏钱。

而如果在这个基础上，稍微  再  提高那么一点点，在低估区域多投一点，在高估区域少投或者忍住不投，还能略微提升一点收益率

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfLjsP7P0IvTJIPWbGBLmS2StIQgk1UoG9LLs9Wia8xVZvpMbK5MSqk9acMXt0cGHs6hG9FmJ4Qbxjg/640?wx_fmt=png)

但是实际上用这种方式赚到钱的人也不多，一方面平均年化10%对于投入小(比如十万上下)的人来说，回报实在是没有什么吸引力，别人一个涨停就10%了，而平均一年只有1W的回报，还不如跳槽一次涨薪多，而且还会遇到年化为负数的年份，比如今年年初本周收盘(8月19号)沪深300的收益率就是-15.98%，这些都足以让大量的人放弃这个策略。  

比如赚打新(可转债/股票)的钱

在今年6月18号之前，开户即可以打可转债，基本上属于无风险收益，一个账户一年大概1500左右

如果是大学生，动员爸爸妈妈爷爷奶奶外公外婆开通，全家就有7个号，一年就是一万块左右

如果是两口子，动员双方爸爸妈妈开通，就有6个号，一年也接近1W，如果加上十万市值指数基金的收益，一年平均能有20%的资金收益，已经算不错了。

不过6月18号之后，新开户要2年经验十万市值才能开通可转债权限，这个将大部分新号堵在了门外，只能说干啥都得趁早。

而股票打新，即使今年破发那么多，从元旦至今单账户10W市值的打新理论折算年化收益率依然在5%以上

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfLjsP7P0IvTJIPWbGBLmS2SU4UhBGx11KNAVHHDD304GiajXariaOibIEWiciaw6dW37VQJyCukRmg8lOw/640?wx_fmt=png)

当然这个差异很大，有人几万市值一年中一个就赚万把块，有人几十万市值一年一个都没中，我今年到目前中了4个赚了一万多，去年中了5个赚了7.5W，只需要配置好合适的门票即可，这个我在《
[ 新股门票筛选思路之一：高息低估
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484131&idx=1&sn=9bd51e137782e2e3e4f63c544260ae26&chksm=cf17b369f8603a7fe6ed4824fd9658944eaf784d4512be98496649d7f93c72eb72b030ac65db&scene=21#wechat_redirect)
》写过，在知识层面没有什么难度。

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLjsP7P0IvTJIPWbGBLmS2S3N7aAneav6ROqUIdOgedket9yyibGtFjWyCnCa7QYL5D61e3QtajzSw/640?wx_fmt=jpeg)

这个我上市第一天250块不到全出了，赚了不到1W，后来最高涨到464，不过后续属于炒新的游戏，我不会玩所以没有参与，错过了就错过了。

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfLjsP7P0IvTJIPWbGBLmS2ScSgbKttswicWAjcRS7v5S6GAOE0UGtwUKPhVwVACRRrV0wYibBX7zPsw/640?wx_fmt=jpeg)

另外还有一个新股赚了几百块，一个亏了两千块。

还有通过可转债轮动赚钱，基本上只需要花上一个周末的空闲时间学习一下就足够了，比起分析企业实在是简单太多太多，这个我在《 [ 双低可转债摊大饼策略简要分析
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483949&idx=1&sn=919077321db56f33fa01ac35b6581927&chksm=cf17b3a7f8603ab1ef1c5d29cbd3acc5e7503ba5a7f99531da17d1d739d61cfcaad85ce2f3f0&scene=21#wechat_redirect)
》中写过，我的可转债组合从2021年1月建仓以来，到现在收益率大概是55%，其中45%是去年获得的，10%是今年元旦至今的收益率，目前市场上的散户对于可转债的价值认识不足，而机构受限于评级要求很多转债不能买。如果说去年的高额收益是可转债的黄金坑带来的，那么今年的收益则是可转债本身的价值提供的。

还有其他套利以及在市场非理性的大涨大跌的时候轮动，这些之前也写过，难度也不大，有兴趣的可以翻以前的帖子。

所以散户只要稍微懂那么一点，并且能控制住自己的手和心态，就可以从市场中赚取属于自己的那一份钱。
