最近 亚药转债 花王转债 久其转债下修

关于可转债的基本概念，这里就不写了，不清楚的可以参考我之前写的《 [ 可转债的期权价值
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484595&idx=1&sn=02adf12c65e887bccf5ce2407b11d027&chksm=cf17b539f8603c2f3ebea54325e6d110895761ecac87941b8bff0b7ba92c5bf25b362c5ccdd8&scene=21#wechat_redirect)
》中间的一段

我曾经持有 花王转债 并且非常关注 亚药转债

当时(2021年1月前后)的亚药转债，是全市场价格最低的转债

最低价到了66.6，而它的回售起始时间是2023年4月3号，也就是说如果在2021年1月13号前后以67元的价格买入的话，到2023年4月3号回售给公司，将获得100元的本金以及
2021年至2023年的派息合计3元

相当于2年零3个月，回报率是53.73%

而在8月19号晚间公布了董事会提议下修后，本周价格就站上了110元。

也就是说如果在2021年1月13号前后以67元的价格买入的话，到现在1年零7个月，回报率大约是66%

当然这个属于后视镜，当时没有多少人会在70块以下买入。

当时大家觉得亚药有破产的风险，2020年的一季报、中报、三季报都是亏损，已经快资不抵债了。

所以这个转债我没有买入，而有人在80多就开始买入，跌了以后又加仓，不知道有没有在70块以下的时候扛不住压力割肉。

而花王转债我在90块附近最低87块多买了一点，作为可转债摊大饼中的一份，占比大概3%多一点

后来花王也爆出一系列问题，股份也大比例被质押了，基本上就是公司有明显的问题，后来也ST了

当时也是觉得这怕不是要破产，真要是破产，等到清算也不知道啥时候能拿到钱。

但是考虑到可转债摊大饼本来就已经把企业破产的风险考虑进去了(见 《 [ 双低可转债摊大饼策略简要分析
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483949&idx=1&sn=919077321db56f33fa01ac35b6581927&chksm=cf17b3a7f8603ab1ef1c5d29cbd3acc5e7503ba5a7f99531da17d1d739d61cfcaad85ce2f3f0&scene=21#wechat_redirect)
》)，不应该过于忧心忡忡，毕竟就算归零也才不过3%多一点的损失而已。

但是这个过程并不轻松愉快，后来又陆陆续续的爆出一些问题，于是我去年10月在90块多一点的价格卖出了大概一半，剩下一半只占转债组合的1%多一点，心想就留着格局一下算了，如果真的破产清算，那就把这个过程经历一下。

然而没想到的是，今年4月29号正股涨停，转债大涨20%多，于是我就趁机在121的价格卖出了剩下的一半，最终这一部分收益11%多。

然而没想到的是，下一个交易日，正股继续涨停，转债最高涨到了144.5。

现在回过头来看这些价格，都如同答案一样呈现在K线图上

很多人看着K线图，会觉得到处都是机会，这其实都是幻觉。

真正身处其中的人，感觉是完全不一样的，他们感受到的是真实的风险。
