#  基于转债强赎的投机，光大转债有机会吗？

在小说阅读器中沉浸阅读

前两周国贸转债为了启动强赎而进行的精准控盘，让我对这一类机会产生了兴趣。

先简要的回顾一下国贸转债为了强赎而精准控盘的过程

今年6月的时候，有一个转债大佬和我们提到他在108元的价格建仓了国贸转债，理由很简单

国贸转债将于2022年1月4号到期，6月距离到期只有半年时间，如果半年内不能启动强赎，则厦门国贸就需要按108元兑付可转债

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIfDUF00Eo0VibiaHCwmAp7YXCJRrN0KrFygNaPzsVicicIBPJ23zgBtcK4rgx2Cpice1KQDMF3kRhzrng/640?wx_fmt=jpeg)

那么在6月的时候用108元买入国贸转债相当于，如果国贸转债将转股价值拉升到130元以上，则投资人半年内能获得20%的回报，而如果厦门国贸的股票一直跌，则半年以后投资人依然可以拿回108元本息，啥都不用干。

而厦门国贸肯定是希望能启动强赎的，也就是说将转股价值拉升到130以上的可能性会很大

那么相当于较小的可能性白玩，较大的可能性能获得20%的回报，这种稳赚不赔的投机，是值得参与的。

在9月24日之前，国贸转债的正股已经在连续29个交易日中有14个交易日满足了强赎的价格要求，那么只要24号收盘时再满足一天就可以启动强赎了。

然而在这一天，厦门国贸的股价却晃晃悠悠到了强赎标准(8.74元)之下，到收盘前的最后一分钟也只有8.66，然而在最后一分钟，一个4万手的买单将收盘价拉升了1毛，最终的收盘价定格在8.76元，至此国贸转债刚好满足了强赎条件。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIfDUF00Eo0VibiaHCwmAp7YXCJzibVfAsaDe8ia9zlRT9XxAVYNunLtCWHDMqA9AVRKmrAsq2WJ4B86A/640?wx_fmt=png)

  

收盘后不到3小时，厦门国贸就发布了强赎公告

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIfDUF00Eo0VibiaHCwmAp7YX1ZWl3c7h87JEicHSfUOicibBFaicd0ryiatUwyssDXicUcto48bTUJsm2XicQ/640?wx_fmt=jpeg)

  

也就是这个过程，让我对这一类的机会产生了兴趣。

那么现在还有没有类似的机会？

有的，比如光大转债

利益相关：我不持有光大银行，也不持有光大转债，以下全部是猜想，不作为投资建议。

先来看一下光大转债的基本情况

> 到期日：2023-03-17
>
>
> 转股价：3.55不低于，《募集说明书》公告前三十个交易日、前二十个交易日发行人股票交易均价和前一个交易日交易均价，以及最近一期经审计的每股净资产和股票面值）。
>
> 下修条款：在本可转债存续期内，当公司 A 股股票在任意连续三十个交易日中至少有十五个交易日的收盘价低于当期转股价格的
> 80%时。修正后的转股价格应不低于该次股东大会召开日前三十个交易日、前二十个交易日和前一交易日的公司A股股票交易均价，同时，修正后的转股价格不得低于最近一期经审计的每股净资产值和股票面值。
>
> 强赎条件：转股期内，公司 A 股股票在任何连续三十个交易日中有至少十五个交易日的收盘价格不低于当期转股价格的 130%（含
> 130%），当本次发行的可转债未转股余额不足3,000万元时，公司有权按照债券面值加当期应计利息的价格赎回全部未转股的可转债。
>
>
> 回售条款：如公司本次发行可转债募集资金的使用与公司在募集说明书中的承诺相比出现重大变化，被中国证监会认定为改变募集资金用途的，可转债持有人享有一次按面值加上当期应计利息的价格向公司回售其持有的部分或全部可转债的权利。
>
>
> 债券到期偿还：在本次发行的可转债期满后五个交易日内，发行人将以本次发行的可转债的票面面值的105%（含最后一期年度利息）的价格向投资者兑付全部未转股的可转债。

  

来一条一条的看

到期时间，还有大约一年半

转股价，发行时的转股价是4.36元，由于派息后调整转股价，目前的转股价是3.55元，参考光大银行的派息历史，明年大约在6月或者7月将再次派息，预计在每股2毛左右，派息后转股价将再一次向下调整。

下修条款，这一条基本是摆设，目前光大银行的每股净资产是6.62元，正常情况下明年的净资产将会有小幅的增加，所以这个下修是没有空间的。

强赎条件，要么转股价值提升到130元，要么转债余额不足3000万。

回售条款，这一条也基本是摆设，其他的转债都是一条类似这样的"在本次发行的可转换公司债券最后两个计息年度，如果公司股票在任何连续三十个交易日的收盘价格低于当期转股价的
70%时"，就是说如果股价跌的厉害，投资人可以将可转债回售给发行公司，但是光大转债没有这一条，同时光大银行正常情况下是不会变更资金用途的，因此这一条就成了摆设。

债券到期偿还，这个就聊胜于无  吧  ，投资人到期连本带息拿回105元(其中利息还要交税)，而目前光大转债的价格是114.65元，转股价值95.49元。

那么光大转债只有  以下  几个结局:

结局1：到期赎回，那么以目前的价格买入，到期赎回时会亏损几块钱。

结局2：在到期前让转股价值到比转债价格高一点(比如高5%)，这样会有大量的套利党买入转债后转股，转债余额少于3000万时光大银行即可强制赎回剩余部分，当然如果转债余额还剩几个亿，到期还钱也关系不大，毕竟发行额度是300亿，目前余额还有大约240亿。

这里有两个途径，一个将转股价值拉升到转债价格之上，如果按目前的价格，股价需要拉升到
114.65*(1+5%)/(100/3.55)≈4.27元，目前的股价是3.39元，需要拉升大约26%，并且前提是转债价格不上涨。

另外一个途径是将可转债的价格砸到比转股价值低一点(比如低5%)，那么按现在的转股价值，则需要将转债的价格砸到90.7元，不过砸到这个位置的可能性很小，因为当价格达到95元附近的时候，光大转债将具备显著的投资价值，95元买入只需要持有一年半，则至少可以拿到105元，一年半的回报率是10.5%，而如果后期股价拉升，那么回报率将更高。

结局3：转股价值提升到130元，并抓住机会启动强赎，就像国贸转债一样，这样就需要把股价拉升到130*/(100/3.55)≈4.62元，相对当前股价需要上涨36.28%

对于结局1
是光大银行和投资人双输，这个也不是光大银行想要的结局，市值才1800多亿，要还回去240亿，白搞了6年，虽然没有付出多少利息，但是要弄这么多钱来还回去也是一项艰巨的任务。

对于结局2
通过拉升股价来制造套利空间，操作套利的人并不会太多，如果需要市场通过套利降低转债余额，维持套利空间的难度很大。而如果通过砸盘转债则成本很高，并且低价买入的人不一定会去套利而是选择持有到期。

对于结局3 这个应该是投资人和光大银行都希望看到的结局，如果到了明年分红调整转股价后，只需要股价在30个交易日内  有
15个交易日维持在大约4.4元(粗略计算)之上，就可以启动强赎，而在3年多之前光大银行就曾经做到过，见图(不复权)

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIfDUF00Eo0VibiaHCwmAp7YXuRPc4lgYqibzVAFWqjcPjYjLBHYUW9TU5HxOJCN72xsJAFdmwOG36oA/640?wx_fmt=jpeg)

那一阵貌似银行股都是大涨的(工行农行建行中行等)

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIfDUF00Eo0VibiaHCwmAp7YXvOtk11jic9GnKeUC4QegWiapbFmrUOHIsQtIWRNibPqL1B8QgU5VQT79Q/640?wx_fmt=jpeg)

所以在未来一年半股价上涨到强赎线也不是不可能，但是这个可能性有多大？而如果没有整体行情，则需要释放利好，或者用一些资金买入，相比起厦门国贸100多亿的市值，光大银行想要拉升难度会要大很多。

然而光大银行似乎还是挺有信心的

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIfDUF00Eo0VibiaHCwmAp7YXGRiaxFpQ7szVUMVkGYCVC7cJG8gSEg9xGuWP2AzEHT3zkqxAiboBBgiaw/640?wx_fmt=png)

不知道这个信心是从何而来，不过倒是可以考虑买入几千股光大银行参与一下。

至于最终会走到哪个结局，现在无法判断，只能拭目以待。
