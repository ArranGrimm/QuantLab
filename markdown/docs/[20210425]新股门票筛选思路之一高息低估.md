#  新股门票筛选思路之一：高息低估

在小说阅读器中沉浸阅读

打新股算是A股市场的制度福利，也是投资之路早期获利的一个重要组成部分，这个我在《 [ 财务自由天梯攀登指南
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247484051&idx=1&sn=30aa543d86960125eb3a0ccd8dfb879e&chksm=cf17b319f8603a0f42df7d6717a49698eef0ea74f81ce722f9844611afaab79f27097890816a&scene=21#wechat_redirect)
》中有写到过，后来很多人问我门票的问题，我仔细想了一下，觉得应该从问题本身入手。

以下内容带有强烈的个人偏好并且也不能保证严谨，仅提供一种思路作为大家的参考，不构成投资建议。

为什么会有筛选门票的问题？

之所以会有如何筛选门票的问题，大概是考虑到钱存定期和理财收益太低了，想参与打新股但是又觉得股票风险大，又或者想从门票入手熟悉股票投资。

新股按统计来说收益率还可以，但是如果资金较少比如10W以内，即使每次申购都没有漏掉，估计大概率3年中不了一个新股，如果开通了科创板以后分3个账户每个10W，一年两年中不了一个也很正常。

而这个期间如果股价跌了(即使以后会涨回来)，难免会让一些人产生心理压力甚至割肉导致亏损。

所以如果想要参与打新股，一方面需要适应股票的波动，另外一个方面是挑选合适的股票作为打新股的门票，从这个角度来看，高息低估的股票是比较合适的。而且也比较合适新入股市的投资者，这个我在《
[ 股市萌新可以考虑从高息股开始入手
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247483873&idx=1&sn=a62ebde57025d705d98df8550daa563b&chksm=cf17b06bf860397dd7be217acc6556741c69e46903aba99979a7dead912662dfa96f90c40dca&scene=21#wechat_redirect)
》也写到过。

那么筛选的方向就是PB较低(最好是2以下)，PE较低(最好是15以下)，股息较高(至少3%)，ROE尚可(至少7%)。

还需要查看这个股票过去5年的财务情况和分红情况，剔除掉起伏较大和分红不稳定的股票。

挑选不同行业的至少10只股票，这样相互的涨跌可以对冲，平抑波动。

同时参考一下上证指数最近几年的起伏，高息低估的股票以国资委控股的成熟企业为主，没有题材、没有故事、没有想象空间，通常是随着上证指数的起伏而起伏的(当然也有例外情况)。

市净率MRQ 设置最小值0.3，最大值3

股息收益率 设置最小值3，最大值15(实际上设置为10就可以了)

总市值 设置最小值100亿以上

第三名，中国神华，看一下分红的情况

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKlTQ3GJvnKpG3VRjia1ianuz5c4MIicibloH6EGCS96ChFlCPR14vsRWvCbZ0V1cdaoH9OY7icjO14FUQ/640?wx_fmt=png)

今年分的比较多，过去3年分红在每股1元上下，如果明年每股分1元，按现在的股票19.55元(上证指数3474点)，则股息回报是1/19.55=5.1%，略高于理财产品，股价波动相对较小，2020年3月最低13块多(上证指数最低2646点)，2018年2月最高24块多(上证指数最高3495)。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKlTQ3GJvnKpG3VRjia1ianuz2amH4Vibw4J80PDXfTDEWpaQ5vlnT7iaRmSLVbUbDJ0Nt1geToEQzmQw/640?wx_fmt=png)

可以看到中国神华由于其业务成熟业绩稳定，价格基本上随着指数涨跌，因此在当前的位置价格不算贵，不过由于今年的高分红已经涨了一波，如果想作为门票，可以价格下来一些以后再考虑，因此作为备选。

接下来挑几个我持有的简短说明一下。

唐山港

今年的分红方案公布之前，过去4年的分红在每股0.1元或者接近每股0.1元。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKlTQ3GJvnKpG3VRjia1ianuzIoRZ7gv42wkFlPPqla3ia4Ybvj8uA5sxAicwO2ZfkMC3uK0OZ3tU6rVg/640?wx_fmt=png)

我是2月5号筛选新成员的时候看到唐山港的，当时的价格是2.26元，PB大约0.72，PE大约7.3，，按去年每股0.09元的分红，股息率为0.09/2.26=3.98%，考虑到估值很低，如果市场稍微回暖一点，估值略微提升比如PB估值提升到0.9，那么就有25%的涨幅，因此我买了20手，没想到的是3月30号公布了分红方案每股分红0.2元，比去年高出了1倍出头，结果就是3月31号涨停了，4月1号最高涨到了2.85，接下来的时间价格就在2.7元左右徘徊，考虑到短期涨幅较大并且按这个价格(不考虑历史成本)股息不太具有吸引力，估值也不是那么低，所以我减了一半，换成其他股票。

这完全是一个意外的收获。

大秦铁路

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKlTQ3GJvnKpG3VRjia1ianuzG2mGLe3HYjPvTsiaqmmOlOickmYRsia9fXoq7ojLiaN48pvEk0llJNqUkw/640?wx_fmt=png)

除了2017年分红略低，其他年份的分红稳中有涨，而且价格比较稳定，过去一年价格基本都在6块多，如果大秦铁路的分红能维持(可能性很大)，那么按现在7.11%的股息率，是非常划算的。  

中国银行，农业银行，工商银行，建设银行

这4大行，分红也是非常稳定，而且盈利能力虽然说没有多大的增长空间，但是不至于倒退。每年的分红总体上稳中有进，目前按股息的高低顺序是
中国银行>农业银行>工商银行>建设银行。

重庆水务

这个企业是地方国资委控股，业绩和分红非常稳定

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKlTQ3GJvnKpG3VRjia1ianuzAVvHVoQOF8OwJnrcMo7EfyAQO9WnYsPn0ibPuTthnGfnYaBdP22J4hA/640?wx_fmt=png)

股息率基本稳定在5%左右，前一段时间由于碳中和概念竟然连续两个涨停，我就顺势减了一点，然后又接回来了。

其他的不再一一列举。

按条件筛选出来的100多个股票，总能选出一些当下合适的作为门票。
