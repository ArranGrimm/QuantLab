#  投资中的种田派和打猎派

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

今年已经收到2W多的股息了，还有几万的股息正在路上。

很显然，我目前是一个种田派

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIMTLOp6BxdWu5pVK25PqzibaPAHuutsGxS5iaufw1JdicMlalJHd0QkKiaJ0m4ey75DuiacLOu145VZYg/640?wx_fmt=jpeg)

不断的种下果树(买入股票)，等待每一年的果实成熟(股息到账)，果实要么用来吃，要么继续播种(股息复投)让第二年的果树(股票数)更多，同时果树本身也会长高(股票的每股净资产增加)并带来更多的果实(股息提升)，几年以后果树长高了，卖出价(股价)也增加了，当然，这个过程中肯定会有波动。

有时候，气候不好(各种局势利空)，很多人担心果树会不涨反缩，担心果实减少，于是折价转让果树，这个时候如果对果树比较熟悉，可以发现很多有利可图的机会，可以趁机低价收购好果树。  

有时候，气候很好(各种利好)，很多人会觉得果树今年能涨出几年的高度，结出更多的果实，于是加价抢购，这时候如果对果树比较熟悉，可以发现有些报价明显高了，可以趁机高价转让果树。

种田为主，倒买倒卖为辅，轻松自在，惬意愉快。

当然这样的生活实在是过于平淡乏味，而且反馈周期长，短则几个月，长则数年，实在是无趣。

于是很多人更倾向于打猎

比如这样(图中为驼鹿)

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIMTLOp6BxdWu5pVK25Pqzibm0qlEo7FckjbWqcAIof65JYwjKXOQZliaVXBzpnd1Dr4iaWYvBnvERWg/640?wx_fmt=jpeg)

是不是既有趣，又有成就感，打到一单就吃大肉了。

但是很明显，打猎对个人能力的要求是远高于种田的

投资中打猎的机会也确实不少，比如企业并购换股的套利、企业业绩爆发之前埋伏、妖债埋伏脉冲高价卖出，等等

操作的好自然获利颇丰，如果操作的不好就会成这样

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfIMTLOp6BxdWu5pVK25PqzibkMQOdUKQZu9q9akKXOvwL70YxTXHZPOTtiaVxnuk8zo4awsmX3OtpEQ/640?wx_fmt=jpeg)

成为别人的肉。

如果哪一天我发现我的知识和能力已经明显超过了市场上大部分的打猎派，我也会在种田之余参与打猎，但是现在显然还不是时候

种田，只是赚多赚少的问题。

而打猎，如果水平不行，会成为别人的猎物。
