#  散户在投资中的持续现金流的3个来源

**首先是 工资收入的结余**

大部分散户的市值都不会很大，小的可能就是十万八万，稍微多一点的可能也就30W到50W，100W以上的比例实际上是很低的

所以如果每年工资结余假设有5W的话，相对10W的持仓，可以在低位买入股票，增加50%的仓位，这个效果是非常显著的，假设最初的10W买在高位，到低位时跌了30%，原本需要涨43%以上才能盈利(1*0.7*1.43=1)，而新增的资金低位买入，买入后综合成本就变成了
(10W+5W)/(10+5/0.7)=0.875，此时只需要涨14.3%以上即可盈利。

**第二个是 股息收入以及打新和套利收入**

比如我的股票组合目前的股息率大约是6.2%，也就是每年能获得6.2%的现金流，如果股价在低位，可以继续买入股票，如果股价在高位，则不买入。

**第三个是 动态平衡**

动态平衡通常指的是股债平衡，股指的是权益类资产(股票期权等)，债指的是债券现金存款等。

最简单的就是55平衡，也就是股票占50%，债类占50%，然后进行再平衡

再平衡可以是比例触发再平衡

比如股票涨了股债比例变成了6比4，那么卖出一部分股票，将比例平衡为5比5

或者股票跌了股债比例变成了4比6，那么买入一部分股票，将比例平衡为5比5

也可以是定期平衡，比如一个月或者一个季度或者一年，进行一次再平衡，将比例平衡为5比5

当然也可以做28平衡

在股票估值20%分位(低估)的时候股债8比2，逐渐过渡到股票估值80%分位(高估)的时候股债2比8

那么在这个过程中，股票涨的时候，分批卖出股票回笼资金，在股票跌的时候，分批买入股票，消耗资金

不论股票涨到多么离谱的高度，都能有股票可以卖出

不论股票跌到多少离谱的低位，都能有现金可以买入

相当于涨的时候不断高抛，跌的时候不断低吸
