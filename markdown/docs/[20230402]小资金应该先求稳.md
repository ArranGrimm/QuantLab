#  小资金应该先求稳

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

我的投资策略是低估分散，并且股息在选股中的权重很高，目前我的股票组合的股息率是5.6%

也就是说以目前的市值，我把账户关闭不看，一年下来能收到大概5.6%的股票分红，不论股票是上涨还是下跌

经常看到有人说，这个模式适合大资金，小资金没意思

这么说也确实有一定的道理

小资金比如10W，如果股票一年没涨，也就能得到5千多块的现金分红，确实显得很鸡肋

所以很多人拿着10W的本金，在市场里面搏杀，希望抓住一个大牛股，几个月大涨甚至翻倍。

愿望当然是好的，然而最终能实现的人却是极少数。

**如果打算把投资作为一个长期的事情来做并且期望能长期获利**

**我觉得应该从小资金就开始求稳，而不是用小资金去搏杀**

所谓搏杀，就是去参与胜率低但是赔率高的投资机会，比如做热点，做题材，打板等等

而求稳，就是去参与胜率高但是赔率低的投资机会，比如低估且稳定派息的稳健企业

我从进入市场第一天就是用求稳的方式在做投资

第一年投入的资金只有几万块，一年下来只赚了一千多块好像。

第二年稍微增加了一点，但是也没超过10W，一年下来赚了不到5千块。

收益额确实很鸡肋，但是意义却很重大

因为我认识到我的策略不仅在逻辑上是可行的，在实践上也是盈利的。

而且还有一个很重要的好处

**能得到家里人的支持**

统计显示市值低于十万的散户大概占70%，而我周围认识的有在买股票的人，大部分确实也是不到10W

这并不是因为他们只有十万，而是他们只敢投入10W，或者家里人只允许他们投入10W

**因为他们长期下来大部分时候是亏钱的**

低胜率高赔率和高胜率低赔率的投资机会从数学期望来讲应该是差不多的，但是实际操作起来却有一些变化。

比如，高胜率低赔率的投资机会，连续三年都获得了8%的回报，则3年累计收益率大概是26%

而低胜率高赔率的投资机会，如果连续三年是20%，30%，-18%，则3年累计收益率大概是28%

两者最终收益率差异不大

但前者更容易获得家里人的支持而增加资金投入

而后者可能会在第三年造成家庭矛盾而不会增加资金投入，可能会减少资金投入甚至禁止投资(并且有可能在牛市又忍不住追进去)

这个是主观上阻碍了投资的进程

还有一些问题可能在客观上阻碍投资进程，让低胜率高赔率的方式在资金小的时候遭受更大的实质性亏损

假设资金量比较小比如20W，遭遇了某一年的-18%，浮亏3.6W，而如果这个时候家里急需用一笔钱15W，被迫卖出15W的股票，市值还剩下1.4W，哪怕接下来的一年有50%的涨幅，市值也只能达到2.1W

而假设资金是200W，遭遇了某一年的-18%，浮亏36W，而如果这个时候家里急需用一笔钱15W，被迫卖出15W的股票，市值还剩下149W，接下来涨35%即可新高，不至于踏空。

这也是我为什么从最开始投资资金少的时候就用求稳的方式

因为我需要在资金变大(不论是增加资金还是盈利累积)的过程中，尽可能的规避可能遭遇的实质性亏损

所以我在市值200W以内，会以低估高息为主，让每年股息带来的现金流逐步达到10W

200W到300W的部分，降低股息在选股中的权重，比如综合股息3%左右，但是依然是低估为重点

300W到400W的部分，进一步降低股息在选股中的权重，比如综合股息1%左右，但是依然是低估为重点

每年仅仅股息带来的的被动现金流将达到15W左右，而低估的个股估值回升的话还可以卖出获得一部分差价收益。

400W以上的部分，会考虑做一些进攻性更强的机会，这部分会伴随更大的波动，但是这部分的波动几乎不会因为外部原因导致实质性亏损。

市场上用小资金做低胜率高赔率机会的散户，除了极少数可以最终胜出，大部分最终都为市场的流动性和超额收益贡献了自己的力量。
