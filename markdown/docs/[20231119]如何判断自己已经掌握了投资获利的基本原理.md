#  如何判断自己已经掌握了投资获利的基本原理

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

上周写了一篇《掌握这些基础知识就能超过市场上90%的投资者》

那么问题来了，如何验证自己已经掌握了？

在我过去大量阅读的过程中，有4篇文章我认为基本上涵盖了基于价值的投资的基本原理，而想要能读懂理解，需要一定的基础知识。

那么，刚好可以用这4篇文章来验证自己是否已经掌握了相关的基础知识(链接在文末)。

####  第一篇

《从净资产角度思考股票投资 》——长期投资可以赚的四种钱 作者：盛京剑客

读懂这一篇，需要理解

什么是净资产，什么是PB，净资产在什么情况下会增厚，净资产和股价的关系，增发如何影响每股净资产等

####  第二篇

用一个公式彻底讲透股票投资 作者：盛京剑客

读懂这一篇，需要理解

什么是PE，PE为何变化，PE变化对投资收益的影响，什么是戴维斯双击，推动某个股票上涨的长期原因是什么

####  第三篇

一篇文章，看懂价值投资的全部原理 作者：破晓笔记

读懂这一篇，需要理解

什么是每股分红，什么是股息率，每股分红不变的情况下，股价下跌后，股息率如何变化？

为什么经营正常并且低估的企业，跌到一定程度就不会继续跌了，并且大概率会估值回归

####  第四篇

穿透式破题：为什么说“无限上涨”和“无限下跌”是一样的 作者：破晓笔记

读懂这一篇，需要理解

为什么经营正常持续分红的企业，下跌也会让投资者赚到钱

分红除权的原理是什么，如果交易所不除权，股价会有什么变化

* * *

如果可以轻松的看懂上面的四篇文章，说明已经掌握了基于价值的投资的基本原理。而仅仅掌握了基本原理和基础知识是不够的，就像一个人刚学会开车，想要成为老司机还需要一定的时间，而对于投资，还需要学会看财报中的几个最基本的科目，根据估值做好仓位管理，然后进场实践，假以时日，想不赚钱都难。

如果看不懂，或者不完全懂，那说明基础还是不够，建议把基础问题先搞懂，打好基础以后再考虑进场，否则贸然进场，很大概率就是给别人送钱。
