在这个过程中，估计有不少人慌了，这个时候投资计划和现金流保底就开始发挥作用了。

投资计划就不再赘述了，上周已经写了，今天说一下现金流的保底。

虽然投资已经有几年并且都取得了正收益，而且现在也已经退出了工作岗位将投资作为以后的主业，但是不得不说，我的投资能力还远远算不上优秀，最多也就勉强算入门。

因此在投资上我极其保守，并且会计算各种情况下未来若干年的回报情况，这个从我之前的文章中就看  得  出来。

所以我的组合目前是低估高息股票为主，并且非常分散，因此即使某一年少数几个公司的分红降低了，影响也是极其有限。

接下来我持仓的公司会陆陆续续公布年报以及分红方案，除了个别去年大幅增加分红的公司(例如千金药业)之外，其他的公司大概率应该都是小幅增加分红。

在我的持仓汇总表里面，我按去年的年报计算了加权平均股息率，大概是5.7%的水平，假设股价不变的情况下，今年这些公司的分红小幅上涨，那么综合股息率可能会接近6%

再加上打新的回报，虽然目前新股时有破发，按集思录统计的过去12个月，开通了双创的账户配置10W的股票打新的收益率是14%，并且还在呈现下降趋势，那么保守一点打个对折按7%计算，那么在无视股价涨跌的情况下，每年我将能得到大约13%的现金流回报，并且每年还能小幅增长。

股价下跌不影响股息，股价下跌不影响股息，股价下跌不影响股息。

先说3遍。

因为权益类资产背后是上市公司，上市公司的利润总体上是增加的，一方面是因为公司的经营，另外一方面是通货膨胀(简单的说，原材料和人工上涨，然后产品价格上调，在其他情况不变的情况下，利润也是等比例增加的)，所以最终股价在总体上长期是上升的，即使出现了少数公司变差(这是一定会出现的)，也不影响股票组合的整体长期上涨。

既然有13%的现金流回报保底，那么股价短期的涨跌对我来说并不会造成什么影响，甚至还能提供超额收益。

在全市场下跌泥沙俱下的时候，我可以分散的加仓增加持股份额(并且这周我也是这样执行的)，而在某些事情的影响下股价出现了不合理的上涨的时候，我可以减持一部分，比如千金药业出现了不合理的上涨，从9块多涨到16块的过程中我就减持了一部分，伟星股份我也减持了大部分。

追逐市场热点，趋势跟踪，踩准风格切换，对我来说难度太大，并且也很累，还会影响情绪进而影响生活。

倒不如做好投资计划并计算好现金流的保底然后以逸待劳，在市场不理智的时候获取一部分超额收益。

投资有计划，现金流有保底，让我在波动面前内心毫无波澜，甚至还有一点想笑。
