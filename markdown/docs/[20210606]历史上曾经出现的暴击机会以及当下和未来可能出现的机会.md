#  历史上曾经出现的暴击机会以及当下和未来可能出现的机会

历史上曾经出现过很多的暴击机会，RETIs会不会成为下一个暴击的机会，现在还不好说，但是现在肯定不是。

所谓的暴击机会，指的是可以让财富上一个台阶的机会，而一个台阶，大约是3倍。

今天来说两个历史上曾经出现的暴击机会，以及当下和未来可能出现的暴击机会。

曾经的这两个机会确定性比较高，普通人也可以把握和参与，一个是折价封基的机会，一个是分级基金套利的机会，而现在可转债大量发行，也可能会出现类似的机会。

##  折价封基的机会

1998年，南方基金和国泰基金首次发行了封闭基金。封基在封闭期内不能找基金公司赎回，但是可以在二级市场交易。

它们刚推出的时候由于关注度较高等原因，大部分时候是溢价的(现在RETIs的关注度也很高，上市后很可能出现类似的情况)，后来因为一些利空的原因，场内的封基开始出现了折价，到了2006年前后折价达到了50%左右。

此时市场已经处于非常低估的区间，向下的空间很小而向上的空间很大，一部分人看到了这个风险和收益不对称的机会，开始在市场上购入折价的封基。

随后股票出现了大比例分红(是不是和今年很像，今年大量企业大幅增加了分红)，加上大牛市的启动，这一批投资者获得了丰厚的回报，其中非常著名的就是《十年十倍》的作者
金伟民 先生，他的网名 持有封基 就是为了纪念2007年靠封基赚了180%。盈利180%，相当于从1到2.8，这就是上了一个台阶。

##  分级基金套利

分级基金于去年年底全面退出市场，这个品种退出了舞台，以后会不会有类似的品种或者换一个马甲再次登台，这个就不得而知了。

2015年的牛市，让不少人在股市上大赚了一笔，当然我相信更多的散户应该在2015年的牛市大亏了一笔，其实大多数散户都是在牛市追涨大亏。

而对于成熟的投资者，2015年这种大牛市在那个估值的情况下是不适合入场的，于是有部分人寻找到了其他的投资品种，那就是错误定价的分级基金A。

分级基金A的错误定价，让投资者在只承担债券风险的情况下(低风险)，能赚到远高于债券的收益(高收益)，这种风险和收益不对称的机会，在理解了这个机会的投资者眼里简直就是bug一般的存在。

而随后的分级基金折价套利，又像是bug之上再叠加buffer，简直妙不可言。

于是这一批投资者在这一次机会中取得了丰厚的回报，

##  当下以及未来

当前的机会，我觉得可转债可能是其中一个。

从1993年第一支可转债挂牌上市开始，可转债已经年满18岁了，开始走向成熟。

最近几年可转债逐年发行量(以申购时间计算，数据来源 东方财富网)是：

2016年11支，2017年40支，2018年67支，2019年125支，2020年195支，2021年到现在51支估计全年能发行200支左右。

可以看到，可转债的发行从2017年开始加速，2019年和2020年的发行数量显著增加，很多人应该就是这两年开始知道可转债的，但是绝大多数参与者都只是打新，包括2021年之前的我。

2021年可转债市场出现了系统机会，于是我开始建仓，这一波半年不到已经浮盈13%左右。

  

可转债在未来会不会出现暴击的机会，这个谁也不知道，毕竟英科转债最高价曾经达到了3600多，相对100元的发行价，获得了35倍的涨幅。

RETIs在未来会不会出现暴击的机会，这个谁也不知道，毕竟早年封闭基金出现过，而首批RETIs的封闭期最短20年最长99年，如果出现也不足为奇。

暴击的机会是命运的馈赠，一旦出现，我们要做的就是给命运来一顿暴击。
