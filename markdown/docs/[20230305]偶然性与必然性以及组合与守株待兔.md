#  偶然性与必然性以及组合与守株待兔

在小说阅读器中沉浸阅读

上周写了一篇《 [ 做胜率赔率占优的交易-以威孚高科获利为例
](http://mp.weixin.qq.com/s?__biz=Mzg3ODIyNTYzNw==&mid=2247485118&idx=1&sn=f2db5e36cda2729ea2d8b255cea66280&chksm=cf17b734f8603e22c9ad3c3be50921aa7cabea22b4ce886cd65ab98e1f2758756d2599d4f119&scene=21#wechat_redirect)
》，有网友评论

“ _运气好出了个新题材拉了三个板，不然你是亏钱的….这种经历没参考性_ ”

从 威孚高科 这个单一的例子来看，他说的没错，三个板当然是运气

但是真的没有参考性吗？

在过去一年的时间里面，我的持仓出现过以下几个股票的涨停

双箭股份，柳药集团，伟星股份，联发股份两连板(2023年1月3号)，千金药业两连板(2022年5月16号)，保利金地建发一起涨停(2022年11月29号)

而且有些不止涨停了一次

我的组合里面有大概40支股票，而过去一年碰到的涨停得有20次，我都已经习以为常了，涨停只不过是上涨过程中很小的一部分而已

每一次都是运气(因为无法预判)，但是出现这么多次，我想这应该无法简单的用运气来解释

马克思主义哲学教导我们

“必然性不是孤立存在的，它通过大量的偶然性表现出来”

举个例子

抛硬币，只要抛的次数足够多，如果硬币没有问题的话，最终正面的次数和反面的次数是基本相等的

如果某个人每次下注1元，押中了拿回自己的1元并且赢1元，押错了输掉1元

那么可能出现连续压中好几次，赢了钱，这个就是运气，是纯粹的偶然。

但是如果赔率变成了，押中了拿回自己的1元并且赢2元，押错了输掉1元

那么结局显而易见，不断地玩下去，必然能赢很多钱。每一次赢钱，都是偶然，但最终赢钱，是必然的

胜率50%，而赔率是1赔2(拿回2元)，是一个胜率赔率既不占优又不吃亏的游戏(实际上在这个胜率赔率下，资金大的人依然占优，具体可以参考
赌徒输光定理，网上有数学证明，只需要学过等比数列即可理解)

而胜率50%，而赔率是1赔3(拿回3元)，那么这就变成了一个胜率赔率占优的有戏，不断地玩下去，赢钱是必然的(但是不能每次都梭哈，不然必定输光)

那么，假设某个股票估值较低同时业绩不是非常差(分红稳定加分)，或者某个股票某一年业绩非常差但是估值够低(比较大可能未来业绩会回复到往年正常水平)，等等，这样的情况就属于胜率赔率占优

如果买入，则有比较大的可能获利，每次获利看起来都是偶然，但是最终赚钱则是必然

这个已经被很多投资者验证过

其中最典型的就是施洛斯，通过48年的投资生涯，充分的证明了这一点。

同样的，这里非常重要的一点就是不能单吊(梭哈)

筛选出一批胜率赔率占优的目标，分散买入构建成组合

剩下的事情就是守株待兔，等待偶然的出现，最终赚到必然的钱。
