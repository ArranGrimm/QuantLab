#  假如高息股成为新的共识

本周二(2023年6月20日)，最新一期LPR出炉，五年期下调至4.2%，而在前一段时间存款利息也下调过

随之而来的是稳健型理财产品的收益率也会慢慢下降

不知道指望靠4%的理财收益来实现FIRE(Financial Independence Retire
Early，财务自由，提前退休)目标的人，是不是离目标又更远了

而在股票市场，前几年被追捧的各种白马股，成长股，也熄火了，前几年买入的基金，到现在大部分都还有不小的浮亏。

当时捧得有多高，现在摔的就有多痛，市场就是这样一直不停的轮回

有人在轮回中一直亏钱，有人在轮回中赚赚亏亏蹉跎岁月，也有人在轮回中一直赚钱

前几年的共识就是，白马好，有前景，贵有贵的理由，打不过就加入。

结果加入以后，吃肉的没几个人，挨打的一大批。

经过几年的惨痛，白马股不少都已经腰斩了，大家又开始不待见白马股了

那些看好白马股的理由，在股价大跌的面前，都显得那么苍白无力。

而经过前一段时间的中特估，一些人开始发现，原来有一些股票的股息回报这么高

比如兴业银行，目前股息回报率高达7.53%

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfKhjY8s2rnzCkpqUk92PRoBWZw5ibt8rvEcXAtyuTmdZxxVeSAhkQqDn2t4BicUUPFg84NYWNpx8NqA/640?wx_fmt=jpeg)

而农业银行去年一直在3块以下，最低甚至到了2.7

那个时候如果有人说银行股息回报高，就会收到一堆嘲讽，什么银行坏账啦，什么你要它利息它要你本金啦

但是后来一顿大涨，银行的坏账好像也无关紧要了，也不觉得图本金了，毕竟从最低2.7到最高3.97，涨幅高达47%

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfKhjY8s2rnzCkpqUk92PRoBO1PCQ3ZxwjnrqCQOxXrduM8ggETn7hFj4erllpBibPADwDvyEwRu2Bg/640?wx_fmt=jpeg)

  

什么风险不风险的，股价一涨，风险都不见了，看到的都是优点。

当然热度过去，股价还是掉下来不少，但是仍然比最低点涨了30%

还是引起了一些注意，但是现在真正下手的少，将信将疑的多。

如果再涨个50%，市场没准就开始形成新的共识

比如，"低利率环境下，高股息回报价值凸显"，"股息回报才是真金白银的回报"，反正各种彩虹屁就会出来

而如果真的形成共识，那么就会因为趋势加强而涨过头

比如本来股息6%到7%，股价涨上去一倍，结果股息回报就只有3%左右

然后可能继续产生新的共识，比如"国有银行是优质资产，PB低于1是不合理的"，等等

要知道现在农业银行的PB是0.54，工商银行的PB也是0.54，涨一倍也才到1.08

而2018年1月到2月的时候，最高可是到了1.31

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfKhjY8s2rnzCkpqUk92PRoBLS3udppRI9xnHrKIGCzaYWNZ8pBB5XSRycIHvuic2iavWh0FOvzxH5Eg/640?wx_fmt=jpeg)

如果市场真的对高息股(包括但不限于银行股)形成了热情而强烈的共识

那么就该考虑将这些烫手的股票转让给热情的市场，再筛选一批被抛弃后跌入谷底的企业，用骨折价买入

然后，等待旧的共识破灭，等待新的共识形成。
