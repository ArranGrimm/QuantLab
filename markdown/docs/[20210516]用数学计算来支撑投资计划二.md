今天来做一点更详细的理论计算。

首先是整体上的认识

从长周期来看，投资A股整体上是赚钱的，毕竟从1991年开始中国的GDP的增速一直在10%上下，最近几年也一直在6%到7%，那么上市公司作为全体企业中的佼佼者，虽然里面垃圾不少，但是真正优秀或者良好的企业，还是很多的，因此投资这些优秀或者良好的企业，是可以取得略高于GDP增长的回报的，也就是至少能有大约10%，如果具备一定的知识，做一些基本筛选和估值判定，将收益率提高到15%的水平，并不是一件非常困难的事情，当然如果想要提高到20%或以上，则难度会大幅增加。

所以从整体上看，一个具备了一定知识的投资者，只要不是瞎投资，从长时间周期来看，是能够取得10%到15%的年化收益的。

然后是我自己的体系

我自己的计划是将可投资的资金，保留一年(两个人都上班的情况下)到两年(一个人不上班的情况下)的家庭开销存为周期长短不一的定期理财(后期也会考虑用债券类代替一部分)，其他的部分全部换成股票以及一部分可转债，并且增量的资金也全部换成股票以及一部分可转债，靠股息和可转债以及打新收益来提供现金流覆盖家庭开销，股票市值自然增长。

所以从这个角度上来看，我是将投资作为一项长期的工程来维护的，因此不存在牛市来临前埋伏后捞一把买房买车的幻想。

而我每年元旦将上一个交易日的收盘价作为这一年的初始成本，那么每年都可能出现相对年初浮亏的情况，这个是必然的。既然是必然的，那就是计划的一部分。

那么如果某一年出现相对年初浮亏，有没有问题？

并没有什么大问题。

按我上周的文章《用数学计算来支撑投资计划》中写到的，我的股票组合加权PE为9.05，加权PB为1.34，加权ROE为14.86%，加权股息率是4.93%。

这意味着什么？

我的组合有30多支股票，其中占比超过2%的有15支合计占比约80%，因此可以认为这是我自己构建的一个基金，假设命名为：增援未来红利增强个人组合基金。

组合中的成员和比例，我也会不定期的调整。

为了便于计算，假设这个组合基金的单价是100元，份额是10000份，PB1.34则每份净资产为74.627元，净资产收益率14.86%则每份净利润11.09元，每份股息4.93元，则股息支付率(股息/净利润)为44.45%。

假设ROE维持14.86%不变，股息支付率也不变，每股利润拿出44.45%分红，55.55%留存到企业增厚净资产，派息不填权，红利复投，会得到什么结果。

  * 第1年，每份净资产74.627，每份利润11.09，分红4.93留存6.16，单价除权后95.07，分红买入518.565份，总份数10518.565，总市值100W。 

  * 第2年，每份净资产80.787，每份利润12.005，分红5.336留存6.669，单价除权后89.734，分红买入625.5份，总份数11144.065，总市值100W。 

  * 第3年，每份净资产87.456，每份利润12.996，分红5.777留存7.219，单价除权后83.957，分红买入766.81份，总份数11910.875，总市值100W。 

由此可以看到，如果分红后一直不填权，总市值是不变的，也就是很多人所谓的"自己的钱分给自己"，事实上真的会一直赚不到钱吗？后续的计算用程序来跑一下

    
    
    # -*- coding: UTF-8 -*-  
    jingzc, roe, zhifulv,gujia, gushu, shizhi = 74.627, 0.1486, 0.4445, 100, 10000, 1000000  
    print "             股息    留存    份数        股息总额      股价      买入      市值"  
    for i in range(1,21):  
        lixi=jingzc*roe*zhifulv  
        liucun=jingzc*roe - lixi  
        gujia-=lixi  
        if gujia<=0:  
            break  
        mairu=lixi*gushu/gujia  
        print "第 {0:2d} 年:    {1:4.2f}    {2:.2f}    {3:.1f}    {4:.2f}万    {5:.2f}    {6:.1f}份    {7:.2f}万"\  
        .format(i, lixi, liucun, gushu, lixi*gushu, gujia, mairu, (gushu+mairu)*gujia)  
        gushu+=mairu  
        jingzc+=liucun  
        shizhi=gushu*gujia

看一下结果(数字做了小数点两位数的四舍五入)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfJee8rr3IHfmyAict9xI2rYDL6B4Hxpze8ibDgvib2RjdMGnB8XaKxQ8CYmFZGDtGEdnQf1jQkCJxp1g/640?wx_fmt=png)

可以看到，如果一直不填权，第12年分红后，股价就降低到5.03元，持有份数59432.4份，每份分红11.8元，分红总额
70.1万元，可以买入139346.1分，买入后持有份数是198778.5份，总市值还是100W。

由于每份分红是11.8元，而当前每份的价格只有5.03元了，第13年如果再分红，单价就会变成负数，先暂且不管，从结果中可以看到分红每年增长约8.25%。

接下来，将会出现一件神奇的事情，第13年能拿到多少分红？

11.8*(1+8.25%)*198778.5=253.9W

假设不做任何交易操作，第14年的分红是253.9W*(1+8.25%)=274.8W，第15年297.5W，第16年322W。

后面就不继续计算了，相当于100W经过12年后，在接下来的4年里面能拿到合计1148W的分红，手里的份额卖不卖已经不重要了。

而如果在前面的12年中，如果股价不涨反跌会怎么？那就会把12年的进度提前完成，因为股价跌了分红不会减少，分红买入的份额却更多了。

简直是做梦都要笑醒。

当然这一切都有一个前提，就是组合能够维持这个ROE。

另外企业变差和增发股份会让这个组合的质地下降。

这个就需要不定期的维护组合，难度并不大。

最难的其实是两点：

1.积攒100W的资金构建一个这种估值和ROE的组合

2.耐心等待十年以上

这两点对我都不是问题，第一点快完成了，第二点就是我的计划。

既然结果基本上已经确定了，剩下的就只剩下等待时间了。

但是只要智商正常的人都能看出上面的情况是不可能出现的，因为如果一直不填权，到了第12年我的组合中的股票综合起来相当于分红11.8元，而价格却只有5.03元，这种买入就能白赚的股票价格显然是不会出现的(但是在信息极不发达的几十年前，巴菲特买过分红超过票价的股票)。

既然不填权的情况是不可能发生的，那么按每年填权再来计算一下。

  * 第1年，每份净资产74.627，每份利润11.09，分红4.93留存6.16，单价100，分红买入493份，总份数10493，总市值104.93W。 

  * 第2年，每份净资产80.787，每份利润12.005，分红5.336留存6.669，单价100，分红买入559.9份，总份数11052.9，总市值110.53W。 

  * 第3年，每份净资产87.456，每份利润12.996，分红5.777留存7.219，单价100，分红买入638.526份，总份数11691.426，总市值116.91W。 

后续的计算用程序来跑一下

    
    
    # -*- coding: UTF-8 -*-  
    jingzc, roe, zhifulv,gujia, gushu, shizhi = 74.627, 0.1486, 0.4445, 100, 10000, 1000000  
    print "             股息    留存    份数      股息总额    股价      买入        市值      增长"  
    for i in range(1,21):  
        lixi=jingzc*roe*zhifulv  
        liucun=jingzc*roe - lixi  
        mairu=lixi*gushu/gujia  
        print "第 {0:2d} 年:    {1:.2f}    {2:.2f}    {3:.1f}    {4:.2f}万    {5:.2f}    {6:.1f}份    {7:.2f}万    {8:.2%}"\  
        .format(i, lixi, liucun, gushu, lixi*gushu/10000, gujia, mairu, (gushu+mairu)*gujia/10000, ((gushu+mairu)*gujia-shizhi)/shizhi)  
        gushu+=mairu  
        jingzc+=liucun  
        shizhi=gushu*gujia

结果显示第20年总市值为879W，折合年化收益率为11.48%。

仔细看一下会发现市值的增长情况

前5年的增长率分别是：4.93%，5.34%，5.78%，6.25%，6.77%

最后5年的增长率分别是：16.2%，17.54%，18.98%，20.55%，22.25%

和当年的股息率(股息/股价)保持一致。

为什么增长率会越来越高？

因为随着每年的利润增加，股价没有增加，新买入的份额估值越来越低，回报率越来越高。

但是股息稳定且略有增长的资产，如果股息是22元多，市场上出现100元的价格的情况也是非常罕见的。

同样的，如果股票每年只是填权，而不上涨，简直是做梦都要笑醒。

那么假设股价填权后每年还增长5%，情况会怎样？

假设股价填权后每年还增长10%，情况会怎样？

用程序来跑一下

    
    
    # -*- coding: UTF-8 -*-  
    jingzc, roe, zhifulv,gujia, gushu, shizhi = 74.627, 0.1486, 0.4445, 100, 10000, 1000000  
    print "             股息    留存    份数      股息总额    股价      买入        市值      增长"  
    for i in range(1,21):  
        lixi=jingzc*roe*zhifulv  
        liucun=jingzc*roe - lixi  
        gujia*=1.1  
        mairu=lixi*gushu/gujia  
        print "第 {0:2d} 年:    {1:.2f}    {2:.2f}    {3:.1f}    {4:.2f}万    {5:.2f}    {6:.1f}份    {7:.2f}万    {8:.2%}"\  
        .format(i, lixi, liucun, gushu, lixi*gushu/10000, gujia, mairu, (gushu+mairu)*gujia/10000, ((gushu+mairu)*gujia-shizhi)/shizhi)  
        gushu+=mairu  
        jingzc+=liucun  
        shizhi=gushu*gujia

结果如下

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfJee8rr3IHfmyAict9xI2rYD9OFYapIrUtQQFRR2VvdWwiaYxrw9XbAU2WfbyuVM4herUMla2W7RCvA/640?wx_fmt=png)

可以看到第1年的收益率为14.93%，实际上就是4.93%的股息，加上填权后股价增长10%。

然后可以看到收益率一路走低

这个不难理解，因为roe只有14.86%，派息后每年净资产的留存是不足10%的，而股价的增长是10%，这样股票的估值会越来越高，而复投的收益率会越来越低。

如果把打新股和后续的增量资金计算进去，收益额还能更高一点。

当然以上的计算是基于理想情况的理论计算，而市场总是处于波动之中，不过不管如何波动，对于roe带来的回报和股息带来的现金流，是和股价没有关系的，只和企业的盈利情况有关。

所以我只需要把主要的注意力放在每年4次的财报上就可以了，同时关注一下组合的估值水平和全市场的估值水平，如果显著低估那就加一点，如果显著高估就减一点，其他时候该干啥干啥。

稳健的组合每年都是平平无奇，直到多年以后，变成一个巨大的数字。
