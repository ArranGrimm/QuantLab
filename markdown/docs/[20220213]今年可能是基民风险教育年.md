#  今年可能是基民风险教育年

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

最近投资社区的热门新闻是基金大跌，什么基金经理掉到黄浦江，什么几百只私募面临清盘，什么某某医疗基金的基金经理人品有问题等等等等。

上一次我听说这个某某医疗基金的时候，还是它在进行铺天盖地的广告宣传的时候，具体时间我已经记忆模糊了，特意百度了一下，应该是2020年818理财节。

遥想那个时候，中午排队吃饭听到很多人在议论这个基金，当时我并没有在意，因为这里有一个很简单的逻辑

**如果大力宣传的基金大部分人都能赚钱，那么下一次还会有更加大力的宣传，如果仍然有大部分人赚钱，那谁还会上班？**

所以基于这个逻辑，很明显的结论就是这个基金的基民，最终仍然逃不过7亏2平1赚的规律。

但是显然不可能刚开始申购就开始亏钱吧，这样的话下次谁还申购，那么什么时候开始亏钱呢？

什么时候开始亏钱我没有具体去了解，但是从现在的情况来看，应该是大部分人已经出现了较大的亏损

于是网上先是出现了吐槽，接着出现了咒骂，然后还有各种黑料(真的假的都有)

那么问题来了，基民在申购基金的时候有没有想过自己会亏钱？

我觉得虽然大家嘴上都说投资基金有风险，但是内心深处是不认为自己会亏钱的，因为很多人既不关注也不了解证券市场，如果觉得会亏钱，肯定是不会进场的，所以一旦开始进场都是认为这一波会赚钱。

这里我不由得又想起了另外一个段子，有人说公交车上听到大妈说炒股很简单，只要买白酒就行了，买入白酒很快就涨了30%，  那个时候  茅  台  正在
从1000块走  向  200  0块。

当时我听到这个段子就知道，又有很多人要亏钱了，因为很多人前面会忍，到了茅台2000块的时候大概就要忍不住了大批进场

当然后来茅台涨到了2600，在2000块看空的人估计在这个阶段不断地被打脸

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfL5sgjibFRoEXggdvrC402sq0TU9ibxKicNakO2zEDWT1U9BsyXxIia6ggDLm4gcHvFvpKiaG1JicmUiaGJg/640?wx_fmt=png)

要说有没有人赚钱，当然也是有的，即便是1700块买入茅台，到现在也还有8%多的收益，但前提是在16xx的时候没有跑路。

基民选择基金而回避股票，主要原因有两个

1.不会选股，所以选基金

2.听说股票的风险大(波动大)，所以选基金

但是真的是这样吗？

根据中基协的数据，截止2021年12月31号，全市场基金数量已经超过了9000，而股票数量是4000多

在9000多里面选，并不会比在4000多里面选要来的容易

而风险大(波动大)，首先波动并不是风险，很多时候甚至是机会，当然不适应波动是普遍心理，那么也可以自己构建组合，比如我的持仓股票就有40只左右，涨跌对冲一下波动也不大

当然可能有人会说这么多股票看得过来吗？其实这个问题，要说清楚得单独写一篇，这里只需要思考另外一个问题，如果自建组合看不过来，那么基金里面的几十个股票是怎么看得过来的？

当然可能又有人说了，基金经理选股了我就不用看了，那么问题来了，从9000个基金里面如何选？

更有趣的是还有很多人在基金上做高抛低吸(高吸低抛)，主动基金的基金经理已经做了投研工作，如果基民的操作水平比基金经理高，那应该参考基金的持仓然后自建组合，如果基民的操作水平比基金经理低，那么操作就只有副作用。

还有很多奇葩的，明显错误的思路和操作，就不一一列举了，  当然并不是说基金不好，在一些特定情况下选择基金是更优的，这个以后再单独写一篇。

今年的基金可能会给大部分基民进行一次风险教育，当然这个教育可能并没有什么用，几年后同样的剧本还会继续为新一届的基民重播。
