#  系统性机会很容易赚钱

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

去年底今年初市场有两大系统性机会，一个就是可转债，一个就是低估值港股，这两个我都额外投入了一些，目前收益还是挺不错的。

以下提到的股票和债券，不构成投资建议，盈亏自负。  

先说可转债，年初的时候可转债全市场大跌，大面积出现了100元以下的低溢价转债，市场的情绪非常悲观，甚至出现了大V发帖说不玩了，简直让人大跌眼镜，明明是市场在送钱，却把大批的人吓跑了，既然如此，我就不浪费市场的馈赠了，构建了一组双低可转债组合。

1月底构建完成的，到现在大概是这样的。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKcfdBwpxmcqso6RrCH7JkwqbBl5Siah1REg15GTYgschkjWDlCx9zaR3s0URajXkCe2KTwqEg2u1A/640?wx_fmt=png)

这里显示的浮盈是1.9W多，清仓的可转债没有显示为浮盈，加上两个清仓的

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKcfdBwpxmcqso6RrCH7Jkw0ct8IUSOC26D2bJus06ibbsbIljgaicAFZkKNmiae4HyAibw4iaXap9qzEQ/640?wx_fmt=png)

差不多4个月时间赚了2.2W。

有难度吗？没有，只要能识别系统机会然后买入即可，当然也要做好短期内不赚钱的准备，毕竟市场什么时候恢复正常谁也不知道，虽然按过往历史计算一下大概两年多能获得大约40%左右的收益。

只是没想到这次恢复的这么快。

实际上我是略有失落的，因为市场长期不恢复，更有利于长期投资者积攒更多的份额，不过恢复了就恢复了，也没有关系。

现在可选的虽然少了，但也不是完全没得选，同时也期待下一次机会。

再说一下港股，大面积出现PB 0.5 PE0.5左右的股票，业绩好好的，市场却给出了一个破产价，不收一点实在可惜。

于是收了一点点上海实业控股(10.5港币)，新华文轩(4.77港币，是A股新华文轩的5折)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKcfdBwpxmcqso6RrCH7JkwUqpaK0ckqVibyWeUhQf5DLJXnuKwgKZ106dku2U28kcqQhT2D2qKZMw/640?wx_fmt=png)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKcfdBwpxmcqso6RrCH7JkwFHK0hOpbCQ7QqfCFGStgBKzypMsib9RwjvRCTMLTI55cqW9qs3ugCKQ/640?wx_fmt=png)

后来又收了一点中国神华，不过16.2出了，略为可惜。

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKcfdBwpxmcqso6RrCH7JkwLrZaD4WiahmYgpGw0nlibBuv53uUAAGO9judZDuakTEunNCLoYL4PJfg/640?wx_fmt=png)

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfKcfdBwpxmcqso6RrCH7Jkw5IELOg0g5mGbMeicwG4BvIbfUR2SPqJsC6MIgN9brLfQH1AgGj24xWg/640?wx_fmt=png)

虽然都赚了百分之十几，但是投入的少，算下来赚的钱不多。

A股也有很多票处于系统机会区，目前还没有到收获的时候，这个是我仓位的大头，继续持仓等待。

简单吗？挺简单的，机会来了接就是了。

轻松吗？太轻松了，买入躺下等就行了。
