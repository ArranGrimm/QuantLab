#  看到一个up主说价值投资让自己亏钱，放弃坚持11年的价值投资

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

前两天有个朋友转发了一个视频，是一个up主讲自己为什么要放弃坚持11年的价值投资，发布时间是2020年12月19

有兴趣的可以自己到B站搜索看看，这里我把部分内容摘录出来

up主说自己看过《聪明的投资者》和《证券分析》，大概看了一下，没有看的很深

貌似他就记住了几个数字，比如市盈率不超过15倍，市净率不超过1.5倍，或者市盈率*市净率不超过22.5，至于流动资产、流动负债，资产质量等等，并没有提到，也不知道仅仅只是没说还是在投资的过程中根本就不关注。

反正看到一条弹幕：“后面的3张表你没学习？”，这3张表指的就是资产负债表、现金流量表、利润表，我感觉这个up主没有学的可能性很大。

然后还有另外一条弹幕：“一股东施效颦的味道”，给我整笑了。

因为按up主来说，按这些标准只能找到银行地产保险，也就是前几年的雪球三傻，当然今年由于中特估这三傻其实还行

至于说只能找到银行地产保险，其他行业都不符合，从我自己实践的经验来看并不是这样的

比如今年上半年大放异彩的传媒(中文传媒、南方传媒等)，在去年都是完全符合标准的，中文传媒去年按低估买入，在今年上半年大概能获得50%到100%的收益率

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfKAUalHmwt1KazPnURdfe54ouzG8mBv3AJFPKwia3fdM2qqnxgibfyfcyN8wyg3N55vpR15Ge0vEllg/640?wx_fmt=jpeg)

而南方传媒就更离谱了，从去年的低位最高涨了300%

![](https://mmbiz.qpic.cn/mmbiz_jpg/xd7OSHTJMfKAUalHmwt1KazPnURdfe54xlcKVmV7CySvUmcMtPhGMvOP7CJakYJf9rpwWibHfbKzeW0AdpwBL5g/640?wx_fmt=jpeg)

类似的例子还有很多，并不是只有银保地

所以学东西，学一半太可怕了。

然后up主又说：“四大行有涨一点点，但是真的涨的太慢了，对于我们这些炒股的人，大家来股市不是想追求一年8%的，都想着一年20%，一年30%，就起码有这种追求，不说能不能做到这个结果先，起码要有这个追求”

嗯，这个追求有点高，确实有点高。

现实中我发现很多一心追求一年30%的，大概率最终赚不到啥钱(当然肯定也有少量确实赚了很多)，而把标准降低到10%左右的，反而能持续赚钱，而且有时候还比10%高出不少。

最逗的是这一句：“为什么放弃价值投资，因为价值投资是让我赔钱最多的方法”

这到底是价值投资的问题，还是这个up主自己的问题。

不过有一句话我觉得他说的还是很对的：“因为你价值投资，你才相信自己是对的，结果最后你是错的”

这个在很多号称价值投资的散户身上都出现过，重仓或者单吊某股票，跌了，然后坚信自己是价值投资(实际上是不是真的也不好说)，然后巨亏。

要知道就算是价值投资公认的大师，巴菲特2020年割肉航空股，芒格2022年割肉阿里巴巴，也就是说价值投资也不能保证买的股票就一定是对的。但是号称价值投资的散户很容易出现死扛，认为自己是对的，但是实际上巴菲特和芒格在发现错误的时候也不会死扛。

不过话说回来，这个up主讲起来有点像相声，听着玩儿也是可以的，然后我就把这个话题的后面几个视频也看了

其中提到一点，倒是事实：“做基金经理的，难道说自己是做投机的吗？难道说自己是做博傻的吗？哪怕我是基金经理，我也吹牛说自己是价值投资，虽然大部分基金经理都是在做投机，但他为了把这个基金卖出去，他要把自己包装成是价值投资的，不能说自己是做投机的”

看完他的视频，我更加能理解为什么很多人不愿意做价值投资了，当然这些人里面大部分人的理解其实是有问题的，还有一部分人的理解没有问题，但是觉得自己可以靠其他方式更快的赚钱。

各行其道，各自安好，也挺好的。
