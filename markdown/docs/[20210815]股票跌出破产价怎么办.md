#  股票跌出破产价怎么办？

如果是一个盈利良好的企业并且没有发生显著的负面变化，在控制仓位的情况下，应该适当买一些。

以下内容带有强烈的个人偏好和偏见，不构成投资建议。

前一段时间地产行业的股票普遍大跌，保利地产甚至最低跌到了9.76元，这意味着什么？

![](https://mmbiz.qpic.cn/mmbiz_png/xd7OSHTJMfIQ7qy4OiaTLric66hVJ3m98arrDTRWXehS9osnzKiab5ScedWD32VQLr5xKgicOfxVBRYrL4q5Kzty8Q/640?wx_fmt=png)

9.76元对应PE 4.016，PB 0.719，相当于买下整个公司，按当前的利润水平4年可以收回成本，直接破产清算相当于7毛2的投入可以拿回一块钱。

如果算上本周五派息0.73元，则9.76元的保利地产相当于只要9.03元。

市场给出这个价格，是觉得保利地产要破产了？

保利地产明明是央企国资控股，市场却担心它破产，这完全是过激反应。

抛开什么房住不炒，什么芯片是未来，什么资金都在布局新能源等等，让我们把思绪回归到常识来问自己几个问题

  1. 房地产行业会不会集体覆灭？ 

  2. 作为央企的保利会不会破产？ 

  3. 作为地产龙头梯队的保利地产会不会没有利润？ 

以上3个问题，答案都是否。

当然市场最大的担心是房住不炒，是三道红线，以及没有资金青睐地产行业的股票。

其实房住不炒和三道红线，我们可以换一个思维来思考一下。

假设一个班级有50个学生，以前考60分以上就有奖学金，分数越高奖学金越多，并且考试时间是3小时。

现在学校发布了新规，奖学金总额只有原来的80%(房住不炒收缩总体利润)，并且考试时间缩短为1小时(三道红线条件变的更严格)。

这个时候家长(股民)慌了，完了完了，这帮孩子们考试成绩肯定要下降了，拿到的奖学金也少了，赶紧换学校(调仓)。

但是事实上真的是这样吗？不全是。

对于原来只考70分左右的学生，那可能这下直接就不及格，没有奖学金了，确实很糟糕(H大)，对于长期排名靠前的学霸，结果可能是原来考100分的现在只能考95分，原来考95分的现在只能考90分。

总分确实很大可能降低(ROE下降)，但是由于原来70分的掉到不及格拿不到奖学金(让出市场份额)，这些学霸们最后分到的奖学金可能没有太大的变化(头部企业的利润)。

这样一看，情况是不是就乐观多了？

所以在这个过程中，我继续小幅加仓，不仅加了保利，还加了万科，以及工商银行(难道工商银行会倒闭？)

当然，我加仓的幅度很小，毕竟细水长流。

这种情况之前也出现过，比如我在《港股通买入上海实业控股顺带简要分析和港股通交易注意点》中写到10.5港币买入上海实业控股，目前价格为11.6，加上0.52的分红，8个月收益率
(11.6+0.52)/10.5-1=15.4%

你要是问我对上面这个逻辑是不是100%确定，那我肯定不能100%的确定。

要是能100%确定，我直接抵押房子借200W在9.76买入保利地产，在前天12.83卖出，10天获利30%赚60W不香吗？

显然我无法做到，谁要是真的做到了，我认为运气的成分更大。

但是这并不妨碍我们分批加仓，毕竟在现在这个位置胜率和赔率都比较可观了，可以适当投入。

我们可以想象一个丢骰子游戏，丢出3点或以上翻倍，丢出1点2点归零，则平均丢6次会有2次归零4次翻倍，数学期望是6元投入收回8元。

那么假设某个人把手里的钱一次性押下去，则可能翻倍也可能立刻归零，并且只要他每次都梭哈则一定会归零，即使把钱分成3份下注，也可能连续3次归零，概率是1/27，大约3.7%，出现的可能性也不是太小。

正确而安全的做法是更小额的下注，比如每次下注1/100，这样长期下来就能获得8/6 -1 = 33.3%的回报。

所以可以看到，我上面都是小幅加仓。

当然这两周地产板块都出现了大幅上涨，估计也会有很多帖子研报啥的分析原因，对于这些分析，我认为基本上都是先射箭后画靶子，都是马后炮。

根本原因就是价格跌到破产价，投资价值凸显，其他的都是表象。即便是没有H大的危机可能解除，也可能是别的利好，至于是什么利好其实并不重要。

就像其他涨的高的板块，后面因为什么利空大幅回调，也并不是因为这个利空，而是因为投机价值大投资价值小，价格脆弱的一碰就倒，和惊弓之鸟一样，至于是哪个消息引起的并不重要，反正总有一个消息背锅。

所以，当机会出现时，你是避之不及，还是果断下注？
