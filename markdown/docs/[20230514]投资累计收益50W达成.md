#  投资累计收益50W达成

原创  增援未来  [ 增援未来 ](javascript:void\(0\);)

__ _ _ _ _

在小说阅读器中沉浸阅读

按道理应该在累计收益达到100W的时候发帖记录，但是考虑到以目前的持仓市值，累计收益额要达到100W，短则还需要一两年，长则需要三五年

那就在累计收益额达到50W的时候先发一个贴来记录一下

一般发这种贴都会迎来下跌，不过对我来说没什么关系，因为我还有比较充裕的现金储备，下跌的话正好可以再增加一些仓位，这样达成100W的累计收益可能还能加快一点，前一段时间的上涨多少让我有点不知所措。

累计收益50W，说多不多，说少不少。

自从2018年初正式开始投资以来，我一直秉承着低估分散的投资风格，后来搭配了一部分可转债。

到目前5年多一点，没有哪一年有惊艳的回报，也没有哪一个股票全程吃到几倍的涨幅(卖掉以后又大涨的倒是有好些)

第一年投入的资金只有几万块，赚了一千多

第二年投入的资金增加了一些，也没有超过10W，赚了不到五千块

第三年投入的资金提高到30W，并另外存了20W做逆回购，合计50W用来开通科创板，这一年赚了2W多

第四年也就是2021年初，可转债黄金坑出现了，构建了可转债组合同时也增加了股票仓位，经过可转债大半年的回升，10月底总仓位达到了100W，这一年的收益接近20W，主要是可转债和新股收益

第五年也就是2022年，继续陆陆加仓，4月的黄金坑上证指数最低2863，10月底的黄金坑上证指数最低2885，这一年加仓合计耗资50W，最终这一年赚了大概10W

第六年也就是今年，加仓减仓，到目前为止增加投入大概10W，盈利接近20W，使累计收益达到了50W

可能有人会疑惑为啥我有那么多资金加仓，实际上在开始投资的第一年家庭积蓄就已经超过了50W

加上每年家庭的结余以及2021年春节前收到了金额非常可观的奖金(这也是我能够退出职场的重要因素)

因此家庭的现金储备一直都是非常充裕的，只不过第一年只投入了几万而已

一来我不确定我的心态是否能适应股票市场的波动，因此最开始只是小比例投入，然后慢慢增加

二来虽然我搞清楚了如何靠投资赚钱，但是那只是理论上逻辑上没问题，实践中是不是真的没问题，需要一点一点验证

最终这5年下来，我发现我的心态完全能适应市场的波动，对涨跌没有什么感觉，尤其是大跌的时候甚至想笑

后来我看到了一句话

**"面对大跌，抛售者损失惨重，而买入者和持有者获利丰厚。"**

而我的投资策略和逻辑，经过5年多的验证，应该也没有明显的问题，毕竟仅仅靠运气应该是不够的。

期待下一个里程碑早日达成
